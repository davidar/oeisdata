## Sage
```Sage
def basepqExpansion(p,q,n):
    L, i = [n], 1
    while L[i-1] >= p:
        x=L[i-1]
        L[i-1]=x.mod(p)
        L.append(q*floor(x/p))
        i+=1
    L.reverse()
    return Integer(''.join([str(x) for x in L]))
[basepqExpansion(4,2,n) for n in [0..50]] # _G. C. Greubel_, Aug 20 2019
```

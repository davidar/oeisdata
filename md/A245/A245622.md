## Maple
```Maple
filter:= proc(n)
local L;
if not isprime(n) then return false fi;
L:= convert(n,base,10);
numboccur(1,L)>numboccur(3,L)
end proc:
select(filter, [2*i+1$i=1..1000]); # _Robert Israel_, Aug 05 2014
```

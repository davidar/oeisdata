## Maple
```Maple
filter:= proc(n)
  local L;
  if not isprime(n) then return false fi;
  L:= convert(n,base,10);
  numboccur(3,L) > numboccur(1,L);
end proc:
select(filter, [$1..1000]); # _Robert Israel_, Aug 03 2014
```

## Maple
```Maple
A245459:= proc(n)
local T,k,x;
T:= 0;
for k from 3 by 2 do
  x:= k^n - 2^k;
  if x <= 0 then return T fi;
  if isprime(x) then T:= T+1 fi;
od:
end proc:
seq(A245459(n),n=1..100); # _Robert Israel_, Jul 23 2014
```

## Python
```Python
from sympy import prime, primerange
def f(p): return 2*p+1 if p%3 == 1 else p+1
def aupton(nn): return [f(p) for p in primerange(5, prime(nn)+1)]
print(aupton(58)) # _Michael S. Branicky_, Jun 15 2021
```

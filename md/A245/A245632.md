## Python
```Python
from sympy import perfect_power
def a(n):
    s, k = str(n), 0
    while not perfect_power(int(s+str(k))): k += 1
    return k
print([a(n) for n in range(1, 81)]) # _Michael S. Branicky_, Jun 05 2021
```

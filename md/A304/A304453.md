## PARI
```PARI
{a(n) = my(B, k);
        if(n >= 0,
          B = List(binary(n)); k = 1;
          while(k <= #B,
            if(B[k] == 1,
              k++; listinsert(B, 0, k));
            k++);
          sum(k = 1, #B, B[k]*(10^(#B - k))))}
```

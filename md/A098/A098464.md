## Python
```Python
from fractions import Fraction
from sympy import lcm
k, l, h, A098464_list = 1, 1, Fraction(1, 1), []
while k < 10**6:
    if l == h.denominator:
        A098464_list.append(k)
    k += 1
    l = lcm(l,k)
    h += Fraction(1,k) # _Chai Wah Wu_, Mar 07 2021
```

## Python
```Python
from sympy import isprime
def afind(limit, startk=1):
    for k in range(startk, limit+1):
        if isprime((k+1)**(k-1) - k): print(k, end=", ")
afind(200) # _Michael S. Branicky_, Aug 17 2021
```

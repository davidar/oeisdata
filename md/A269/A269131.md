## Python
```Python
seq = []
for n in range(2, 1000):
    temp = n
    factor = 2
    while temp > 1:
        if temp % factor == 0:
            temp //= factor
            if temp == 1:
                continue
            if temp < factor * factor:
                seq.append(n)
        else:
            factor += 1
print(seq)
```

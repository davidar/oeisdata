## Python
```Python
from sympy import factorint
def prod6(n): return sum(factorint(n).values()) >= 6
def aupto(lim): return [k for k in range(lim+1) if prod6(k) and prod6(k+1)]
print(aupto(27135)) # _Michael S. Branicky_, Jul 10 2021
```

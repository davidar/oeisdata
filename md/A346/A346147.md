## Maple
```Maple
P:= select(isprime, [2,seq(i,i=3..10^5,2)]):
f:= proc(n) local s,t;
   s:= P[n]+P[n+1];
   t:= P[n]*P[n+1];
   if isprime(t mod s) and isprime(floor(t/s)) then return P[n] fi
end proc:
map(f, [$1..nops(P)-1]);
```

## Python
```Python
from sympy import isprime
for n in range(1, 100):
    d = 1 if n%2 == 0 else 2
    ct = 1 if isprime(n) else 0
    while n - d > 2:
        k = isprime(n+d) + isprime(n-d)
        if k == 2: ct += 2
        elif k == 1: break
        d += 2
    print(ct)
```

## Python
```Python
from math import prod
def nzpd(n): return prod([int(d) for d in str(n) if d != '0'])
def ok(sqr): return sqr > 0 and sqr%nzpd(sqr) == 0
print(list(filter(ok, (i*i for i in range(541))))) # _Michael S. Branicky_, Jul 23 2021
```

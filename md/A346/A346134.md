## Python
```Python
from sympy import prime, nextprime
def a(n):
    p, s, lim = 1, 0, prime(n)**2
    while s <= lim: p = nextprime(p); s += p
    return s - p
print([a(n) for n in range(1, 51)]) # _Michael S. Branicky_, Jul 05 2021
```

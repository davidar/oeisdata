## Python
```Python
from sympy import divisors
def auptorow(nn):
    for n in range(1, nn+1):
        yield from [d for d in divisors(n) if d%10 == n%10]
print([an for an in auptorow(50)]) # _Michael S. Branicky_, Jul 21 2021
```

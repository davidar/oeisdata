## Python
```Python
def ok(n):
    s = str(n*n)
    if len(s) < 4: return False
    return s[0] == s[1] == s[2] != s[3] and s[-1] == s[-2] == s[-3] != s[-4]
print(list(filter(ok, range(10**6)))) # _Michael S. Branicky_, Aug 06 2021
```

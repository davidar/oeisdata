## Python
```Python
from sympy import isprime
def a(n):
    if n > 1 and n%3 == 1: return 0
    k = 2
    while not isprime(n**k + n + 1): k += 1
    return k
print([a(n) for n in range(1, 96)]) # _Michael S. Branicky_, Jul 08 2021
```

## Python
```Python
def ok(n):
    s = str(n*n)
    if len(s) < 4: return False # there are no ok squares with < 4 digits
    return s[0] == s[1] != s[2] and s[-1] == s[-2] != s[-3]
print(list(filter(ok, range(3411)))) # _Michael S. Branicky_, Aug 03 2021
```

## Python
```Python
from sympy import nextprime, isprime
rec = -1; p1 = 1
while p1 < 1000000000:
    p = nextprime(p1); m = p; ct = 0
    while m > 2:
        if isprime(m): ct += 1
        m //= 2
    if ct > rec: print(p); rec = ct
    p1 = p
```

## Python
```Python
def ok(n):
  s = str(n*n)
  return len(s) > 4 and s[0] == s[1] == s[2] == s[3] != s[4] and s[-1] == s[-2] == s[-3] == s[-4] != s[-5]
print(list(filter(ok, range(3333333)))) # _Michael S. Branicky_, Aug 08 2021
```

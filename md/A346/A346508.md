## Python
```Python
def aupto(lim): return sorted(set(a*b//10 for a in range(11, 10*lim//11+2, 10) for b in range(a, 10*lim//a+2, 10) if a*b//10 <= lim))
print(aupto(318)) # _Michael S. Branicky_, Aug 21 2021
```

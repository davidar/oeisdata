## Python
```Python
from sympy.ntheory.digits import digits
def a(n):
    return sum(bi*(-1)**k for k, bi in enumerate(digits(n, 9)[1:][::-1]))
print([a(n) for n in range(105)]) # _Michael S. Branicky_, Jul 31 2021
```

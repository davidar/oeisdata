## Mathematica
```Mathematica
pQ[n_] := (f = FactorInteger[n]; 1/n + Sum[1/f[[i]][[1]], {i, Length[f]}] == 1)
```

## Python

from sympy import primefactors
A054377 = [n for n in range(2,10**5) if sum([n/p for p in primefactors(n)]) +1 == n] # _Chai Wah Wu_, Aug 20 2014

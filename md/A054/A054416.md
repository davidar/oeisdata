## Python
```Python
from sympy import isprime, prime
def afind(limit, startk=1):
    s = "90"*(startk-1)
    for k in range(startk, limit+1):
        if isprime(int(s+"91")):
            print(k, end=", ")
        s += "90"
afind(400) # _Michael S. Branicky_, Jan 13 2022
```

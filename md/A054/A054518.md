## Python
```Python
from sympy import isprime
def ok(n):
    r = int(str(n)[::-1])
    if r == n: return False
    return not isprime(n) and not isprime(r)
print([k for k in range(121) if ok(k)]) # _Michael S. Branicky_, Oct 24 2022
```

## Python
```Python
MAXTERM=10**9
cache, oldk = [MAXTERM], 1
for nleft in range(1,MAXTERM+1):
  for k in range(1,nleft+1):
    if k<cache[nleft-k]:
      mk=(k+2)//3
      break
  cache.append(mk)
  if mk>oldk:
    print(nleft)
    oldk=mk
```

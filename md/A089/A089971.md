## Python
```Python
from sympy import isprime, primerange
def aupto(limit):
    alst = []
    for p in primerange(2, limit+1):
        t = int(bin(p)[2:])
        if isprime(t): alst.append(t)
    return alst
print(aupto(2**11)) # _Michael S. Branicky_, Aug 19 2021
```

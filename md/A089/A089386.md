## Python
```Python
def a(n):
  if n == 0: return 1
  pow10, s = 10**n, ""
  for d in [8, 5, 4, 2]:
    while pow10%d == 0: pow10//=d; s += str(d)
  return int(s[::-1])
print([a(n) for n in range(18)]) # _Michael S. Branicky_, Jan 25 2021
```

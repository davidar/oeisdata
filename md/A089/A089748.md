## Python
```Python
from math import prod
from sympy import divisors
def ok(n): d = divisors(n)[:-1]; return n > 1 and (sum(d) + prod(d))%n == 0
print([k for k in range(10**5) if ok(k)]) # _Michael S. Branicky_, Oct 15 2021
```

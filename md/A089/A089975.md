## Python
```Python
from math import factorial as f
def T(n,k):
    return sum(f(n)*f(k)//f(n-2*i)//f(i)//f(k-n+i)//2**i for i in range(max(0,n-k),n//2+1))
[T(n-k,k) for n in range(21) for k in range(n+1)]
# _Robert FERREOL_, Oct 17 2017
```

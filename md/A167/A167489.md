## Python
```Python
def A167489(n):
  '''Product of run lengths in binary representation of n.'''
  p = 1
  b = n%2
  i = 0
  while (n != 0):
    n >>= 1
    i += 1
    if ((n%2) != b):
      p *= i
      i = 0
      b = n%2
  return(p)
```

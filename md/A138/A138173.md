## Python
```Python
from gmpy2 import iroot
def A138173(n):
    d, nd = 1, n**2
    while True:
        x = iroot(nd-1,3)[0]+1
        if x**3 < nd+d:
            return int(x)
        d *= 10
        nd *= 10 # _Chai Wah Wu_, May 24 2016
```

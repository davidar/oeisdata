## Python
```Python
from sympy import divisors
def a(n):
    s = str(n)
    return sum(1 for d in divisors(n, generator=True) if str(d) in s)
print([a(n) for n in range(1, 106)]) # _Michael S. Branicky_, Jul 11 2022
```

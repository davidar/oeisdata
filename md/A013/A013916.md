## Python
```Python
from sympy import isprime, prime
def aupto(lim):
  s = 0
  for k in range(1, lim+1):
    s += prime(k)
    if isprime(s): print(k, end=", ")
aupto(680) # _Michael S. Branicky_, Feb 28 2021
```

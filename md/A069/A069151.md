## Python
```Python
from sympy import isprime, nextprime
def afind(terms, verbose=False):
  n, p, pstr = 0, 2, "2"
  while n < terms:
    if isprime(int(pstr)): n += 1; print(n, int(pstr))
    p = nextprime(p); pstr += str(p)
afind(5) # _Michael S. Branicky_, Feb 23 2021
```

## Python
```Python
from sympy import isprime, primerange
def ok(p): return isprime(int('9'+str(p)+'9'))
def aupto(lim): return [p for p in primerange(2, lim+1) if ok(p)]
print(aupto(911)) # _Michael S. Branicky_, Feb 19 2021
```

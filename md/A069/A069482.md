## Python
```Python
from sympy import prime, primerange
def aupton(terms):
  p = list(primerange(1, prime(terms+1)+1))
  return [p[n+1]**2-p[n]**2 for n in range(terms)]
print(aupton(50)) # _Michael S. Branicky_, May 16 2021
```

## Python
```Python
from sympy import nextprime
def ok(n):
    t, targets = 100*n, [100*n+d for d in [1, 3, 7, 9, 13, 27]]
    return all((t:=nextprime(t)) == targets[i] for i in range(6))
print([k for k in range(10**6) if ok(k)]) # _Michael S. Branicky_, Jun 21 2022
```

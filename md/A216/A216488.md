## Python
```Python
def afind(limit):
    bkm1, bk = 2, 1
    for k in range(2, limit+1):
        bkm1, bk = bk, bkm1 + bk
        if set(str(bk)[-9:]) == set("123456789"): print(k, end=", ")
afind(10**6) # _Michael S. Branicky_, Jul 04 2021
```

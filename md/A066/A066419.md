## Python
```Python
from math import factorial
def sd(n): return sum(map(int, str(n)))
def ok(f): return f%sd(f) != 0
print([n for n in range(1, 715) if ok(factorial(n))]) # _Michael S. Branicky_, Jun 13 2021
```

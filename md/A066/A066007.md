## Python
```Python
def k(r): return (10**r - 1)//9 + r + 2
def a(n):
  r = 0
  while k(r+1) <= n: r += 1
  return int('1' + '0'*r + '9'*(n-r-1))
print([a(n) for n in range(1, 21)]) # _Michael S. Branicky_, Jan 19 2021
```

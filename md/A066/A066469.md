## Mathematica
```Mathematica
antid[n_] := Select[ Union[ Join[ Select[ Divisors[2n - 1], OddQ[ # ] && # != 1 & ], Select[ Divisors[2n + 1], OddQ[ # ] && # != 1 & ], 2n/Select[ Divisors[ 2*n], OddQ[ # ] && # != 1 &]]] }, # < n & ]]; Select[ Range[10^5], Length[ antid[ # ]] == 4 & ]
```

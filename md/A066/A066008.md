## Python
```Python
def sd(m): return sum(map(int, str(m)))
def is_harshad(m): return m > 0 and m%sd(m) == 0
def a(n): return sum(is_harshad(m) for m in range(10**(n-1), 10**n))
print([a(n) for n in range(1, 7)]) # _Michael S. Branicky_, Sep 23 2021
```

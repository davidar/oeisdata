## Python
```Python
from math import prod
def s(n): return sum(map(int, str(n)))
def p(n): return prod(map(int, str(n)))
for n in range(0, 10**6):
  if p(n)*s(n)==2*n:
    print(n) # _Mohammed Yaseen_, Jul 22 2022
```

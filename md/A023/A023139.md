## Python
```Python
from sympy import totient, n_order, divisors
def A023139(n):
    a, b = divmod(n,7)
    while not b:
        n = a
        a, b = divmod(n,7)
    return sum(totient(d)//n_order(7,d) for d in divisors(n,generator=True) if d>1)+1 # _Chai Wah Wu_, Apr 09 2024
```

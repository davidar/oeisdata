## Python
```Python
import sympy.ntheory as snt
n=2
while n > 1 and n < 10**6:
    n1=n+4
    n2=((n**2)+4)
    n3=((n**3)+4)
    ##Check if n1, n2 and n3 are also primes.
    if snt.isprime(n1)== True and snt.isprime(n2)== True and snt.isprime(n3)== True:
        print(n, end=', ')
    n=snt.nextprime(n)
```

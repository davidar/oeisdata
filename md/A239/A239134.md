## Maple
```Maple
f:= proc(n) local k;
  for k from 1 to 9 do
    if member(k,convert(n^k,base,10)) then return k fi
  od;
  FAIL
end proc:
map(f, [$1..100]); # _Robert Israel_, Sep 16 2024
```

```Python
def Sub(x):
  for n in range(10**3):
    if str(x**n).find(str(n)) > -1:
      return n
x = 1
while x < 10**3:
  print(Sub(x))
  x += 1
```

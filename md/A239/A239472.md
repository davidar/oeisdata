## Maple
```Maple
f:= proc(n) local x, r, k;
  r:= 0; x:= 2^n;
  for k from 3 do
    r:= r + (k-1)^n;
    x:= k^n - r;
    if x < 2 then return 0 fi;
    if isprime(x) then return k fi;
  od
end proc:
f(1):= 2:
map(f, [$1..100]); # _Robert Israel_, Jan 03 2024
```

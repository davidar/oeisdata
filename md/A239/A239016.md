## Python
```Python
def ok(n):
    s = str(n)
    if "".join(sorted(s)) == s: return True
    return all(n <= int(s[i:] + s[:i]) for i in range(1, len(s)))
print(list(filter(ok, range(133)))) # _Michael S. Branicky_, Aug 21 2021
```

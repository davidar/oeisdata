## Python
```Python
import sympy
prpr = 2
prev = 3
for i in range(5,1000,2):
    if sympy.isprime(i):
        x = i - prev
        y = prev - prpr
        if x%y==0 or y%x==0: print str(prev)+',',
        prpr = prev
        prev = i
```

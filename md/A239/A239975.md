## Python
```Python
from sympy.ntheory.primetest import is_square
def a(n):
    k = 1
    while not is_square(n**2 + (n+k)**2): k += 1
    return k
print([a(n) for n in range(5, 70)]) # _Michael S. Branicky_, Jul 02 2021
```

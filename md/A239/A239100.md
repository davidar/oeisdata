## Python
```Python
def A239100(n):
    if n <= 8: return (0,1,1,2,3,4,6,8)[n-1]
    r, b = divmod(n,3)
    return ((107<<r-3)//7-2 if b==1 else ((17<<r)-6)//7-1) if b else (43<<r-2)//7-2 # _Chai Wah Wu_, Mar 28 2023
```

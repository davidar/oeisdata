## Python
```Python
from sympy import primepi
def A106092(n):
    m, k = n, 1+(n<<1)-primepi(n)-(n>>1)
    while m != k:
        m, k = k, 1+n+k-primepi(k)-(k>>1)
    return m # _Chai Wah Wu_, Jul 31 2024
```

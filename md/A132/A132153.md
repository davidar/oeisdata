## Python
```Python
from sympy import prevprime, integer_nthroot
def a(n): return prevprime(integer_nthroot(10**n, 2)[0]+1)
print([a(n) for n in range(1, 28)]) # _Michael S. Branicky_, Dec 23 2021
```

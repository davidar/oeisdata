## Python
```Python
from math import factorial
def a(n):
    nf, t = factorial(n), 0
    for i in range(n+1, nf+1):
        if nf%i == 0: t += 1
        if t == n: return i
print([a(n) for n in range(4, 66)]) # _Michael S. Branicky_, Apr 25 2022 after _David A. Corneth_
```

## Python
```Python
def sd(n): return sum(map(int, str(n)))
def aupton(terms):
    alst, s = [1], 1
    for n in range(2, terms+1): alst.append(sd(s**2)); s += alst[-1]
    return alst
print(aupton(67)) # _Michael S. Branicky_, Jun 15 2021
```

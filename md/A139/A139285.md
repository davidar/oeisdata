# Smallest nontrivial number m that when expressed in base n and then interpreted in base 10, is a multiple of the original number\.

## Python
```Python
m,M=2,11
while M%m>0:
    m,M=m+1,10*M+1
print(1,m)
for n in range(2,10):
    m,M,d=n,10,10-n
    while M%m>0:
        m=m+1
        u,s=m,0
        while u%n==0:
            u,s=u//n,10*s+d
        M=M+1+s
    print(n,m)
```

## Maple
```Maple
f:=proc(n) local k, v, r;
  v:= 1;
  for k from 1 do
    v:= v*n;
    r:= v/10^(ilog10(v));
    if r < 101/100 or r >= 99/10 then return k fi
  od
end proc:
map(f, [$1..100]);
```
## Python
```Python
def A306540(n):
    if n == 1 or n == 10:
        return 1
    k, nk = 1, n
    while True:
        s = str(nk)
        if s[:2] == '99' or s[:3] == '100':
            return k
        k += 1
        nk *= n # _Chai Wah Wu_, Feb 22 2019
```

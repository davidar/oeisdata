## Python
```Python
seq = [0]
for n in range(1, 1387):
  num_blanks = seq.count(-1)
  if num_blanks >= n: seq[[index for index, value in enumerate(seq) if value == -1][num_blanks - n]] = n
  else: seq += [n] + [-1] * n
print(seq[:100])
```

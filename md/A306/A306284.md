## Maple
```Maple
f:= proc(n) local S, x,t;
S:= {0}:
for x from 1 do
  t:= x^2 mod n;
  if member(t,S) then return x
    else S:= S union {t}
  fi
od
end proc:
map(f, [$1..100]); # _Robert Israel_, Feb 05 2019
```

## Python
```Python
seq = [0, 1]
for n in range(2, 281):
  for k in range(len(seq) - 1, 0, -1):
    if seq[k] + seq[k - 1] == n:
      seq.insert(k, n)
      break
  else: seq += [n]
print(seq[:100])
```

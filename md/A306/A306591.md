## Python
```Python
from sympy import prime
from fractions import Fraction
def a(n): return (Fraction(1, 2) - Fraction(1, (prime(n)+1))).denominator
print([a(n) for n in range(1, 66)]) # _Michael S. Branicky_, Jun 04 2021
```

## Maple
```Maple
cat3:= proc(x)
  local t;
  t:= 10^length(x+2);
  x*(1 + t*(1+10^length(x+1)))+t+2
end proc:
f:= proc(x) local q,a,b;
  q:= cat3(x);
  a:= (q/x)::integer;
  b:= (q/(x+1))::integer;
  if a and b then return q elif not(a) and not(b) then return NULL fi;
  if (q/(x+2))::integer then q else NULL fi
end proc:
map(f, [$1..1000]);
```

## Python
```Python
from sympy import isprime, primerange
def ok(p): s = str(p); return isprime(int(s[0]) + int(s[-1]))
def aupto(limit): return [p for p in primerange(1, limit+1) if ok(p)]
print(aupto(691)) # _Michael S. Branicky_, Nov 23 2021
```

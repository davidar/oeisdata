## Python
```Python
def ispal(n):
    return n==int(str(n)[::-1])
def A088000(n):
    s=0
    for i in range(1, n+1):
        if n%i==0 and ispal(i):
             s+=i
    return s
print([A088000(n) for n in range(1,30)]) # _Indranil Ghosh_, Feb 10 2017
```

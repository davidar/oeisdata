## Python
```Python
from __future__ import division
from sympy import divisors, divisor_sigma, gcd
def A267310(n):
    m = sum(d*divisor_sigma(d)**(n//d)  for d in divisors(n, generator=True))
    return m//gcd(m,n) # _Chai Wah Wu_, Oct 02 2017
```

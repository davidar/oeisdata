## Sage
```Sage
def A267574(n):
    gen = (p for p in srange(2, 10^n) if p.is_prime_power() and not p.is_prime())
    return sum(1 for _ in gen)
print([A267574(n) for n in range(1, 7)])  # _Peter Luschny_, Sep 16 2023
```

## Python
```Python
def aupton(terms):
    alst, seen = [1, 2], {1, 2}
    for n in range(2, terms, 2):
        anp1 = alst[-1] + 1
        while anp1 in seen: anp1 += 1
        an = alst[n-1] + anp1
        alst, seen = alst + [an, anp1], seen | {an, anp1}
    return alst[:terms]
print(aupton(67)) # _Michael S. Branicky_, Jun 18 2021
```

## Python
```Python
from sympy import nextprime
def a(n):
  s = p = nextprime(10**(n-1))
  for n in range(n-1): p = nextprime(p); s += p
  return s
print([a(n) for n in range(1, 21)]) # _Michael S. Branicky_, Feb 24 2021
```

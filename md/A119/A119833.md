## Python
```Python
from sympy import isprime, nextprime
def afind(limit):
    p = 2
    twoprimorialp = 4
    while p <= limit:
        if isprime(twoprimorialp - 1):
            print(p, end=", ")
        p = nextprime(p)
        twoprimorialp *= p
afind(1000) # _Michael S. Branicky_, Jan 08 2022
```

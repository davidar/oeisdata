## Python
```Python
def a(n):
    m = n
    b = bin(m)[2:]
    while len(b) != 2*b.count("1"):
        m += n
        b = bin(m)[2:]
    return m
print([a(n) for n in range(1, 61)]) # _Michael S. Branicky_, May 15 2022
```

## Python
```Python
from math import isqrt
from sympy import mobius
def A143658(n):
    m = 1<<n
    return sum(mobius(k)*(m//k**2) for k in range(1,isqrt(m)+1)) # _Chai Wah Wu_, Jun 01 2024
```

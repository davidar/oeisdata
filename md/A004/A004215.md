## Python
```Python
def valuation(n, b):
    v = 0
    while n > 1 and n%b == 0: n //= b; v += 1
    return v
def ok(n): return n//4**valuation(n, 4)%8 == 7 # after _M. F. Hasler_
print(list(filter(ok, range(344)))) # _Michael S. Branicky_, Jul 15 2021
```

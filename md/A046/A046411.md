## Python
```Python
from sympy import isprime, factorint
def ok(n):
    f = factorint(n)
    if sum(e for e in f.values()) < 2: return False
    return isprime(int("".join(str(p)*e for p, e in f.items())))
print(list(filter(ok, range(2, 262)))) # _Michael S. Branicky_, Jun 12 2021
```

## Python
```Python
from sympy import isprime
def afind(limit, startk=1):
    k, pow10 = startk, 10**startk
    for k in range(startk, limit+1):
        if isprime(k*(pow10 - 1) + 1): print(k, end=", ")
        k += 1
        pow10 *= 10
afind(500) # _Michael S. Branicky_, Aug 26 2021
```

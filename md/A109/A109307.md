## Python
```Python
from sympy import factorint
def issemiprime(n): return sum(factorint(n).values()) == 2
def ok(n): return all(issemiprime(n**2 + (n+k)**2) for k in [1, -1])
print([k for k in range(296) if ok(k)]) # _Michael S. Branicky_, Nov 16 2021
```

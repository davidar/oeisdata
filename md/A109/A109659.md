## Python
```Python
from sympy import divisor_sigma
def ok(n): return n and (sum(map(int, str(divisor_sigma(n, 1)**n)))%n == 0)
print([k for k in range(3000) if ok(k)]) # _Michael S. Branicky_, Jan 27 2023
```

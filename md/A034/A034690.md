## Python
```Python
from sympy import divisors
def sd(n): return sum(map(int, str(n)))
def a(n): return sum(sd(d) for d in divisors(n))
print([a(n) for n in range(1, 76)]) # _Michael S. Branicky_, Oct 06 2021
```

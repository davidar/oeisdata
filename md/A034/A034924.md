## Python
```Python
from sympy import isprime
def afind(limit, startat=1):
  pow47 = 47**startat
  for k in range(startat, limit+1):
    if isprime(pow47 - 46): print(k, end=", ")
    pow47 *= 47
afind(900) # _Michael S. Branicky_, May 11 2021
```

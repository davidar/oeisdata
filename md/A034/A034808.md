## Python
```Python
from sympy import isprime, prevprime
def aupto(m):
  return [k for k in range(3, m+1) if isprime(int(str(prevprime(k))+str(k)))]
print(aupto(513)) # _Michael S. Branicky_, Mar 09 2021
```

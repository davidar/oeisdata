## Python
```Python
from sympy import primerange
def aupto(lim):
    pitri = (p*(p+1)//2 for p in primerange(2, int((2*lim)**.5)+1))
    return [t for t in pitri if t%2 == 0]
print(aupto(90000)) # _Michael S. Branicky_, Jun 28 2021
```

## Python
```Python
from sympy import isprime
def a(n):
    Rn = (10**n-1)//9
    return sum(1 for i in range(n-1) if isprime(Rn-10**i))
print([a(n) for n in range(1, 101)]) # _Michael S. Branicky_, Nov 04 2023
```

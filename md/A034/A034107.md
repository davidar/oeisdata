## Python
```Python
from math import sqrt
def a(n):
    k = (n + 2)**2
    while int(10*sqrt(k))%10 != 1: k += 1
    return k
print([a(n) for n in range(1, 50)]) # _Michael S. Branicky_, Oct 17 2021
```

## Python
```Python
from sympy import divisors, isprime
def A290126(n):
    i = 1
    while len(divisors(i)) < n or not isprime(sum(divisors(i)[-n:])):
        i += 1
    return i # _Chai Wah Wu_, Aug 05 2017
```

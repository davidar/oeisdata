## Python
```Python
from sympy.core.cache import cacheit
@cacheit
def A(n, k):
    return 1 if n==0 else sum(A(j, k)*A(n - j - 1, k)*(-k)**j for j in range(n))
for d in range(16): print([A(n, d - n) for n in range(d + 1)]) # _Indranil Ghosh_, Aug 13 2017
```

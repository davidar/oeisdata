## Python
```Python
from sympy import divisor_sigma
l=[]
def T(n, k):
    return 0 if n==0 or k==0 else divisor_sigma(k + n) - divisor_sigma(k) - n
for n in range(11): l+=[T(k, n - k + 1) for k in range(n + 1)]
print(l) # _Indranil Ghosh_, Jul 21 2017
```

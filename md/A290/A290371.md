## Python
```Python
from sympy import floor, exp, sqrt, log
def f(n): return floor(n/exp(sqrt(log(n))))
print([n for n in range(2, 1001) if f(n) - f(n - 1) == 1]) # _Indranil Ghosh_, Jul 28 2017
```

## Python
```Python
# see link for faster version
def aupto(limit):
  alst = []
  for k in range(1, limit+1):
    if set(str(k*k)) <= set("235"): alst.append(k)
  return alst
print(aupto(2*10**6)) # _Michael S. Branicky_, May 15 2021
```

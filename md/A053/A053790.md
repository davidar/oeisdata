## Python
```Python
from sympy import isprime, nextprime
def aupto(limit):
    alst, s, p = [], 2, 2
    while s < limit:
        if not isprime(s): alst.append(s)
        p = nextprime(p)
        s += p
    return alst
print(aupto(6000)) # _Michael S. Branicky_, Oct 28 2021
```

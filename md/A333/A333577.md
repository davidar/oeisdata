## Python
```Python
from sympy import prime, nextprime, mod_inverse
def A333577(n):
    if n == 2:
        return 0
    p = prime(n)
    q, r = nextprime(p), 10**len(str(p))
    return p*q*mod_inverse(q,r) % (q*r) # _Chai Wah Wu_, Mar 31 2020
```

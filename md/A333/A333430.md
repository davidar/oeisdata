## Python
```Python
def a(n):
    if n == 1: return 1
    m = 0
    for c in range(1, n+1):
        m += 1
        while not (pow(n, m, m) + 1)%m == 0: m += 1
    return m
print([a(n) for n in range(1, 15)]) # _Michael S. Branicky_, Aug 28 2021
```

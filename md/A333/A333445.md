## Python
```Python
def T(n,k): # T(n,k) for A333445
    c, l = 1, list(range(1,k*n+1,k))
    lt = list(l)
    for i in range(n):
        for j in range(1,k):
            lt[i] += l[i]+j
        c *= lt[i]
    return c
```

## Python
```Python
n=100
t=[]
prev=0
A333443=[]
for i in range(1,n+1):
    t.append(i*i)
for j in range(n**2):
    n=0
    for k in t[:j+1]:
        if j-k in t and k<=j-k:
            n=n+1
            if n>1:
                if j-prev==1:
                    A333443.append(j-1)
                prev=j
```

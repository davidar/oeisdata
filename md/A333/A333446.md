## Python
```Python
def T(n,k): # T(n,k) for A333446
    c, l = 0, list(range(1,k*n+1,k))
    lt = list(l)
    for i in range(n):
        for j in range(1,k):
            lt[i] *= l[i]+j
        c += lt[i]
    return c
```

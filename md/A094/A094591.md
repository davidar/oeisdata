## Python
```Python
def aupton(nn):
    alst = [1]
    for n in range(1, nn+1):
        alst.append(n + max(k for k in alst if k <= n))
    return alst
print(aupton(65)) # _Michael S. Branicky_, Oct 28 2021
```

## Python
```Python
from math import factorial
def a(n):
    fn, k, p = factorial(n), 1, n+1
    while fn >= p: k += 1; p *= (n+k)
    return k
print([a(n) for n in range(1, 75)]) # _Michael S. Branicky_, Apr 11 2022
```

# Numbers n such that \(2n\)\!\! mod \(2n\-1\)\!\! is greater than \(2n\+2\)\!\! mod \(2n\+1\)\!\!\.
https://oeis.org/A232620
## JSON
```JSON

```
## Python
```Python
o=e=1  # odd, even
prev=0
for n in range(1,1000000000,2):
  o*=n
  e*=n+1
  cur = e%o
  if prev>cur: print(str(n//2), end=', ')
  prev=cur
```

## Python
```Python
import math
for n in range(77):
  n2 = n*n
  y=1
  for k in range(1,10000001):
    sum = n2 + k*(k+1)//2
    r = int(math.sqrt(sum))
    if r*r == sum:
      print(str(k), end=',')
      y=0
      break
  if y: print('-', end=',')
```

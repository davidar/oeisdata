## Python
```Python
import math
for n in range(77):
  tn = n*(n+1)//2
  for k in range(1, n+9):
    sum = tn + k*(k+1)//2
    r = int(math.sqrt(sum))
    if r*r == sum:
      print(str(k), end=',')
      break
```

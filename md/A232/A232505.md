## Python
```Python
from sympy.ntheory.residue_ntheory import quadratic_residues as q
from sympy import prime
def a(n): return sum(i for i in q(prime(n)) if i%2)
print([a(n) for n in range(1, 101)]) # _Indranil Ghosh_, May 14 2017
```

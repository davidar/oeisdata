## Python
```Python
from sympy.ntheory import totient, divisors
def T(n,k): return sum(totient(d)*k**(n//d) for d in divisors(n))//n
for n in range(1, 16):
    print([T(k, n - k + 1) for k in range(1, n + 1)]) # _Indranil Ghosh_, Mar 25 2017
```

## Python
```Python
from sympy import divisors, factorint
def a(n):
    f=factorint(n)
    l=[f[i] for i in f]
    return min(i for i in divisors(n) if i not in l)
print([a(n) for n in range(1, 97)]) # _Indranil Ghosh_, May 16 2017
```

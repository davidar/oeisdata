## Python
```Python
from sympy import prime, nextprime
def a(n):
    p = prime(3*n - 2); q = nextprime(p); r = nextprime(q)
    return p + q + r
print([a(n) for n in range(1, 49)]) # _Michael S. Branicky_, Oct 31 2021
```

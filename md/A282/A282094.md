## Python
```Python
from sympy import factorint
def is_cube(n):
    if n==0: return True
    return all(i%3==0 for i in factorint(n).values())
def ok(n):
    return any(is_cube(x**2 + n**2) for x in range(n + 1))
print([n for n in range(501) if ok(n)]) # _Indranil Ghosh_, Jun 30 2017
```

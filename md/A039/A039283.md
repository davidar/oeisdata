## Python
```Python
from sympy.ntheory import count_digits
def ok(n): c = count_digits(n, 5); return 0 in c and 1 in c and c[0] == c[1]
print(list(filter(ok, range(291)))) # _Michael S. Branicky_, May 27 2021
```

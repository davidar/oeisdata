## Python
```Python
from sympy import primerange
def a(n):
    target = str(n)
    return sum(1 for p in primerange(1, 10**n) if target not in str(p))
print([a(n) for n in range(7)]) # _Michael S. Branicky_, Oct 30 2022
```

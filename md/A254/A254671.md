## Python
```Python
def aupto(limit):
    cands = range(2, limit//3+1)
    nums = [x*y+x+y for i, y in enumerate(cands) for x in cands[i:]]
    return sorted(set(k for k in nums if k <= limit))
print(aupto(113)) # _Michael S. Branicky_, Aug 11 2021
```

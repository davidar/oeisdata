## Python
```Python
def a(n):
    s, pow3, lim = set(), 1, 10**n
    while pow3 < lim:
        for j in range((lim-pow3).bit_length()):
            s.add(2**j + pow3)
        pow3 *= 3
    return len(s)
print([a(n) for n in range(1, 46)]) # _Michael S. Branicky_, Jul 29 2021
```

## Python
```Python
from sympy import isprime
def afind(limit):
  k, twok = 0, 1
  while k <= limit:
    if isprime(int(str(twok) + str(k))): print(k, end = ", ")
    k, twok = k+1, twok*2
afind(2000) # _Michael S. Branicky_, Mar 23 2021
```

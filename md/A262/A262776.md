## Python
```Python
from gmpy2 import fac, fib
def A262776(n):
    if n < 2:
        return 0
    a, b, m = 0, 1, fac(fib(n))
    for i in range(fac(n)-1):
        b, a = (b+a) % m, b
    return int(b) # _Chai Wah Wu_, Oct 03 2015
```

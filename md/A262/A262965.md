## Python
```Python
from sympy import prime, primefactors
def a(n):
    k, target = 2, prime(n)
    while k%sum(primefactors(k)) != target: k += 1
    return k
print([a(n) for n in range(1, 53)]) # _Michael S. Branicky_, Dec 10 2021
```

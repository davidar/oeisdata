## Python
```Python
from sympy import isprime
def afind(limit, startk=1):
    for k in range(startk, limit+1):
        if isprime(k*(k-1)**k - (k-1)*k**(k-1) - 1):
            print(k, end=", ")
afind(200) # _Michael S. Branicky_, Jan 10 2022
```

## Python
```Python
import math
for x in range(2, 1000):
  false_witnesses = 0
  relatively_prime_values = 0
  for y in range(x):
    if math.gcd(y, x) == 1:
      relatively_prime_values += 1
    if (pow(y, x-1, x) == 1):
      false_witnesses += 1
  if false_witnesses * 2 == relatively_prime_values:
    print(x, "is a Fermat Half-Prime")
```

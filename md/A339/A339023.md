## Python
```Python
def digitalroot(n):
    return 0 if n == 0 else (n-1)%9 + 1
def a(n):
    return int(''.join([str(digitalroot(n*int(d))) for d in str(n)]))
for n in range(0, 68):
    print(a(n), end=', ')
```

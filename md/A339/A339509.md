## Python
```Python
from sympy import primefactors
def test(n):
    if n<2: return n
    return max(primefactors(n))
def a(n):
    tests = [test(i) for i in range(n+1)]
    return sum(2**tests.count(v)-1 for v in set(tests))
print([a(n) for n in range(57)]) # _Michael S. Branicky_, Dec 11 2020
```

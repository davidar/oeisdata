## Python
```Python
from collections import Counter
a = [0]
for i in range(1, 10000):
    counts = Counter(str(bin(i))[2:])
    if counts['0'] > counts['1']:
        a.append(a[-1] - counts['0'])
    elif counts['1'] > counts['0']:
        a.append(a[-1] + counts['1'])
    else:
        a.append(a[-1])
print(a)
```

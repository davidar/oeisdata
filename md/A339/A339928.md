## Python
```Python
from sympy import factorial, isprime
def ok(n):
    fn = factorial(n)
    while fn > 0 and fn%2 == 0: fn //= 10
    return fn > 0 and isprime(fn)
print(list(filter(ok, range(200)))) # _Michael S. Branicky_, Jun 07 2021
```

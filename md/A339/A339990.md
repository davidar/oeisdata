## Python
```Python
def aupto(lim):
  ocs = [k**3 for k in range(1, int(lim**(1/3)+2), 2)]
  return sorted(set(c1+c2 for c1 in ocs for c2 in ocs if c1+c2 <= lim))
print(aupto(7202)) # _Michael S. Branicky_, Feb 28 2021
```

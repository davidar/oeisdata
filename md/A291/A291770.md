## Python
```Python
from sympy.ntheory.factor_ import digits
def a(n):
    k=digits(n, 3)[1:]
    return int("".join('1' if i==0 else '0' for i in k), 2)
print([a(n) for n in range(1, 111)]) # _Indranil Ghosh_, Sep 21 2017
```

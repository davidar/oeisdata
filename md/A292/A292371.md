## Python
```Python
from sympy.ntheory.factor_ import digits
def a(n):
    k=digits(n, 4)[1:]
    return 0 if n==0 else int("".join('1' if i==1 else '0' for i in k), 2)
print([a(n) for n in range(116)]) # _Indranil Ghosh_, Sep 21 2017
```

## Python
```Python
from sympy.core.cache import cacheit
@cacheit
def a(n): return n*2 + 1 if n < 2 else 4*n*a(n - 1) + sum([a(k)*a(n - k - 1) for k in range(1, n - 1)])
[a(n) for n in range(21)]
```

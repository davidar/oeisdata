## Python
```Python
from sympy import integer_nthroot
def A077110(n):
    n2 = n**2
    a = integer_nthroot(n2,3)[0]
    a2, a3 = a**3, (a+1)**3
    return a3 if a3+a2-2*n2 < 0 else a2 # _Chai Wah Wu_, Sep 24 2021
```

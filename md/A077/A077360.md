## Python
```Python
from sympy import isprime, primerange
for p in primerange(100, 1440):
    if isprime(int(str(p)[1:-1])) and isprime(int(str(p)[0]+str(p)[-1])):
        print(p)  # _Jason Yuen_, Apr 21 2024
```

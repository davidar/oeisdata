## Python
```Python
from sympy import divisors, totient, mobius, prod
def a(n): return prod((2**totient(d) - 1)**mobius(n//d) for d in divisors(n)).denominator()
print([a(n) for n in range(1, 51)]) # _Indranil Ghosh_, Apr 14 2017
```

## Python
```Python
from itertools import islice
from collections import Counter
def afull(): yield from (x**2 for x in range(10**10) if set(Counter(str(x**2)).values()) == {2})
print(list(islice(afull(), 47))) # _Michael S. Branicky_, May 12 2023
```

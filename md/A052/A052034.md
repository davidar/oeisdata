## Python
```Python
from sympy import isprime, primerange
def ok(p): return isprime(sum(int(d)**2 for d in str(p)))
def aupto(limit): return [p for p in primerange(1, limit+1) if ok(p)]
print(aupto(1013)) # _Michael S. Branicky_, Nov 23 2021
```

```Python
def get_num_orbits(n: int) -> int:
    orbits = 0
    mod = 10*n - 1
    seen = set()
    for i in range(1, mod):
        if i not in seen:
            seen.add(i)
            orbits += 1
        x = 10*i % mod
        while x != i:
            seen.add(x)
            x = 10*x % mod
    return orbits
```

## Python
```Python
def ok(n):
    w = bin(n)[2:]
    return any(any(w[:k] < w[i:i+k] for i in range(1, len(w)-k+1)) for k in range(2, len(w)))
print([k for k in range(157) if ok(k)]) # _Michael S. Branicky_, Nov 09 2023
```

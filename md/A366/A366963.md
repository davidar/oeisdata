## Mathematica
```Mathematica

```
## Python
```Python
def ok(n): return max(d:=list(map(int, str(n))))-min(d) == 6
print([k for k in range(400) if ok(k)]) # _Michael S. Branicky_, Oct 30 2023
```


## Python
```Python
from math import gcd
from sympy.utilities.iterables import partitions
def A366844(n): return sum(1 for p in partitions(n) if all(d==1 for d in p.values()) and all(d&1 for d in p) and gcd(*p)==1) # _Chai Wah Wu_, Oct 30 2023
```

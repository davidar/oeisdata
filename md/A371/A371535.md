## Python
```Python
def a(n):
    b, s = bin(n)[2:], str(n)
    return int("".join(d for i, d in enumerate(s) if b[i]=="1"))
print([a(n) for n in range(1, 81)]) # _Michael S. Branicky_, Mar 26 2024
```

## Sage
```Sage
def v2(n): return valuation(2*n, 2)
def a(n):
    if n == 0: return 0
    return 4^n*(v2(n) + 1) if n % 2 else 2*4^n*v2(n) - v2(n//2)
print([a(n) for n in range(0, 25)])
```

## Maple
```Maple
f:= proc(n) local k,s,m;
  s:= (-1 + n - n^2 + n^n)/((-1 + n)^2);
  m:= n^n;
  for k from 1 do if isprime(k*m+s) then return k*m+s fi od
end proc:
map(f, [$2..20]); # _Robert Israel_, Apr 28 2023
```

## Python
```Python
from sympy import isprime
from itertools import count
def a(n):
    c, d = n**n, (-1 + n - n**2 + n**n)//((-1 + n)**2)
    return next(ak for ak in count(c+d, c) if isprime(ak))
print([a(n) for n in range(2, 23)]) # _Michael S. Branicky_, Apr 24 2023
```

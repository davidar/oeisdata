## Python
```Python
from sympy import divisors
def ok(n): return n == 0 or any(set(str(x)) == set(str(n//x)) for x in divisors(n))
print([k for k in range(1025) if ok(k)]) # _Michael S. Branicky_, Apr 23 2023
```

## Python
```Python
a = [2]
split = []
for i in range(100):
    split += [int(j) for j in str(abs(a[i]))]
    if split[i] % 2 == 0:
        a.append((a[i] + split[i]))
    else:
        a.append((a[i] - split[i]))
print(a)
```

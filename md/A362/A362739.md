## Maple
```Maple
f:= proc(n) local P,p;
  P:= numtheory:-factorset(n) minus {2};
  if P = {} then return n/2 fi;
  p:= min(P);
  n*(p-1)/(2*p)
end proc:
map(f, [$3..100]); # _Robert Israel_, May 01 2023
```
## Sage
```Sage
def a(n):
    F = list(factor(n))
    if F[0][0] == 2:
        if len(F) == 1:
            return n / 2
        del F[0]
    m = F[0][0]
    return n * (m - 1) / (2 * m)
```

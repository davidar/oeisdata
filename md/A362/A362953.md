## Python
```Python
aupto = 103000
A362953 = []
A000578 = set(cu**3 for cu in range(0, int(aupto**(1/3)+3)))
for n in range(0,aupto+1):
    if n + sum(int(digit)**3 for digit in str(n)) in A000578: A362953.append(n)
print(A362953) # _Karl-Heinz Hofmann_, May 24 2023
```

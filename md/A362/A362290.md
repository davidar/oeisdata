## Python
```Python
from math import isqrt
a,M = 3,[]
for N in range(200):
   k = 4 * N + 1
   G = isqrt(k)
   a += 2 * G - (G ** 2 == k)
   M.append(a)
print(M) # _Oleg Sorokin_, Apr 22 2023
```

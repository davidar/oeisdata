## Python
```Python
aupto   = 7869300
A362954 = []
A000583 = set(fp**4 for fp in range(0, int(aupto**(1/4)+3)))
for n in range(0, aupto+1):
    if n + sum(int(digit)**4 for digit in str(n)) in A000583: A362954.append(n)
```

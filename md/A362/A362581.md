## Python
```Python
from itertools import accumulate
from math import comb
def A362581(n):
    if n <= 1: return n+1
    blist = (0,1)
    for _ in range(n-1):
        blist = tuple(accumulate(reversed(blist),initial=0))
    return blist[-1]**2*comb(n<<1,n) # _Chai Wah Wu_, Apr 25 2023
```

## Python
```Python
from sympy import primerange; a0 = 0; L = []
def ct(m1, m2): return len(list(primerange(m1, m2)))
for n in range(1,75): s = n*n; a = a0+ct(s,s+n+1)-ct(s-n+1,s); L.append(a); a0 = a
print(*L, sep = ", ")
```

## Python
```Python
from math import isqrt
def aupto(lim): return sorted(set(t for t in (k + int(str(k)[::-1]) for k in range(1, lim+1)) if t <= lim and isqrt(t)**2 == t))
print(aupto(10**7)) # _Michael S. Branicky_, Dec 25 2022
```

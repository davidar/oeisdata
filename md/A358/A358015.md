## Sage
```Sage
from sage.arith.misc import dedekind_psi
def A358015(n):
    k = valuation(n, 2)
    j = k if mod(n, 4) == 0 else 0
    return dedekind_psi(n*2^(-k))*2^(j-1)
print([A358015(n) for n in range(3, 76)]) # _Peter Luschny_, Oct 26 2022
```

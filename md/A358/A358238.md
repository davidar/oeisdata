## Maple
```Maple
f:= proc(n) local p, S, q;
  p:= ithprime(n);
  S:= {$1..p-1};
  q:= p;
  do
    q:= nextprime(q);
    S:= S minus {q mod p};
    if S = {} then return q fi;
  od;
end proc:
map(f, [$1..100]);
```

## Python
```Python
from sympy import prime, nextprime
def a(n):
    pn = p = prime(n); res = set()
    while True:
        res.add(p%pn)
        if len(res) == pn: return p
        p = nextprime(p)
print([a(n) for n in range(1, 54)]) # _Michael S. Branicky_, Jan 31 2023
```

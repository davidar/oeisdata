## Python
```Python
a_n = 2
result = [2]
for n in range(30):
    temp = []
    for i in range(1, a_n):
        if a_n % i == 0:
            if (i % 2 != 0) and (i != a_n):
                temp.append(i)
    result.append(a_n + max(temp))
    a_n = a_n + max(temp)
print(result)
```

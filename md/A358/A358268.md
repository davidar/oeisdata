## Maple
```Maple
f:= proc(n) local k;
    for k from 1 by 2 do
      if convert(convert(k^n,base,2),`+`) = n*convert(convert(k,base,2),`+`) then return k
      fi
    od
end proc:
map(f, [$1..50]);
```

## Python
```Python
def a(n):
    k = 1
    while bin(k**n).count("1") != n*bin(k).count("1"): k += 2
    return k
print([a(n) for n in range(1, 66)]) # _Michael S. Branicky_, Nov 06 2022
```

## Python
```Python
from math import prod
def sp(n): d = list(map(int, str(n))); return sum(d) + prod(d)
def ok(n): return any(m + sp(m) == n for m in range(n))
print([k for k in range(111) if ok(k)]) # _Michael S. Branicky_, Dec 19 2022
```

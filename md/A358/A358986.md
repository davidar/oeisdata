## Python
```Python
def A358986(n):
    kset = set()
    for i in range(1,10**(n-1)):
        for j in range(int((s:=str(i))[0])+1):
            kset.add(10*i+j+int(str(j)+s[::-1]))
    return 10+len(kset) # _Chai Wah Wu_, Dec 09 2022
```

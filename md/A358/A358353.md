## Python
```Python
from math import prod
def sp(n): d = list(map(int, str(n))); return sum(d) + prod(d)
def ok(n): return all(m + sp(m) != n for m in range(n+1))
print([k for k in range(181) if ok(k)]) # _Michael S. Branicky_, Dec 19 2022
```

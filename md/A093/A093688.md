## Python
```Python
from sympy import divisors
def c(n): return n == 1 or bin(n).count("1")&1 == 0
def ok(n): return n > 0 and all(c(d) for d in divisors(n, generator=True))
print([k for k in range(480) if ok(k)]) # _Michael S. Branicky_, Jul 24 2022
```

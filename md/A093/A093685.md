## Python
```Python
def aupto(limit):
  kfact, alst = 1, [0]
  for k in range(1, limit+1):
    kb, kfact = bin(k)[2:], kfact * k
    kfactb = bin(kfact)[2:]
    if kb not in kfactb: alst.append(k)
  return alst
print(aupto(25000)) # _Michael S. Branicky_, May 01 2021
```

## Python
```Python
from sympy import divisors
def ok(n):
    b = bin(n)[2:]
    return not all(bin(d)[2:] in b for d in divisors(n, generator=True))
print([k for k in range(119) if ok(k)]) # _Michael S. Branicky_, Jun 05 2022
```

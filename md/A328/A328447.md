## Maple
```Maple
f:= proc(n) local L,i,t;
  L:= sort(convert(n,base,10));
  if L[1]=0 then
    t:= numboccur(0,L)+1;
    L:= [L[t],op(L[1..t-1]),op(L[t+1..-1])];
  fi;
  add(L[-i]*10^(i-1),i=1..nops(L))
end proc:
f(0):= 0:
map(f, [$0..100]);
```

def A328447(n):
    if n == 0: return 0
    s = str(n)
    l, s = len(s), ''.join(sorted(s.replace('0','')))
    return int(s[0]+'0'*(l-len(s))+s[1:]) # _Chai Wah Wu_, Dec 06 2021

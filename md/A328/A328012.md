## Python
```Python
n, a = 0, 1
while n < 58:
    aa, k = a, 0
    while aa > 0:
        d, aa = aa%2, aa//2
        if d == 0:
            k = k+1
    if k%2 == 0:
        n = n+1
        print(n,a)
    a = a+2 # _A.H.M. Smeets_, Oct 04 2019
```

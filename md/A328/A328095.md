## Python
```Python
from functools import reduce
from operator import mul
n, A328095_list = 0, []
while len(A328095_list) < 10000:
    sn = str(n)
    if  sn in str(n*reduce(mul,(int(d) for d in sn))):
        A328095_list.append(n)
    n += 1 # _Chai Wah Wu_, Oct 19 2019
```

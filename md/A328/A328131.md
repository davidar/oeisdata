## Python
```Python
def A328131(n):
    s, tlist = str(n), ('2468','369','468','5','689','7','8','9')
    dset = set('0'+''.join(t if t[0] in s and sum(s.count(d) for d in t) > 1 else '' for t in tlist))
    return int('0'+''.join(d for d in s if d not in dset)) # _Chai Wah Wu_, Apr 02 2020
```

```Maple
A007732 := proc(n)
    a132740 := 1 ;

        end if;
```

## Sage
```Sage
def a(n):
    n = ZZ(n)
    rad = 2**n.valuation(2) * 5**n.valuation(5)
    return Zmod(n // rad)(10).multiplicative_order()
[a(n) for n in range(1, 20)]
# _F. Chapoton_, May 03 2020
```

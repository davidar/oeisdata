## Python
```Python
from sympy import prime
A007540_list = []
for n in range(1,10**4):
    p, m = prime(n), 1
    p2 = p*p
    for i in range(2,p):
        m = (m*i) % p2
    if m == p2-1:
        A007540_list.append(p) # _Chai Wah Wu_, Dec 04 2014
```

## Python
```Python
from sympy import prime, isprime, bernoulli
def ok(n):
    for k in range(2, n - 2, 2):
        if bernoulli(k).as_numer_denom()[0] % n == 0:
            return 0
    return isprime(n)
[n for n in range(3, 501) if ok(n)] # _Indranil Ghosh_, Jun 27 2017, after _Charles R Greathouse IV_
```

## Python
```Python
from sympy import nextprime
def a(n):
  p = nextprime(10**(n-1))
  while len(set(str(p))) < n: p = nextprime(p)
  return p
for n in range(1, 10):
  print(a(n), end=", ") # _Michael S. Branicky_, Feb 13 2021
```

## Python
```Python
import math
for n in range(10000):
    c1 = c2 = 0
    t = n
    while t:
        c1+=t&1
        t>>=1
    t = n*n*n
    while t:
        c2+=t&1
        t>>=1
    if c1*3==c2: print(str(n), end=',')
```

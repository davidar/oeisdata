## Python
```Python
from sympy import isprime
from itertools import count
def a(n): return next(k for k in count(n+1) if isprime(int(str(n)+str(k))))
print([a(n) for n in range(1, 68)]) # _Michael S. Branicky_, Oct 18 2022
```

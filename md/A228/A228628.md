## Python
```Python
from sympy import primerange
def nc(n): return 10**len(str(n)) - 1 - n
def auptop(limit): return [nc(p) for p in primerange(1, limit+1)]
print(auptop(271)) # _Michael S. Branicky_, Jul 06 2021
```

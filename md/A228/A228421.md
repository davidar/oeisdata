## Python
```Python
def a(n):
    count = 0
    for k in range(1,10**n+1):
        if '9' not in str(prime(k)):
            count += 1
    return count
n = 0
while n < 10:
    print(a(n), end=', ')
    n += 1
# _Derek Orr_, Jul 27 2014
```

## Python
```Python
def a(n):
  k, n2, np2 = 1, '3'*n, '3'*(n+1)
  while True:
    while not n2 in str(2**k): k += 1
    if np2 not in str(2**k): return k
    k += 1
print([a(n) for n in range(1, 9)]) # _Michael S. Branicky_, May 25 2021
```

## Python
```Python
def a(n):
  target, antitarget = '4'*n, '4'*(n+1)
  k, pow3 = 1, 3
  while True:
    t = str(pow3)
    if target in t and antitarget not in t: return k
    k, pow3 = k+1, pow3*3
print([a(n) for n in range(1, 9)]) # _Michael S. Branicky_, May 12 2021
```

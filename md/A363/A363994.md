## Mathematica
```Mathematica
s[n_, k_] := s[n, k] = Subsets[IntegerPartitions[n][[k]], {2}]
```
```Mathematica
g[n_, k_] := g[n, k] = DuplicateFreeQ[Map[Differences, s[n, k]]]
```
```Mathematica
t[n_] := t[n] = Table[g[n, k], {k, 1, PartitionsP[n]}];
```
```Mathematica
a[n_] := Count[t[n], True];
```
```Mathematica
Table[a[n], {n, 1, 20}]
```
## Python
```Python
from collections import Counter
from itertools import combinations
from sympy.utilities.iterables import partitions
def A363994(n): return sum(1 for p in partitions(n) if max(list(Counter(abs(d[0]-d[1]) for d in combinations(list(Counter(p).elements()),2)).values()),default=1)==1) # _Chai Wah Wu_, Sep 17 2023
```

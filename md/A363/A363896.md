## Python
```Python
from sympy import factorint,totient
A001414 = lambda k: sum(p*e for p, e in factorint(k).items())
def g():
  k = 2
  while True:
    if A001414(k) == totient(k): yield(k)
    k += 1
for a_n in g():
  print(a_n)
```

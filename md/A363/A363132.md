## Python
```Python
from sympy.utilities.iterables import partitions
def A363132(n): return sum(1 for s,p in partitions(n<<1,m=n,size=True) if n==s*min(p,default=0)) if n else 0 # _Chai Wah Wu_, Sep 21 2023
```

## Maple
```Maple
filter:= proc(n) local F2,Fp,v2,vp, t;
  F2, Fp:= selectremove(t -> t[1]=2, ifactors(n)[2]);
  if Fp = [] then return (n=2) fi;
  v2:= 2^F2[1,2];
  vp:= max(map(t -> t[1]^t[2],Fp));
  v2 > vp and v2/2 <= vp;
end proc:
select(filter, [seq(i,i=2.10000,2)]); # _Robert Israel_, May 18 2023
```

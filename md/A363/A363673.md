## Python
```Python
from sympy import prime, primefactors
def A363673(n):
    m = (1<<(prime(n)<<1))-1
    a, b = divmod(m,3)
    while not b:
        m = a
        a, b = divmod(m,3)
    return min(primefactors(m)) # _Chai Wah Wu_, Jun 26 2023
```

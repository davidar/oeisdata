## Python
```Python
from sympy import primefactors, totient as phi
def a(n): return 1 if n == 1 else phi(n) + a(n//min(primefactors(n)))
print([a(n) for n in range(1, 71)]) # _Michael S. Branicky_, Jun 30 2021
```

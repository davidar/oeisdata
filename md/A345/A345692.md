## Python
```Python
from statistics import pvariance
from sympy.core.numbers import igcdex
def A345692(n):
    zlist = [z for z in (igcdex(x,y) for x in range(1,n+1) for y in range(1,n+1)) if z[2] == 1]
    return pvariance(len(zlist)*u for u, v, w in zlist)
```

## Python
```Python
from sympy import divisors
def sd(n): return sum(map(int, str(n)))
def ok(n): return sd(n) == sd(divisors(n)[-2])
print(list(filter(ok, range(2, 840)))) # _Michael S. Branicky_, Jun 13 2021
```

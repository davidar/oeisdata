## Python
```Python
from sympy import isprime
def ok(n):
    if n < 10 or n%10 not in {1, 3, 7, 9}: return False
    s = str(n)[1:]
    return all(not isprime(int(d+s)) for d in "123456789")
print(list(filter(ok, range(1, 6000, 2)))) # _Michael S. Branicky_, Jun 29 2021
```

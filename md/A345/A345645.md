## Python
```Python
terms = []
for i in range(1, 700):
    occur = 0
    ii = i*i
    for j in range(1, i):
        k = int((ii - j*j) ** 0.25)
        if k*k*k*k + j*j == ii:
            occur += 1
    if occur == 1:
        terms.append(i)
print(terms)
```

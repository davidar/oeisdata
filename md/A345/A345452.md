## Python
```Python
from sympy import factorint
def ok(n):
    f = factorint(n)
    return sum(f.values())%2 == 0 and sum(p*f[p] for p in f)%2 == 0
print(list(filter(ok, range(1, 216)))) # _Michael S. Branicky_, Jun 24 2021
```

## Python
```Python
def pal(s): return s == s[::-1]
def rmz(s): return s.strip('0')
def ok(n): s = str(n); return not (pal(s) or pal(rmz(s)) or n%int(s[::-1]))
print(list(filter(ok, range(82000)))) # _Michael S. Branicky_, Jun 16 2021
```

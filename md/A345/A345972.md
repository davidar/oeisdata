## Python
```Python
def filter(n):
    seg = 0
    for c in str(n):
        seg += { 0: 6, 1: 2, 2: 5, 3: 5, 4: 4, 5: 5, 6: 6, 7: 3, 8: 7, 9: 6 }[int(c)]
    return(n % seg == 0)
```

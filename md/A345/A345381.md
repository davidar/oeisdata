## Python
```Python
from sympy import factorint
def ok(n):
    e = sorted(factorint(n).values())
    return len(e) == 2 and e[0] == 1 and e[1] > 1
print([k for k in range(276) if ok(k)]) # _Michael S. Branicky_, Dec 18 2021
```

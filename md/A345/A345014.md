## Python
```Python
from sympy import isprime
def a(n):
    k = 0
    while True:
        if isprime(2 ** n - k) and isprime(2 * (2 ** n - k) + 1):
            return k
        k += 1
print([a(i) for i in range(1, 21)])
```

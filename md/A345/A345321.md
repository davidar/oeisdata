## Python
```Python
from math import prod
from sympy import factorint
def A345321(n):
    f = factorint(n).items()
    return prod((p**(q+1)-1)//(p-1) for p, q in f) - prod((p**(q//3+1)-1)//(p-1) for p, q in f) # _Chai Wah Wu_, Jun 14 2021
```

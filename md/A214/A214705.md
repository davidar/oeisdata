## Python
```Python
from sympy import primerange
def ok(p): return set(str(p)) <= set("257")
def aupto(limit): return [p for p in primerange(2, limit+1) if ok(p)]
print(aupto(222557)) # _Michael S. Branicky_, Feb 05 2021
```

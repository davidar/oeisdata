## Python
```Python
SIZE = 300
a = [-8]*SIZE
top=0
for n in range(SIZE):
    if a[n]==-8:       # if a[n] is undefined yet
        top+=1
        a[n]=top
    if n+1+a[n]<SIZE and a[n+1+a[n]]==-8:   # if a[n+1+a[n]] is undefined yet
        a[n+1+a[n]]=a[n]
    print(a[n], end=',')
```

## Python
```Python
import math
sum = k = 0
for n in range(1, 220000):
    sum += n*n
    k += 1
    sqr = int(math.sqrt(sum*1.0/k))
    while sqr*sqr*k<sum:
        sqr+=1
    while sqr*sqr*k>sum:
        sqr-=1
    if sqr*sqr*k==sum and k>1:
        print(k, end=',')
        sum = k = 0
```

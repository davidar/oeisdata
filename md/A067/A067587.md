## Python
```Python
from math import comb
def A067587(n):
    c, k = 0, 0
    for i,j in enumerate(bin(n)[-1:1:-1]):
        if j == '1':
            k += 1
            c += comb(i,k)
    return comb(n.bit_count()+c,2)+c+1 # _Chai Wah Wu_, Mar 02 2023
```

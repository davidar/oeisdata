## Python
```Python
def A067872(n):
    y, x, n2 = n*(n+2), 2*n+3, n**2
    m, r = divmod(y,n2)
    while r:
        y += x
        x += 2
        m, r = divmod(y,n2)
    return m # _Chai Wah Wu_, Jan 25 2016
```

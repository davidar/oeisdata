## Python
```Python
from sympy import factorint
def A067695(n):
    if n == 1: return 1
    f, g = map(tuple,zip(*sorted(factorint(n).items())))
    return f[g.index(min(g))] # _Chai Wah Wu_, Feb 07 2023
```

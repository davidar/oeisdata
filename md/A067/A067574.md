## Python
```Python
def T(i, j): return int(str(i) + str(j))
def auptodiag(maxd):
    return [T(d+1-j, j) for d in range(1, maxd+1) for j in range(1, d+1)]
print(auptodiag(12)) # _Michael S. Branicky_, Nov 21 2021
```

## Python
```Python
from sympy.matrices import Matrix
from sympy import sieve
def a(n):
    sieve.extend_to_no(n**2)
    return Matrix(n, n, sieve[1:n**2+1]).det()
print([a(n) for n in range(1, 20)]) # _Indranil Ghosh_, Jul 31 2017
```

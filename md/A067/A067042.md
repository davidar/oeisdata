## Python
```Python
from math import prod
def ok(n):
    s = str(n)
    return s != '1' and prod(map(int, s[::2])) == prod(map(int, s[1::2]))
print([k for k in range(1010) if ok(k)]) # _Michael S. Branicky_, Nov 22 2021
```

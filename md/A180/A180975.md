## Python
```Python
from functools import lru_cache
@lru_cache(maxsize=None)
def T(n, k): return 0 if n*k == 0 else T(n-1, k-1) + 1 + T(n-1, k)
print([T(k, n-k) for n in range(1, 12) for k in range(1, n)]) # _Michael S. Branicky_, Apr 04 2021
```

## Python
```Python
def A180056_list(m):
    ret = [1]
    M = m + 1
    A = [1 for i in range(0, M)]
    for n in range(2, M):
        for k in range(2, M):
            if n == k:
                ret.append(A[k])
            A[k] = n*A[k-1] + k*A[k]
    return ret
```

## Python
```Python
from numpy import convolve
from sympy import prime, primerange
def aupton(nn):
    primes = list(primerange(2, prime(nn+1)+1))
    return list(convolve(primes[:-1], primes[1:]))[:nn]
print(aupton(41)) # _Michael S. Branicky_, Jun 19 2021
```

## Python
```Python
from math import prod
from sympy import factorint
def A162742(n): return prod(int(bin(f)[2:][::-1], 2)**e for f, e in factorint(n).items())
print([A162742(n) for n in range(1, 81)]) # _Michael S. Branicky_, Oct 07 2024
```

## Python
```Python
def sd(n): return sum(map(int, str(n)))
def ok(n): s = str(sd(n)); return s in str(n) and s in str(n**3)
print([k for k in range(1, 5411) if ok(k)]) # _Michael S. Branicky_, Jan 31 2022
```

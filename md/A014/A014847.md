## Maple
```Maple
filter:= proc(n) local F, f, r, c, t,j;
  F:= ifactors(n)[2];
  for f in F do
    r:= convert(n,base,f[1]);
    c:= 0: t:= 0:
    for j from 1 to nops(r) do
      if 2*r[j]+c >= f[1] then
          c:= 1; t:= t+1;
      else c:= 0
      fi;
    od;
    if t < f[2] then return false fi;
  od;
  true
end proc:
select(filter, [$1..1000]); # _Robert Israel_, Jan 07 2018
```

## Python
```Python
from numpy import convolve
from sympy import prime, primerange
def aupton(terms):
  p = list(primerange(2, prime(terms)+1))
  return list(convolve(p, p))[:terms]
print(aupton(41)) # _Michael S. Branicky_, Apr 12 2021
```

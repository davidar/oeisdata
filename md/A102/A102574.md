## Mathematica
```Mathematica
b[n_] := Product[{p, e} = pe; If[Mod[p, 4] == 3, p^e, 1], {pe, FactorInteger[n]}];
```
```Mathematica
a[n_] := With[{r = b[n]}, DivisorSigma[2, r] DivisorSigma[1, (n/r)^2]];
```

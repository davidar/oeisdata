## Python
```Python
from functools import cache
@cache
def a(n: int) -> int:
    return a(n - 1) + a(n // 2) + 1 if n > 1 else 0
print([a(n) for n in range(1, 51)])  # _Peter Luschny_, Jul 24 2022
```

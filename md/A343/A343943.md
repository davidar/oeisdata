## Python
```Python
from sympy import factorint
from sympy.utilities.iterables import multiset_combinations
def A343943(n):
    fs = factorint(n)
    return len(set(sum(d) for d in multiset_combinations(fs,(sum(fs.values())+1)//2))) # _Chai Wah Wu_, Aug 23 2021
```

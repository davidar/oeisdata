## Python
```Python
def A343145(n):
    k = 1
    while True:
        m = k
        for _ in range(n):
            m = prime(m)
        if m % 10**(len(str(k))) == k:
            return k
        k += 1
        while not (k % 2 and k % 5):
            k += 1
```

## Python
```Python
def a(n):
  b = bin(n)[2:]
  return sum(bi=='1' and b[i:].count('0')%2==0 for i, bi in enumerate(b))
print([a(n) for n in range(87)]) # _Michael S. Branicky_, Apr 03 2021
```

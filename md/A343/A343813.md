## Python
```Python
from sympy.utilities.iterables import partitions
from sympy import sieve, prime
def A343813(n):
    p = prime(n)
    pset = set(sieve.primerange(2,p+1))
    return sum(1 for d in partitions(p) if len(set(d)&pset) > 0) # _Chai Wah Wu_, May 01 2021
```

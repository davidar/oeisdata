## Java
```Java
public class Oeis2 {
    public static void main(String[] args) {
        StringBuilder str = new StringBuilder();
        long n = 1;
        while (true) {
            str.append(n);
            if (str.indexOf(String.valueOf(n * n)) >= 0) {
                System.out.print(n + ", ");
            }
            n++;
        }
    }
}
```

## Python
```Python
def ok(sq): return sum(d in "02468" for d in str(sq)) == 1
def aupto(limit):
  sqs = (i*i for i in range(int(limit**.5)+2) if i*i <= limit)
  return list(filter(ok, sqs))
print(aupto(131769)) # _Michael S. Branicky_, May 20 2021
```

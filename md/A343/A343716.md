## Python
```Python
def afind(startat=1, limit=10**9):
  for k in range(startat, limit+1):
    kk = k*k
    if (pow(5, k, kk) - pow(4, k, kk) - pow(3, k, kk))%kk == 0:
      print(k, end=", ")
afind(limit=10**5) # _Michael S. Branicky_, May 16 2021
```

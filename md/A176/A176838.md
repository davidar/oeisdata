## Python
```Python
from sympy import isprime, primerange
def ok(p): q, r = divmod(p**3, 10); return r == 3 and isprime(q)
print(list(filter(ok, primerange(2, 18200)))) # _Michael S. Branicky_, Aug 31 2021
```

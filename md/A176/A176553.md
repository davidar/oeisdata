## Python
```Python
from sympy import divisors, isprime
def ok(m): return m==1 or isprime(int("".join(str(d) for d in divisors(m))))
print([m for m in range(1, 900) if ok(m)]) # _Michael S. Branicky_, Feb 05 2022
```

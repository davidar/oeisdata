## Python
```Python
from sympy import primerange, isprime
def fromdigits(d, b):
  n = 0
  for di in d: n *= b; n += di
  return n
def ok(b): return isprime(fromdigits([p for p in primerange(1, b)], b))
print([b for b in range(3, 700) if ok(b)]) # _Michael S. Branicky_, Mar 04 2021
```

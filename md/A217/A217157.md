   for k from 1 do
      L:= convert(n^k,base,10);

## Python
```Python
def A217157(n):
    m, k = 1, n
    while True:
        s = str(k)
        for i in range(1,len(s)):
            if s[i] == s[i-1]:
                return m
        m += 1
        k *= n # _Chai Wah Wu_, Feb 20 2019
```

## Python
```Python
def agen():
  yield 2
  digits, adder = 1, 20
  while True:
    for i in range(10**digits): yield adder + i
    digits, adder = digits+1, adder*10
g = agen()
print([next(g) for i in range(54)]) # _Michael S. Branicky_, Feb 20 2021
```

## Python
```Python
from sympy import isprime
def a(n):
    prefix = 1
    while not all(isprime(int(str(prefix) + "9"*k)) for k in range(1, n+1)):
        prefix += 1
    return prefix
print([a(n) for n in range(1, 9)]) # _Michael S. Branicky_, Nov 19 2021
```

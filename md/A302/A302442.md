## Python
```Python
from sympy import isprime
def aupton(terms):
  s, alst = 0, []
  for n in range(1, terms+1):
    s += sum(isprime(b**2-2) for b in range(10**(n-1), 10**n))
    alst.append(s)
  return alst
print(aupton(6)) # _Michael S. Branicky_, May 26 2021
```

## Python
```Python
from python import isprime
k, klist, A302021_list = 0, [isprime(i**2+1) for i in range(6)], []
while len(A302021_list) < 10000:
    i = isprime((k+6)**2+1)
    if klist[0] and klist[2] and i:
        A302021_list.append(k)
    k += 1
    klist = klist[1:] + [i] # _Chai Wah Wu_, Apr 01 2018
```

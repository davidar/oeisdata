## Python
```Python
def a(n):
  b = bin(n)[2:]
  m = len(b)
  return len(set(int(b[i:j]) for i in range(m) for j in range(i+1,m+1))-{0})
print([a(n) for n in range(1, 91)]) # _Michael S. Branicky_, Jan 20 2021
```

## Python
```Python
A177029_list = []
for m in range(1,10**4):
    n, c = 3, 0
    while n*(n+1) <= 2*m:
        if not 2*(n*(n-2) + m) % (n*(n - 1)):
            c += 1
            if c > 1:
                break
        n += 1
    if c == 1:
        A177029_list.append(m) # _Chai Wah Wu_, Jul 28 2016
```

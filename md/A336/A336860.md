## Python
```Python
from gmpy2 import is_square, isqrt
def a(n):
  if n == 1:
    return 1
  return a(isqrt(n)) + n - isqrt(n) ** 2
```
```Python
from gmpy2 import is_square, isqrt
a = [0,1]
for i in range(2,n+1):
  if is_square(i):
    a.append(a[isqrt(i)])
  else:
    a.append(a[i-1]+1)
```

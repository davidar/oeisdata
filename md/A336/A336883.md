## Python
```Python
n_start=5
n_end=n_start+10000
k = 1
for n in range(n_start, n_end, 4):
    c=(n-1)//2
    r=1
    for i in range(2, c+1):
        r=r*i % n
        if r==0:
            break
    if (n-r)*r % n ==1:
        print(k, r)
        k = k + 1
# modified by _Georg Fischer_, Oct 16 2020
```

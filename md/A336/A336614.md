## Python
```Python
from itertools import product
from sympy import Matrix
def A336614(n):
    c = 0
    for d in product((0,1),repeat=n*n):
        M = Matrix(d).reshape(n,n)
        if M*M == M.T:
            c += 1
    return c # _Chai Wah Wu_, Sep 29 2020
```

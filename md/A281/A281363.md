## Python
```Python
def A281363(n):
    m, q = 1, 4*n**2-1
    p = pow(2, 2*n, q)
    r = p
    while r != 1:
        m += 1
        r = (r*p) % q
    return m # _Chai Wah Wu_, Jan 28 2017
```

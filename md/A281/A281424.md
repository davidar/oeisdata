## Python
```Python
from sympy import isprime
def afind(limit, startk=1):
    pow10 = 10**startk
    for k in range(startk, limit+1):
        if isprime(16*(pow10 - 1)//3 + 1): print(k, end=", ")
        pow10 *= 10
afind(600) # _Michael S. Branicky_, Aug 17 2021
```

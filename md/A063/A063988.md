## Python
```Python
from sympy import jacobi_symbol as J, prime
def a(n):
    p=prime(n)
    return [i for i in range(1, p) if J(i, p)==-1]
print([a(n) for n in range(2, 13)]) # _Indranil Ghosh_, May 27 2017
```

## Python
```Python
def iter(n): return 7*n-1 if n%2 == 1 else n//2
def aupton(terms):
  alst = [3]
  for n in range(1, terms+1): alst.append(iter(alst[-1]))
  return alst
print(aupton(42)) # _Michael S. Branicky_, Mar 20 2021
```

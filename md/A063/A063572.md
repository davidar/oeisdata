## Python
```Python
def a(n):
  target, k, pow9 = str(n), 1, 9
  while not target in str(pow9): k, pow9 = k+1, pow9*9
  return k
print([a(n) for n in range(73)]) # _Michael S. Branicky_, May 02 2021
```

## Python
```Python
def a(n):
  target, k, pow4 = str(n), 1, 4
  while not target in str(pow4): k, pow4 = k+1, pow4*4
  return k
print([a(n) for n in range(72)]) # _Michael S. Branicky_, May 02 2021
```

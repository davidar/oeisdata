## Python
```Python
def a(n):
  k, pow8, s = 1, 8, str(n)
  while s not in str(pow8): k += 1; pow8 *= 8
  return k
print([a(n) for n in range(61)]) # _Michael S. Branicky_, Apr 03 2021
```

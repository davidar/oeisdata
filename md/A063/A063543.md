## Python
```Python
def a(n):
    digits = map(int, str(n))
    product = 1
    for d in digits:
        if d != 0:
            product *= d
    return n - product
[a(n) for n in range(20)]
# _Elisabeth Zemack_, Sep 16 2019; corrected by _Fabio Somenzi_, Jan 13 2020
```

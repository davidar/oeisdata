## Python
```Python
from math import isqrt
from sympy import mobius
def A063035(n): return (m:=10**n)-sum(mobius(k)*(m//k**2) for k in range(1,isqrt(m)+1)) # _Chai Wah Wu_, Jul 20 2024
```

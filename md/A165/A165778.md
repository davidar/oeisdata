## Python
```Python
from sympy import isprime, nextprime
def afind(limit):
    k, pow2 = 1, 2
    for k in range(1, limit+1):
        if isprime(abs(pow2-57)):
            print(k, end=", ")
        k += 1
        pow2 *= 2
afind(2100) # _Michael S. Branicky_, Dec 27 2021
```

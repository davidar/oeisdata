## Python
```Python
def a(n):
    if n == 2: return 7
    digs = n.bit_length()
    return int(2**digs) if n != 2**(digs - 1) else int(2**digs + 2)
print([a(n) for n in range(2, 71)]) # _Michael S. Branicky_, Jul 04 2021
```

## Python
```Python
def R(n): return int(str(n)[::-1])
def a(n):
    Rn2 = R(n**2)
    q, r = divmod((n-Rn2)**2, n+Rn2)
    return q if r == 0 else q + 1
print([a(n) for n in range(1, 67)]) # _Michael S. Branicky_, Jan 17 2022
```

## Python
```Python
from sympy import Matrix,prime,nextprime
def A350200(n,k):
    p = [prime(k)] if n > 0 else []
    for i in range(2*n-2): p.append(nextprime(p[-1]))
    return Matrix(n,n,lambda i,j:p[i+j]).det()
```

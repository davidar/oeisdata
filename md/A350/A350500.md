## Python
```Python
from sympy import isprime, factorint
def ok(n): return n%2 == 0 and isprime(n//2-1) and isprime(n//2+1) and sum(factorint(n-1).values()) == 2
print([k for k in range(8005) if ok(k)]) # _Michael S. Branicky_, Jan 02 2022
```

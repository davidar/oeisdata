## Python
```Python
from sympy import isprime, nextprime, prevprime
def ok(n):
    return n > 2 and isprime(nextprime(n) - n) and isprime(n - prevprime(n))
print([k for k in range(341) if ok(k)]) # _Michael S. Branicky_, Jan 01 2022
```

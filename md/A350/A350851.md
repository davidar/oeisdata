## Python
```Python
seq=[];prev=[];total=0
for n in range(30):
  row=[1]
  last=int(n/2)
  for k in range(last):
    row.append(prev[k]+prev[k+1])
  if n%2==1:
    row.append(row[-1])
  prev=row
  total+=sum(row)
  seq.append(total)
print(seq)
```

## Python
```Python
def R(n): return int(str(n)[::-1])
def a(n):
    Rn = R(n)
    q, r = divmod((n-Rn)**2, n+Rn)
    return q if r == 0 else q + 1
print([a(n) for n in range(1, 106)]) # _Michael S. Branicky_, Jan 17 2022
```

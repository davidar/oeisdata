## Maple
```Maple
f:= proc(n) local S,k,x,s;
  S:= mul(1 + x^(2*k^3),k=1..n);
  s:= sum(k^3,k=1..n);
  coeff(S,x,s) + coeff(S,x,s+1)
end proc:
map(f, [$0..50]); # _Robert Israel_, Mar 15 2023
```

## Python
```Python
from functools import lru_cache
@lru_cache(maxsize=None)
def b(n, i):
    if n > (i*(i+1)//2)**2: return 0
    if i == 0: return 1
    return b(n+i**3, i-1) + b(abs(n-i**3), i-1)
def a(n): return b(0, n) + b(1, n)
print([a(n) for n in range(46)]) # _Michael S. Branicky_, Jan 19 2022
```

## Python
```Python
from sympy import isprime, nextprime, factorial
def A350782(n):
    m, p, c = factorial(n), 3, 0
    while p <= m:
        if isprime(2*m-p):
            c += 1
        p = nextprime(p)
    return c # _Chai Wah Wu_, Jan 16 2022
```

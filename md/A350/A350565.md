# a\(n\) is the minimal permanent of an n X n matrix using the integers 1 to n^2\.
https://oeis.org/A350565

## Python
```Python
from itertools import permutations
from sympy import Matrix
def A350565(n): return 1 if n == 0 else min(Matrix(n,n,p).per() for p in permutations(range(1,n**2+1))) # _Chai Wah Wu_, Jan 21 2022
```

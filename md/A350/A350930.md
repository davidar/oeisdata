## Python
```Python
from itertools import permutations
from sympy import Matrix
def A350930(n): return min(Matrix([p[n-1-i:2*n-1-i] for i in range(n)]).det() for p in permutations(range(1,2*n))) # _Chai Wah Wu_, Jan 27 2022
```

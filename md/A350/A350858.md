## Python
```Python
from itertools import permutations
from sympy import Matrix
def A350858(n): return 1 if n == 0 else min(Matrix(n,n,p).per() for p in permutations(prime(m) for m in range(1,n**2+1))) # _Chai Wah Wu_, Jan 21 2022
```

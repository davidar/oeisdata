## Python
```Python
from sympy import prime
from itertools import count, islice
def agen():
    for i in count(1):
        pi = prime(i)
        yield from range(pi, pi-i, -1)
print(list(islice(agen(), 69))) # _Michael S. Branicky_, Dec 18 2021
```

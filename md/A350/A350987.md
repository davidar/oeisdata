## Python
```Python
from itertools import count, islice
def agen():
    for i in count(0):
        t = i*(i+1)//2
        b = bin(t)[2:]
        if b == b[::-1]:
            yield t
print(list(islice(agen(), 37))) # _Michael S. Branicky_, Jan 28 2022
```

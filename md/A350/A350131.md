## PHP
```PHP
<?php
$segments = [6,3,4,4,4,5,5,3,7,5];
$maxvalue = 100; // how many integers needs to be generated
for($i=0;$i<=$maxvalue;$i++) {
    $number = str_split($i);
    $usedSegments = 0;
    foreach($number as $n) {
        $usedSegments += $segments[$n];
    }
    echo $usedSegments."\n";
}
```

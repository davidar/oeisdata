## Maple
```Maple
f:= proc(n) local i;
  min(map(t -> LinearAlgebra:-Determinant(LinearAlgebra:-ToeplitzMatrix(t)), combinat:-permute([seq(ithprime(i),i=1..2*n-1)]))) end proc:
f(0):= 1:
map(f, [$0..5]); # _Robert Israel_, Apr 01 2024
```

## Python
```Python
from itertools import permutations
from sympy import Matrix, prime
def A350932(n): return min(Matrix([p[n-1-i:2*n-1-i] for i in range(n)]).det() for p in permutations(prime(i) for i in range(1,2*n))) # _Chai Wah Wu_, Jan 27 2022
```

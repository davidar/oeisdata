## Python
```Python
from sympy.ntheory.factor_ import core
def squarefree(n): return core(n, 2) == n
def ok(n): return squarefree(n) and squarefree(n + int(str(n)[::-1]))
print([k for k in range(1, 200) if ok(k)]) # _Michael S. Branicky_, Jan 07 2022
```

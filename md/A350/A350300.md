## Python
```Python
from math import log
n = 1
while n <= 50:
    k = 2
    while 9*n*log(9*n*log(k,10),10) >= k:
        s1 = sum(int(d) for d in str(k**n))
        s2 = sum(int(d) for d in str(s1**n))
        if k != s1 and k == s2:
            print(k)
        k += 1
    n += 1
```

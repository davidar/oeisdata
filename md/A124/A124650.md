# Number of n\-digit numbers m such that m and m^10 are zeroless\.

## Python
```Python
from itertools import product
def A124650(n):
    return sum(1 for s in product('123456789',repeat=n) if '0' not in str(int(''.join(s))**10)) # _Chai Wah Wu_, Mar 24 2020
```

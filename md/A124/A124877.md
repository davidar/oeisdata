## Python
```Python
# uses A007408gen() and imports from A007408
from sympy import isprime
def agen(): yield from (k for k, ak in enumerate(A007408gen(), 1) if isprime(ak))
print(list(islice(agen(), 15))) # _Michael S. Branicky_, Jun 26 2022
```

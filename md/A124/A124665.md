## Python
```Python
from sympy import isprime
def ok(n):
    s = str(n)
    if any(isprime(int(s+c)) for c in "1379"): return False
    return not any(isprime(int(c+s)) for c in "0123456789")
print([k for k in range(419) if ok(k)]) # _Michael S. Branicky_, Aug 01 2022
```

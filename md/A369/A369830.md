## Python
```Python
from itertools import permutations
from sympy import Matrix
def A369830(n): return len({Matrix([p[i:0:-1]+p[:n-i] for i in range(n)]).det() for p in permutations(range(1,n+1))}) # _Chai Wah Wu_, Feb 12 2024
```

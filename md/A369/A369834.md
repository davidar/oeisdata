## Python
```Python
from itertools import permutations
from sympy import Matrix
def A369834(n): return len({Matrix([p[i:0:-1]+p[:n-i] for i in range(n)]).det() for p in permutations(range(n))}) # _Chai Wah Wu_, Feb 11 2024
```

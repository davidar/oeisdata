## Python
```Python
from collections import Counter
def A369764(n):
    c = {0:1,1:-1}
    for i in range(2,n+1):
        d = Counter(c)
        for k in c:
            d[k+i**3] -= c[k]
        c = d
    return max(c.values()) # _Chai Wah Wu_, Jan 31 2024
```

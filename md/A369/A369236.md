## Maple
```Maple
N:= 200: # for a(0) to a(N)
with(GraphTheory):
V:= [$0..N]:
E:= {seq([i,i+1],i=0..N-1),seq([i,i^2],i=1..floor(sqrt(N)))}:
G:= Graph(V,E):
map(t -> t[2], DijkstrasAlgorithm(G,0)); # _Robert Israel_, Jan 23 2024
```
## Python
```Python
k = 10 #generates k^2 terms
a = [0,1,2,3]
for n in range(2,k):
    a = a + list(range(a[n] + 1, a[n] + 2 * (n + 1)))
```

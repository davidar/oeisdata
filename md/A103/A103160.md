## Python
```Python
from math import factorial, gcd
def a(n):
    f = factorial(n)
    return gcd(int(str(f)[::-1]), int(str(f*(n+1))[::-1]))
print([a(n) for n in range(1, 64)]) # _Michael S. Branicky_, Dec 12 2021
```

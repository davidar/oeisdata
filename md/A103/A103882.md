## Python
```Python
def A103882(n):
    if n == 0: return 1
    m, g = 1, 0
    for k in range(n+1):
        g += m*n//(n+k)
        m *= (n+k+1)*(n-k)**2
        m //= (k+1)**3
    return g # _Chai Wah Wu_, Oct 04 2022
```

## Python
```Python
from sympy import isprime, nextprime
def ok(p, n): return isprime(int(str(p**n)[::-1]))
def a(n):
  p = 2
  while not ok(p, n): p = nextprime(p)
  return p
print([a(n) for n in range(1, 63)]) # _Michael S. Branicky_, Feb 19 2021
```

## Python
```Python
from sympy import integer_log
def A059485(n):
    def f(x): return n+sum((x//3**i).bit_length() for i in range(integer_log(x,3)[0]+1))
    m, k = n, f(n)
    while m != k: m, k = k, f(k)
    return m # _Chai Wah Wu_, Sep 16 2024
```

## Python
```Python
from sympy import hermite, Poly, Symbol
x = Symbol('x')
def a(n):
    return Poly(hermite(n, x), x).coeffs()[::-1]
for n in range(21): print(a(n)) # _Indranil Ghosh_, May 26 2017
```

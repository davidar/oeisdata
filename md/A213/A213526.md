## Maple
```Maple
a:= proc(n) local i, k, m, r;
      k, m, r:= n, 3*n, 0;
      for i from 0 while (m>0 or k>0) do
        r:= r +2^i* irem(m, 2, 'm') *irem(k, 2, 'k')
      od; r
    end:
seq(a(n), n=0..100);  # _Alois P. Heinz_, Jun 22 2012
```

## Python
```Python
src = [0]*1024
dst = [0]*1024
for n in range(19):
    dst[0] = dst[n] = 1
    for k in range(1, n):
        dst[k] = (src[k-1]+src[k]) % n
    for k in range(n+1):
        src[k] = dst[k]
        print(dst[k], end=',')
```

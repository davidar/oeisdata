## Python
```Python
from math import gcd
terms, appears = [1], {}
for n in range(2, 100):
    t = 2
    while not(appears.get(t) is None and gcd(terms[-1]*n, t)>1 and t!=terms[-1] and t!=n):
        t += 1
    appears[t] = True; terms.append(t);
print(terms) # _Gleb Ivanov_, Nov 20 2021
```

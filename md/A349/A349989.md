## Python
```Python
from sympy.ntheory.factor_ import core
def squarefree(n): return core(n, 2) == n
def a(n):
    k = 1
    while squarefree(k**n + (k+1)**n): k += 1
    return k
print([a(n) for n in range(1, 16)]) # _Michael S. Branicky_, Dec 09 2021
```

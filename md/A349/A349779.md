## Python
```Python
from sympy import nextprime, divisor_sigma
d = dict()
j, p = 0, 0
while True:
    j, p = j+1, nextprime(p)
    n = j+p
    a = divisor_sigma(n)-n
    d[(n, a)] = j
    if (a, n) in d:
        print(d[(a, n)], j) # _Martin Ehrenstein_, Dec 02 2021
```

## Python
```Python
def main(N): # prints all terms <= N
  for k in range(1,N+1):
    n1=str(k)
    n2 = 1
    for i in range(1,len(n1)+1):
      sum1 = 0
      for j in range(0,i):
        sum1 += int(n1[j])
      n2 = n2*sum1
    if n2 == k:
       print(k, end=", ")
```

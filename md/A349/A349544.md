## Python
```Python
def f(k,n):
    if k == 0 and n == 0: return (x for x in (1,))
    if k < n: return (y*3 for y in f(k,n-1))
    return (abs(x+y) for x in f(k-1,n) for y in (2**n,-2**n))
def A349544(n): return min(f(n,n)) # _Chai Wah Wu_, Nov 24 2021
```

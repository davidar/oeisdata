# Array read by antidiagonals, n \>\= 2, m \>\= 0: T\(n,m\) is the smallest prime p \= prime\(k\) such that all n\-th differences of \(prime\(k\), \.\.\., prime\(k\+n\+m\)\) are zero\.

## Python
```Python
from sympy import nextprime
def A349644(n,m):
    d = [float('inf')]*(n-1)
    p = [0]*(n+m)+[2]
    c = 0
    while 1:
        del p[0]
        p.append(nextprime(p[-1]))
        d.insert(0,p[-1]-p[-2])
        for i in range(1,n):
            d[i] = d[i-1]-d[i]
        if d.pop() == 0:
            if c == m: return p[0]
            c += 1
        else:
            c = 0
```

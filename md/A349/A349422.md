## Haskell
```Haskell
import Data.Char (digitToInt)
a n = read [s !! mod (i - digitToInt (s !! i)) l | i <- [0..l-1]] :: Integer
    where s = show n; l = length s
```
## Python
```Python
def a(n):
    s, l = str(n), len(str(n))
    return int("".join(s[(i - int(s[i])) % l] for i in range(l)))
```

## Python
```Python
from math import prod
from sympy import prime, primepi
def a(n): r = primepi(n); return prod(prime(i) for i in range(r+1, r+n+1))
print([a(n) for n in range(1, 17)]) # _Michael S. Branicky_, Feb 15 2021
```

## Python
```Python
from itertools import product
from sympy import Matrix
def A272659(n): return len({tuple(Matrix(n,n,p).charpoly().as_list()) for p in product(range(3),repeat=n**2)}) if n else 1 # _Chai Wah Wu_, Sep 30 2023
```

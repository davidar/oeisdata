## Python
```Python
xf, yf = 2, 1
x, n = 2*xf, 0
while n < 20000:
    n = n+1
    y1, y0, i = 0, yf, 1
    while y0%n != 0:
        y1, y0, i = y0, x*y0-y1, i+1
    print(n, i)
```

## Python
```Python
from collections import Counter
def A316706(n):
    c = {0:1}
    for k in range(1,n+1):
        b = Counter(c)
        for j in c:
            a = c[j]
            b[j+k] += a
            b[j-k] += a
        c = b
    return c[n] # _Chai Wah Wu_, Feb 05 2024
```

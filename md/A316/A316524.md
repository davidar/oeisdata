## Python
```Python
from sympy import factorint, primepi
def A316524(n):
    fs = [primepi(p) for p in factorint(n,multiple=True)]
    return sum(fs[::2])-sum(fs[1::2]) # _Chai Wah Wu_, Aug 23 2021
```

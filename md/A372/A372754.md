## Maple
```Maple
f:= proc(x) local b,L;
  for b from 2 do
    L:= convert(x,base,b);
    if L = ListTools:-Reverse(L) then return b fi
  od
end proc:
map(f, [seq(combinat:-fibonacci(n), n=1..70)]);
```

## Python
```Python
from itertools import count
from sympy import fibonacci
from sympy.ntheory.factor_ import digits
def A372754(n): return next(b for b in count(2) if (s := digits(fibonacci(n),b)[1:])[:(t:=len(s)+1>>1)]==s[:-t-1:-1]) # _Chai Wah Wu_, May 13 2024
```

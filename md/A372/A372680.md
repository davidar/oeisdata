## Python
```Python
def f(m):
  a = str(2**m)
  for i in range(0, m.bit_length()):
    if str(2**i) not in a:
      return 0
  return 1
def a(n):
  m = 0
  i = 0
  while i != n:
    m += 1
    i += f(m)
  return m
```

## Python
```Python
from sympy import divisors
def a(n):
    divs = set(divisors(n)[:-1])
    return sum(1 for d in divs if pow(d, n, n) in divs)
print([a(n) for n in range(1, 101)]) # _Michael S. Branicky_, May 13 2024
```

## Python
```Python
def t(n):
    s=0
    for a in range(-n,n+1):
        for b in range(-n,n+1):
            for c in range(-n,n+1):
                for d in range(-n,n+1):
                    if (a*d-b*c)==(a*d+b*c):
                        s+=1
    return s
for i in range(0,1001):
    print str(i)+" "+str(t(i))
```

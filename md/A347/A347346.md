## Python
```Python
def A347346(n):
    if n % 10 == 0:
        return 0
    s = str(n)
    if s == s[::-1]:
        return n
    for i in range(1,len(s)):
        if s[:-i]== s[-i-1::-1]:
            return int(s[:-i-1:-1]+s)
```

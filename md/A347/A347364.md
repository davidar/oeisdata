## Python
```Python
from sympy import isprime, primerange
def auptop(maxp): return list(filter(isprime, (p**4 + p**2 -1 for p in primerange(1, maxp+1))))
print(auptop(580)) # _Michael S. Branicky_, Aug 29 2021
```

## Python
```Python
def a(n):
    s = str(n)
    cyclic_perms = [int("".join(s[i:] + s[:i])) for i in range(len(s))]
    c, C = min(cyclic_perms), max(cyclic_perms)
    return C - c
print([a(n) for n in range(113)]) # _Michael S. Branicky_, Sep 26 2021
```

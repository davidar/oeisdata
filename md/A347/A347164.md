## Python
```Python
k,r,n=1,2,1
while n<=6000:
    s,t=set(),r
    for i in range(10):
        s.add(t%10)
        t=t//10
    if len(s)==10:
        print(n,k)
        n=n+1
    k,r=k+1,2*r%10**10
```

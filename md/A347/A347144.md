## Python
```Python
def a(n):
    k, target = 1, set(str(i) for i in range(10) if i != n)
    while set(str(k*k)) != target: k += 1
    return k
print([a(n) for n in range(10)]) # _Michael S. Branicky_, Aug 19 2021
```

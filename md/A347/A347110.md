## Python
```Python
from sympy import isprime, primerange
def ok(p): return isprime(2*p+1) and isprime((2*p)**2 + (2*p+1)**2)
print(list(filter(ok, primerange(1, 50592)))) # _Michael S. Branicky_, Aug 18 2021
```

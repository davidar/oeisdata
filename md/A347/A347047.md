## Python
```Python
from sympy import prime, sieve
def a(n):
    p = [0] + list(sieve.primerange(1, prime(n)+1))
    return min(p[i]*p[n-i] for i in range(1, (n+1)//2))
print([a(n) for n in range(3, 58)]) # _Michael S. Branicky_, Sep 05 2021
```

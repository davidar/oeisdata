## Maple
```Maple
b:= proc(n, k) option remember; `if`(n<2, 1,
      b(n-1, k)*(k^n-1)/(k-1))
    end:
a:= n-> b(n$2):
seq(a(n), n=0..12);
```

## Python
```Python
from math import prod
def a(n):
    return 1 if n <= 1 else prod((n**k - 1)//(n - 1) for k in range(1, n+1))
print([a(n) for n in range(12)]) # _Michael S. Branicky_, Sep 09 2021
```

## Python
```Python
from sympy import isprime
def comp(s): z, o = ord('0'), ord('1'); return s.translate({z:o, o:z})
def ok(n): return isprime(int(comp(bin(n)[2:]), 2))
print(list(filter(ok, range(214)))) # _Michael S. Branicky_, Sep 03 2021
```

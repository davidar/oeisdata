## Maple
```Maple
A271497:=proc(n) local t1,t2,len; global b;
if n mod (b+1) = 0 then return(n/(b+1)); fi;
t1:=convert(n,base,b); len:=nops(t1);
t2:=sort(t1);
add(t2[i]*(b^(i-1)+b^(len-i)),i=1..len);
end;
b:=2; [seq(A271497(n),n=0..100)];
```

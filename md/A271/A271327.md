## Python
```Python
from sympy import prime
A271327_list = []
for n in range(1,10**3):
    p, a, b = prime(n), 1, 1
    for i in range(n):
        if not a:
            A271327_list.append(n)
            break
        a, b = b, (a + b) % p # _Chai Wah Wu_, Apr 08 2016
```

## Python
```Python
d=[0]*200
for n in range(1024):
    b = bin(n)[2:]
    c0 = b.count('0')
    c1 = len(b) - c0
    diff = c0 - c1
    print str(d[100+diff])+',',
    d[100+diff] += 1
```

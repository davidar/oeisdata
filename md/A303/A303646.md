## Python
```Python
import math
for n in range (1, 100):
   sqn = math.ceil(math.sqrt(2)*n)
   count = 0
   for x in range(-sqn, sqn):
       for y in range(-sqn, sqn):
           if (x*x+y*y>n*n and abs(x)+abs(y)<n*math.sqrt(2)):
               count += 1
   print(count)
```

## Python
```Python
import math
tan=math.sqrt(3)/3
for n in range (1,71):
  count=0
  for x in range (-n, 2*n):
   for y in range (-2*n, 2*n):
    if (x*x+y*y>n*n and y<-tan*x+2*tan*n and y>tan*x-2*tan*n and x>-n):
     count=count+1
  print(count)
```

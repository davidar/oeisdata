## Python
```Python
from sympy import prime
A030082_list = []
for i in range(1,10**6):
    p = prime(i)
    q = p**3
    if set(str(p)) <= set(str(q)):
        A030082_list.append(q) # _Chai Wah Wu_, Mar 17 2021
```

## Haskell
```Haskell
a030308 n k = a030308_tabf !! n !! k
a030308_row n = a030308_tabf !! n
a030308_tabf = iterate bSucc [0] where
   bSucc []       = [1]
   bSucc (0 : bs) = 1 : bs
   bSucc (1 : bs) = 0 : bSucc bs
-- _Reinhard Zumkeller_, Jun 17 2012
```

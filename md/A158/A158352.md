## Python
```Python
from num2words import num2words
def A158352(n):
  i = 1
  while sum(1 for c in num2words(i).replace(' and ', '') if c in "aeiouy")!=n:
    i += 1
  return i
print([A158352(n) for n in range(1,16)]) # _Michael S. Branicky_, Oct 23 2020
```

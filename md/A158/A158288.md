## Python
```Python
from sympy import primerange, isprime
def a(n):
    s=0
    for p in primerange(100*n - 99, 100*n - 1):
        if isprime(p + 2):s+=1
    return s
print([a(n) for n in range(1, 201)]) # _Indranil Ghosh_, Jul 20 2017, after PARI code
```

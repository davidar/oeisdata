## Maple
```Maple
d := 8 ;
for n from 1 do
        convert(d*n^d,base,10) ;
        if verify([8,8,8,8,8,8,8,8],%,'sublist') then
                print(n) ;
        end if;
end do: # _R. J. Mathar_, Jan 11 2013
```

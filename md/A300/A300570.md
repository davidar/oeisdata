## Python
```Python
from itertools import count, islice
def agen(): # generator of terms
    s = ""
    for k in count(1):
        s = bin(k)[2:] + s
        yield int(s)
print(list(islice(agen(), 15))) # _Michael S. Branicky_, Feb 19 2023
```

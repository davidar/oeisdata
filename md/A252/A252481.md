## Python
```Python
def ok(n):
  s = set(str(n)); return '1' in s and len(s) == ord(max(s))-ord(min(s))+1
def aupto(nn): return [m for m in range(1, nn+1) if ok(m)]
print(aupto(1223)) # _Michael S. Branicky_, Jan 10 2021
```

## Python
```Python
def A043261(n):
    if n == 1: return 0
    a = 1<<(l:=n.bit_length()-2)
    m = a|(n&a-1)
    return (m.bit_count()<<1) - (0 if a&n else m&1) # _Chai Wah Wu_, Jun 13 2024
```

## Python
```Python
from itertools import count, islice
def agen(): yield from ((1<<k)-(1<<j)+(1<<i)-1 for k in count(1) for j in range(k-1, 1, -1) for i in range(1, j))
print(list(islice(agen(), 53))) # _Michael S. Branicky_, Feb 25 2023
```

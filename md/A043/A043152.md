## Python
```Python
def ok(n):
    on, onm1 = oct(n)[2:], oct(n-1)[2:]
    return ("01" in on or "10" in on) and not ("01" in onm1 or "10" in onm1)
print(list(filter(ok, range(1, 1538)))) # _Michael S. Branicky_, Aug 22 2021
```

## Python
```Python
def has13or31(n): o = oct(n); return "13" in o or "31" in o
def ok(n): return has13or31(n) and not has13or31(n+1)
print([k for k in range(1600) if ok(k)]) # _Michael S. Branicky_, Nov 25 2021
```

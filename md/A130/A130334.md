## Python
```Python
from math import gcd
def A130334(n):
    k, Tn, Tm = n+1, n*(n+1)//2, (n+1)*(n+2)//2
    while gcd(Tn,Tm) != 1:
        k += 1
        Tm += k
    return k # _Chai Wah Wu_, Sep 16 2021
```

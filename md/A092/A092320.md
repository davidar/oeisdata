## Python
```Python
from num2words import num2words as n2w
def letters(n): return sum(c.isalpha() for c in n2w(n).replace(" and", ""))
def ok(n): return n%letters(n) == 0
print([k for k in range(1, 1000) if ok(k)]) # _Michael S. Branicky_, Jan 17 2022
```

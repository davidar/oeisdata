## Java
```Java
int n = 100;
int[] terms = new int[n];
for (int i = 0; i < n; i++) {
     for (int j = 0; j < i; j++) {
          if (Math.sqrt(i+j) == Math.floor(Math.sqrt(i+j))) {
               terms[i]++;
          }
     }
     System.out.print(terms[i] + ", ");
}
```

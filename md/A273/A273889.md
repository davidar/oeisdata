## Python
```Python
for n in range(1,101):
    if n == 1:
        a = 1
        b = 2
    else:
        a = a*(4*n-5)*(4*n-3)
        b = b*(4*n-4)*(4*n-2)
    c = (a+b)/(4*n-1)
    print(str(n)+" "+str(c))
```

## Python
```Python
from sympy import isprime
def afind(limit):
    tenk = 10**10
    for k in range(10, limit+1):
        if isprime(tenk - 1000000001): print(k, end=", ")
        tenk *= 10
afind(100000) # _Michael S. Branicky_, Nov 18 2021
```

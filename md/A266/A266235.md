## Python
```Python
from sympy import isprime
a=[]
TOP=1000000
for p in range(TOP):
    if isprime(p):
        q=p
        while q<TOP:
            q = q*q+1
            if isprime(q):
                a.append(q)
print(sorted(set(a)))
```

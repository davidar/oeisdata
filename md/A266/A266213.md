## Python
```Python
from sympy import binomial
def T(n, r):
    if r==0: return 1
    return sum(binomial(r - 1, k - 1) * binomial(n - r, k) * 2**k for k in range(min(n - r, r) + 1))
for n in range(11): print([T(n, r) for r in range(n + 1)]) # _Indranil Ghosh_, May 23 2017
```

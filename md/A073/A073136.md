## Python
```Python
from sympy import prime, sieve
def aupton(terms):
  p = [None] + list(sieve.primerange(1, prime(prime(terms))+1))
  return [p[n] + p[p[n]] for n in range(1, terms+1)]
print(aupton(51)) # _Michael S. Branicky_, May 08 2021
```

## Python
```Python
from sympy import prime, primerange
def ok(p): s = str(p); return len(set(s)) < len(s)
def aupto(limit):
  alst = []
  for pi, p in enumerate(primerange(1, prime(limit)+1), start = 1):
    if ok(p): alst.append(pi)
  return alst
print(aupto(186)) # _Michael S. Branicky_, May 26 2021
```

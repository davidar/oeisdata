## Python
```Python
from sympy import primefactors, nextprime
def ok(n):
    pf = primefactors(n)
    return sum(p2 != nextprime(p1) for p1, p2 in zip(pf[:-1], pf[1:])) == 1
print(list(filter(ok, range(1, 130)))) # _Michael S. Branicky_, Oct 14 2021
```

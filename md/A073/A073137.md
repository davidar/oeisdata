## Python
```Python
def a(n):
    b = bin(n)[2:]; z = b.count('0'); w = len(b) - z
    return int('1'*(w > 0) + '0'*z + '1'*(w-1), 2)
print([a(n) for n in range(73)]) # _Michael S. Branicky_, Jun 26 2021
```

## Python
```Python
from sympy import isprime
def cond(k, n): m = (k**n)//2; return isprime(m-1) and isprime(m+1)
def a(n):
    k = 2
    while not cond(k, n): k += 2
    return k
print([a(n) for n in range(1, 25)]) # _Michael S. Branicky_, Aug 06 2021
```

## Python
```Python
def aupto(limit):
    s, alst = "", []
    for k in range(1, limit+1):
        ss = str(k*k)
        if ss in s: alst.append(k)
        s += ss
    return alst
print(aupto(28028)) # _Michael S. Branicky_, Jul 08 2021
```

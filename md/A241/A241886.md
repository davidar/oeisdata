## Python
```Python
import sympy
n = 1
while n >= 0 and n <= 10:
    p = 2
    while sympy.nextprime(p) - p != (n * (n + 1)):
        p = sympy.nextprime(p)
    print(p, end=", ")
    n = n + 1
    p = sympy.nextprime(p)
```

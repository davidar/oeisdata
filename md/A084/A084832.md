## Python
```Python
from sympy import isprime
def afind(limit):
  n, twoRn = 1, 2
  for n in range(1, limit+1):
    if isprime(twoRn-1): print(n, end=", ")
    twoRn = 10*twoRn + 2
afind(700) # _Michael S. Branicky_, Apr 18 2021
```

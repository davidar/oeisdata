## Python
```Python
from sympy import primefactors
def ok(n):
    pf = set(primefactors(n))
    return pf & {2, 3, 5, 7} and pf - {2, 3, 5, 7}
print(list(filter(ok, range(147)))) # _Michael S. Branicky_, Oct 15 2021
```

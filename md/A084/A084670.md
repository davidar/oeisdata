## Python
```Python
from sympy import isprime, prime
def aupto(lim):
  return [k for k in range(1, lim+1) if isprime(int(str(prime(k))+str(k)))]
print(aupto(837)) # _Michael S. Branicky_, Mar 09 2021
```

## Python
```Python
from numpy import convolve
from sympy import prime, primerange
def aupton(terms):
    p = list(primerange(3, prime(terms+1)+1))
    return list(convolve(p, p))[:terms]
print(aupton(40)) # _Michael S. Branicky_, Sep 30 2021
```

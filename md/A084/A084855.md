## Python
```Python
def T(n, k): return int(str(k) + str(n))
def auptorow(maxrow):
    return [T(n, k) for n in range(1, maxrow+1) for k in range(1, n+1)]
print(auptorow(11)) # _Michael S. Branicky_, Nov 21 2021
```

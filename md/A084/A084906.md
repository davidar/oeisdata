## Python
```Python
def ok(n):
    s = str(n)
    pairs = ((int(s[:i]), int(s[i:])) for i in range(1, len(s)))
    return any(c%d == 0 for c, d in pairs if d > 0)
print([k for k in range(214) if ok(k)]) # _Michael S. Branicky_, Nov 21 2021
```

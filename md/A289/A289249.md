## Python
```Python
from sympy import fibonacci, npartitions
def a(n): return npartitions(n) + sum([fibonacci(k)*npartitions(n - 2 - k) for k in range(1, n - 1)])
print([a(n) for n in range(51)]) # _Indranil Ghosh_, Jun 29 2017
```

## Python
```Python
from gmpy2 import is_square
from mpmath import *
mp.dps=100
def ok(n): return is_square(int(nint(floor(exp(1)*n + pi))))
print([n for n in range(1, 6001) if ok(n)]) # _Indranil Ghosh_, Jul 15 2017
```

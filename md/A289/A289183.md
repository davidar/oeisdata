## Python
```Python
from sympy import harmonic
def a(n):
  hn2 = 2 * harmonic(n)
  m = n
  while harmonic(m) <= hn2: m += 1
  return m - 1
print([a(n) for n in range(1, 49)]) # _Michael S. Branicky_, Mar 10 2021
```

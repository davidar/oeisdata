## Python
```Python
from sympy import gcd, primepi, primefactors
def ok(n): return gcd([primepi(p) for p in primefactors(n)]) == 1
print([n for n in range(1, 151) if ok(n)]) # _Indranil Ghosh_, Aug 06 2017
```

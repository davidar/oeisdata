## Python
```Python
from sympy import divisors, floor
def a(n): return int(n*max(int(floor((d - 2)/4) + 1)/d for d in divisors(n)))
print([a(n) for n in range(2, 101)]) # _Indranil Ghosh_, Jul 08 2017
```

## Python
```Python
from sympy import factorial, isprime, Rational
def a(n): return factorial(n) * sum(Rational(1, c) for c in range(4, n+1) if not isprime(c))
print([a(n) for n in range(4, 23)]) # _Michael S. Branicky_, Jun 30 2021
```

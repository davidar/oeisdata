## Python
```Python
from sympy import nextprime
def a(n):
    m = 1
    while nextprime(n*m-1) <= n*(m+1): m += 1
    return m
print([a(n) for n in range(1, 58)]) # _Michael S. Branicky_, Aug 04 2021
```

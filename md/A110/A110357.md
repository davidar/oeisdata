## Python
```Python
from sympy.abc import x, y
from sympy.solvers.diophantine.diophantine import diop_quadratic
def A110357(n): return min(int(x) for x,y in diop_quadratic(n*(n+y)-x*(n-y)) if x>0 and y>0) # _Chai Wah Wu_, Oct 06 2023
```

## Python
```Python
def aupto(limit):
    cands = range(1, int(limit**.5)+2)
    nums = [a**2 + 7*b**2 for a in cands for b in cands]
    return sorted(set(k for k in nums if k <= limit))
print(aupto(268)) # _Michael S. Branicky_, Aug 11 2021
```

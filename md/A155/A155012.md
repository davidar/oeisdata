## Mathematica
```Mathematica
a={};Do[f=Fibonacci[n];If[PrimeQ[f],If[PrimeQ[3*f+2],AppendTo[a,f]]],{n,4*6!}];a
```
## Python
```Python
from gmpy2 import is_prime
A155012_list = []
a, b, a2, b2 = 0, 1, 2, 5
for _ in range(10**3):
    if is_prime(b) and is_prime(b2):
        A155012_list.append(b)
    a, b, a2, b2 = b, a+b, b2, a2+b2-2 # _Chai Wah Wu_, Nov 04 2015
```

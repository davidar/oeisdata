## Python
```Python
def a(n):
    locs = [e for e in range(n.bit_length()) if 1 & (n>>e)]
    diffs = set(abs(e1-e2) for i, e1 in enumerate(locs) for e2 in locs[i:])
    return sum(2**d for d in diffs)
print([a(n) for n in range(71)]) # _Michael S. Branicky_, Oct 16 2021
```

## Python
```Python
from sympy import divisors
def a(n):
    return 1 - any(n/2 <= d*d < 2*n for d in divisors(n, generator=True))
print([a(n) for n in range(1, 106)]) # _Michael S. Branicky_, Oct 14 2021
```

## Python
```Python
def a(n):
  lo, hi = 10**(n-1), 10**n
  return len(set(a*b for a in range(7, hi//7+1, 10) for b in range(a, hi//a+1, 10) if lo <= a*b < hi))
print([a(n) for n in range(1, 9)]) # _Michael S. Branicky_, Sep 26 2021
```

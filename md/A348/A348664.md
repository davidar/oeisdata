## Python
```Python
def ispal(s): return s == s[::-1]
def ok(n):
  s = bin(n)[2:]
  return len(s) >= 1 + len(set(s[i:j] for i in range(len(s)) for j in range(i+1, len(s)+1) if ispal(s[i:j])))
print([k for k in range(870) if ok(k)]) # _Michael S. Branicky_, Oct 29 2021
```

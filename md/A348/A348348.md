## Python
```Python
from sympy import isprime,nextprime
def A348348(n):
    p = 2
    while 1:
        p_next = nextprime(p)
        if p_next == p+2 and all(isprime(j*(p+1)-1) and isprime(j*(p+1)+1) for j in range(2,n+1)):
            return p+1
        p = p_next
```

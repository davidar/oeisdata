# Minimal absolute value of determinant of a nonsingular n X n symmetric Toeplitz matrix using the first n prime numbers\.

## Python
```Python
from itertools import permutations
from sympy import Matrix, prime
def A348891(n): return min(d for d in (abs(Matrix([p[i:0:-1]+p[0:n-i] for i in range(n)]).det()) for p in permutations(prime(i) for i in range(1,n+1))) if d > 0) # _Chai Wah Wu_, Jan 28 2022
```

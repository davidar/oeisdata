# The list of all prime numbers is split into sublists with the 1st sublist L\_1 \= \{2\} and n\-th sublist L\_n \= \{p\_1, p\_2, \.\.\., p\_m\}\. a\(n\) is the largest m such that the maximum prime gap in L\_n is < p\_1 \- prevprime\(p\_1\)\.

## Python
```Python
from sympy import nextprime
L = [2]
for n in range(1, 50):
    print(len(L), end = ', ')
    p0 = L[-1]; p1 = nextprime(p0); g0 = p1 - p0; M = [p1]; p = nextprime(p1)
    while p - p1 < g0: M.append(p); p1 = p; p = nextprime(p)
    L = M
```

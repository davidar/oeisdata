## Python
```Python
from sympy import Matrix
A348428_list = []
for n in range(1,10**6):
    s = [int(d) for d in str(n)]
    m = len(s)
    if n == Matrix(m, m, lambda i, j: s[(i+j) % m]).det():
        A348428_list.append(n)
```

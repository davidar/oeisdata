## Python
```Python
from sympy import divisors
terms = [1]
for i in range(100):
    for j in divisors(terms[-1]):
        if j not in terms:
            terms.append(j)
            break
    else:
        terms.append(terms[-1]*3-1)
print(terms) # _Gleb Ivanov_, Nov 09 2021
```

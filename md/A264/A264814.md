## Python
```Python
from sympy import isprime
def aupto(N):
  return [k for k in range(1, N+1, 2) if isprime(int(str(k)+str(k+1)+str(k)))]
print(aupto(530)) # _Michael S. Branicky_, Jul 09 2021
```

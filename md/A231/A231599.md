## Python
```Python
from sympy import poly, symbols
def a231599_row(n):
    if n == 0:
        return [1]
    x = symbols('x')
    p = 1
    for i in range(1, n+1):
        p *= poly(1-x**i)
    p = p.all_coeffs()
    return p[::-1]
# _Tilman Piesk_, Feb 21 2016
```

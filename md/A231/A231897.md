## Python
```Python
def wt(n): return bin(n).count('1')
def a(n):
    m = 2**(n//2) - 1
    while wt(m**2) != n: m += 1
    return m
print([a(n) for n in range(32)]) # _Michael S. Branicky_, Feb 06 2022
```

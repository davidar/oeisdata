## Python
```Python
def ispal(s): return s == s[::-1]
def a(n):
  s = bin(n)[2:]
  return 1 + len(set(s[i:j] for i in range(len(s))
    for j in range(i+1, len(s)+1) if ispal(s[i:j])))
print([a(n) for n in range(105)]) # _Michael S. Branicky_, Feb 02 2021
```

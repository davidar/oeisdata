## Maple
```Maple
mx:= 10000: # maximal index needed
b:= proc(n) n end:
a:= proc(n) option remember; global mx; local h, t;
      if n=0 then 0 else a(n-1); t:= b(n);
        if n+t<=mx then h:=b(t+n); b(t+n):=h+n fi; t
      fi
    end:
seq(a(n), n=1..100);  # _Alois P. Heinz_, Mar 04 2015
```

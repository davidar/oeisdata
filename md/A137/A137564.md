## Python
```Python
def a(n):
    seen, out, s = set(), "", str(n)
    for d in s:
        if d not in seen: out += d; seen.add(d)
    return int(out)
print([a(n) for n in range(113)]) # _Michael S. Branicky_, Jul 23 2022
```

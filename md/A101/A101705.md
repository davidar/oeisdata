## Python
```Python
def A101705(n):
    if n == 1: return 0
    a = 1<<n.bit_length()-2
    s = bin(a|(n&a-1))[2:]
    return 540*int(s+(s[::-1] if a&n else s[-2::-1])) # _Chai Wah Wu_, Jul 23 2024
```

## Python
```Python
from math import isqrt
from sympy import prime, primerange
def aupton(terms):
    return [isqrt(2*p) - isqrt(p-1) for p in primerange(1, prime(terms)+1)]
print(aupton(103)) # _Michael S. Branicky_, Jul 22 2021
```

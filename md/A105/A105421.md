## Python
```Python
def A105421(max) :
    terms = []
    p = 3
    for n in range(1, max+1) :
        if (p - 7) % n == 0 : terms.append(n)
        p = nextprime(p)
    return terms
# Eric M. Schmidt, Feb 05 2013
```

## Python
```Python
from math import prod
def b(n): return n + prod(map(int, str(n)))
def aupto(n): return sorted(set(range(n+1)) - set(b(m) for m in range(n+1)))
print(aupto(183)) # _Michael S. Branicky_, Jan 09 2023
```

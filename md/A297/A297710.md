## Python
```Python
from sympy import npartitions
A297710_list = []
for i in range(1,10**6):
    n = npartitions(i)
    s = [int(d) for d in str(n)]
    for j in range(len(s)-1):
        if not (s[j]+s[j+1]) % 2:
            break
    else:
        A297710_list.append(n) # _Chai Wah Wu_, Jan 04 2018
```

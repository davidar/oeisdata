## Python
```Python
from sympy import factorint, primepi
def ok(n):
    f = factorint(n)
    return all(f[i] <= primepi(i) for i in f)
print([n for n in range(1, 151) if ok(n)]) # _Indranil Ghosh_, Jun 24 2017
```

## Python
```Python
from sympy import isprime, primerange
def aupto(N): return [t for t in (2*p for p in primerange(2, N//2+1)) if isprime(t-1) or isprime(t+1)]
print(aupto(1322)) # _Michael S. Branicky_, Aug 21 2022
```

## Python
```Python
from math import prod
from sympy import factorint
A287055_list, a, n = [], 1, 1
while n < 10**5:
    b = prod(p**e-1 for p, e in factorint(n+1).items())
    if a == b:
        A287055_list.append(n)
    a, n = b, n+1 # _Chai Wah Wu_, Sep 24 2021
```

## Python
```Python
from sympy import prime
def a(n):
  iter = pn = prime(n)
  for i in range(1, pn): iter = prime(iter)
  return iter
print([a(n) for n in range(1, 6)]) # _Michael S. Branicky_, Jun 03 2021
```

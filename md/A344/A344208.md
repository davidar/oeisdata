## Python
```Python
def f(x): return sum(map(int, str(x)))**2 + 1
def ok(n):
  iter = f(n)  # set to n for number of iterations >= 0
  while iter > 9:
    if iter in {65, 122, 26}: return False
    iter = f(iter)
  return True
print(list(filter(ok, range(1, 128)))) # _Michael S. Branicky_, May 14 2021
```

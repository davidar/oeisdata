## Python
```Python
from sympy import isprime
def istwin(m): return 1 if isprime(6*m-1)*isprime(6*m+1) == 1 else 0
ct1 = 0
for n in range(1, 100):
    ct1 += istwin(n); ct = 0
    for m in range (n + 1, n + ct1 + 1): ct += istwin(m)
    print(ct)
```

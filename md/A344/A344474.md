## Python
```Python
for z in range (0, 10):
    z_s = str(z)
    counts=0
    for x in range (0,1000000000):
        x_s = str(x)
        if z_s in x_s:
            counts += 1
        if counts / (x+1) == 0.5:
            print(x)
            break
```

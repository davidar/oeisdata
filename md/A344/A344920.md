## Mathematica
```Mathematica
W[n_, k_] := (-1)^k k! StirlingS2[n + 1, k + 1];
```
```Mathematica
WT[a_, len_] := Table[Sum[W[n, k] a[k], {k, 0, n}], {n, 0, len-1}];
```
```Mathematica
WT[#^2 &, 32] (* The Worpitzky transform applied to the squares. *)
```

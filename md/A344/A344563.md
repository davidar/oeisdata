## Python
```Python
from math import comb
def T(n, k):
    return comb(n-1, k-1)*comb(n, k)*2**k if k > 0 else k**n
print([T(n, k) for n in range(10) for k in range(n+1)]) # _Michael S. Branicky_, May 30 2021
```

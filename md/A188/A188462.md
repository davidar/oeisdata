## Python
```Python
from itertools import count
from sympy.solvers.diophantine.diophantine import power_representation
def A188462(n):
    if n == 1: return 1
    for k in count(1):
        try:
            next(power_representation(n,5,k))
        except:
            continue
        return k # _Chai Wah Wu_, Jun 25 2024
```

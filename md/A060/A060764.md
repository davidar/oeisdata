## Python
```Python
from itertools import pairwise
from sympy import divisors
def A060764(n):
    e = map(lambda x:x[1]-x[0],pairwise(d:=divisors(n)))
    return len(set(d)-set(e)) # _Chai Wah Wu_, Feb 21 2023
```

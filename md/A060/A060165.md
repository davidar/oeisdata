## Python
```Python
from sympy import mobius, binomial, divisors
def a(n): return sum(mobius(n//d) * binomial(2*d, d) for d in divisors(n))//n
print([a(n) for n in range(1, 31)])  # _Indranil Ghosh_, Jul 24 2017
```

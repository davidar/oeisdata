## Python
```Python
from sympy import isprime, primerange
def ok(p): return not isprime(int(bin(p)[:1:-1], 2))
def aupto(lim): return [p for p in primerange(2, lim+1) if ok(p)]
print(aupto(1000)) # _Michael S. Branicky_, Feb 19 2021
```

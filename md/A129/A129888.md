## Python
```Python
def next2(n): sd = sum(map(int, str(n))); return [sd, n+sd]
def aupton(terms):
    alst = [10]
    while len(alst) < terms: alst.extend(next2(alst[-1]))
    return alst[:terms]
print(aupton(66)) # _Michael S. Branicky_, Oct 01 2021
```

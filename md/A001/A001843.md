## Maple
```Maple
A001843 :=  proc(n)
    floor((n-1)/3* floor((n-2)/2) ) ;
    if modp(n,6) = 0 then
        floor(n*(%-1)/4) ;
    else
        floor(n*%/4) ;
    end if;
end proc:
seq(A001843(n),n=4..80) ; # _R. J. Mathar_, Oct 01 2021
```

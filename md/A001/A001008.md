## Sage
```Sage
def harmonic(a, b):  # See the F. Johansson link.
    if b - a == 1:
        return 1, a
    m = (a+b)//2
    p, q = harmonic(a,m)
    r, s = harmonic(m,b)
    return p*s+q*r, q*s
def A001008(n): H = harmonic(1,n+1); return numerator(H[0]/H[1])
[A001008(n) for n in (1..29)] # _Peter Luschny_, Sep 01 2012
```

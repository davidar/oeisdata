## Python
```Python
def A001468(length):
    a = [1]
    for i in range(length):
        for _ in range(a[i]):
            a.append(2)
        a.append(1)
        if len(a)>=length:
            break
    return a[:length] # _Nicholas Stefan Georgescu_, Jun 02 2022
```

## Python
```Python
import math
def getSeq(n):
....if n == 1:
........return [2]
....prev = getSeq(n-1)
....cand = max(prev)
....while True:
........cand += 1
........if len( [n for n in prev if cand % n == 0] ) % 4 == 0:
............prev.append(cand)
............return prev
print(getSeq(100))
```

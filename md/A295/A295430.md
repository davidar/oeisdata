## Maple
```Maple
f:= proc(n) local m,k;
  for m from 0 do
    k := max(2, ceil(3*10^m/n));
    if k*n < 4*10^m then return k*n end if
  end do
end proc:
seq(f(i),i=1..100);
```

```Python
def A295430(n):
    m = 2*n
    while True:
        if str(m)[0] == '3':
            return m
        m += n # _Chai Wah Wu_, Feb 13 2018
```

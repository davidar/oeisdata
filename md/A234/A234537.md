## Python
```Python
from sympy import isprime
def a(n): return sum(1 for k in range(3, n + 1, 2) if not isprime(k) or not isprime(2*n - k))
print([a(n) for n in range(1, 101)]) # _Indranil Ghosh_, Jul 11 2017
```

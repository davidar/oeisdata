## Python
```Python
from sympy import prime
def a(n):
    s = "".join(sorted(set(str(prime(n)))))
    return int(s) if "0" not in s else int(s[1] + "0" + s[2:])
print([a(n) for n in range(1, 63)]) # _Michael S. Branicky_, Sep 12 2022
```

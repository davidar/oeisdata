## Python
```Python
from sympy import isprime
def sw(n): return isprime(n-1) and isprime(n+1)
def ok(n): return n%2 == 0 and sw(n) and sw(n*int(str(n)[::-1]))
print([k for k in range(10**5) if ok(k)]) # _Michael S. Branicky_, Sep 10 2022
```

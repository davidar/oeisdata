## Python
```Python
from sympy import factorint
def isA033992(n): return len(factorint(n)) == 3
def ok(n): return isA033992(n-1) and isA033992(n+1)
print([k for k in range(700) if ok(k)]) # _Michael S. Branicky_, Sep 10 2022
```

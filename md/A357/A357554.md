## Maple
```Maple
g:= proc(n,k) local x,S;
  S:= {k};
  x:= k;
  do
     x:= iquo(n,x) + irem(n,x);
     if member(x,S) then return x fi;
     S:= S union {x};
  od
end proc:
for n from 1 to 20 do seq(g(n,k),k=1..n) od;
```

## Python
```Python
def T(n, k):
    seen, x = set(), k
    while x not in seen: seen.add(x); q, r = divmod(n, x); x = q + r
    return x
print([T(n, k) for n in range(1, 14) for k in range(1, n+1)]) # _Michael S. Branicky_, Oct 04 2022
```

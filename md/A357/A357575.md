## Python
```Python
from math import isqrt
from sympy import convex_hull
def A357575(n): return int(2*convex_hull(*[(n,0),(0, 0)]+[(x, isqrt((n-x)*(n+x))) for x in range(n)]).area) if n else 0 # _Chai Wah Wu_, Oct 23 2022
```

## Python
```Python
from sympy import sieve, isprime
def leastk(p, plusminus):
    k=1
    while not isprime(k * p + plusminus): k += 1
    return k
print([p for p in sieve[1:1000] if leastk(p, 1) == leastk(p, -1)])
```

## Python
```Python
from math import isqrt
from itertools import count
from sympy import isprime
def A357899(n): return next(filter(isprime, ((m:=isqrt(k*n))+ int((k*n-m*(m+1)<<2)>=1) for k in count(1)))) # _Chai Wah Wu_, Oct 19 2022
```

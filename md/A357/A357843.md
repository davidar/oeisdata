## Python
```Python
from fractions import Fraction
from sympy import divisor_count
def A357843(n): return sum(Fraction(1 if k&1 else -1, divisor_count(k)) for k in range(1,n+1)).numerator # _Chai Wah Wu_, Oct 16 2022
```

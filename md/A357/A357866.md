## Python
```Python
from sympy.ntheory import digits
def a(n): return max((n%sum(digits(n, b)[1:]) for b in range(2, n+1)), default=0)
print([a(n) for n in range(1, 72)]) # _Michael S. Branicky_, Oct 17 2022
```

## Maple
```Maple
g:= proc(n, k) local x, S;
  S:= {k};
  x:= k;
  do
     x:= iquo(n, x) + irem(n, x);
     if member(x, S) then return x fi;
     S:= S union {x};
  od
end proc:
seq(g(n,3), n=1..100);
```

## Python

def a(n):
    seen, x = set(), 3
    while x not in seen: seen.add(x); q, r = divmod(n, x); x = q + r
    return x

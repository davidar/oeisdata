## Python
```Python
from sympy import factorint
def isA007774(n): return len(factorint(n)) == 2
def ok(n): return isA007774(n-1) and isA007774(n+1)
print([k for k in range(200) if ok(k)]) # _Michael S. Branicky_, Sep 10 2022
```

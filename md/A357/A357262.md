## Python
```Python
from math import prod
from sympy import primefactors
def ok(n): return n and prod(map(int, set(str(n)))) == sum(primefactors(n))
print([k for k in range(10**5) if ok(k)]) # _Michael S. Branicky_, Sep 21 2022
```

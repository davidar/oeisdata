## Python
```Python
pal10 = lambda n: str(n) == str(n)[::-1]
def seq(n):
    k = 1
    while not pal10(k*(2*n+k-1)//2):k+=1
    return k*(2*n+k-1)//2
print([seq(n) for n in range(1, 100)])
```

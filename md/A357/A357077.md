## Maple
```Maple
R:= NULL: count:= 0: state:= 0:
for n from 1 while count < 100 do
  if numtheory:-bigomega(n) >= 3 then
     if state = 1 then R:= R, n-1; count:= count+1
     else state:= 1
     fi
  else state := 0
  fi
od:
R; # _Robert Israel_, Sep 16 2022
```

## Python
```Python
b = [3, 1, 2, 2, 3, 2, 3, 4, 3, 2]
def a(n): return sum(b[d] for d in map(int, str(n)))
print([a(n) for n in range(100)]) # _Michael S. Branicky_, Mar 01 2021
```

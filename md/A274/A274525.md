## Python
```Python
from sympy import isprime, primerange
def aupto(n):
    t = []
    for p in primerange(2, n+1):
        if isprime(p-2) and isprime(p**2 - p - 1) and isprime(p**2 - p + 1):
            t.append(p)
    return t # _Paul Muljadi_, Jun 21 2024
```

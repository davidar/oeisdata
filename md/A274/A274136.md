## Python
```Python
# subroutine
def B (n, k, a, b):
    pa = pb = 1
    for i in range(n+1):
        pa *= (i*k+a)
        pb *= (-i*k-b)
    m = n*k+a+b
    p = pa-pb
    if m == 0:
        return "NaN"
    else:
        return p/m
# main program
for j in range(101):
    print(str(j)+" "+str(B(2*j+1, 1, 2, 1)))  # _Hong-Chang Wang_, Jun 14 2016
```

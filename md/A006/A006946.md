## Python
```Python
import networkx as nx
def deBruijn(n):
    return nx.MultiDiGraph(((0, 0), (0, 0))) if n==0 else nx.line_graph(deBruijn(n-1))
def A006946(n):
    return nx.max_weight_clique(nx.complement(nx.Graph(deBruijn(n))),weight=None)[1] #_Pontus von Br√∂mssen_, Mar 07 2020 (updated Nov 12 2023)
```

## Python
```Python
from num2words import num2words
def a(n): return sum(1 for c in num2words(n, to='ordinal').replace(" and", "") if c.isalpha())
print([a(n) for n in range(1, 77)]) # _Michael S. Branicky_, Aug 08 2021 edited Jul 12 2022
```

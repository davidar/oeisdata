## Python
```Python
def a(n):
    k, target = 1, pow(n, n, 10)
    while pow(n+k, n+k, 10) != target: k += 1
    return k
print([a(n) for n in range(1, 82)]) # _Michael S. Branicky_, Oct 16 2021
```

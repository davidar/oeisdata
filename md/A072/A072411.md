## Python
```Python
from sympy import lcm, factorint
def a(n):
    l=[]
    f=factorint(n)
    for i in f: l+=[f[i],]
    return lcm(l)
print([a(n) for n in range(1, 151)]) # _Indranil Ghosh_, Mar 25 2017
```

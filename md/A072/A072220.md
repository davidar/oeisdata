## Python
```Python
def a(n, multiple_limit=300):
    fk, limit = 1, multiple_limit*n
    for k in range(1, limit+1):
        fk *= k
        if str(fk).count("9") == n: return k
    return 0
print([a(n) for n in range(1, 57)]) # _Michael S. Branicky_, Dec 11 2021
```

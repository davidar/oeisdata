## Python
```Python
from sympy import fibonacci
def a(n):
    if n<1: return 0
    m=0
    while fibonacci(m)<=n: m+=1
    return m-2
print([a(n) for n in range(1, 101)]) # _Indranil Ghosh_, Jun 09 2017
```

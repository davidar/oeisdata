## Python
```Python
from math import isqrt
from sympy import mobius
def A072490(n): return int(sum(mobius(k)*((n-1)//k**2) for k in range(1, isqrt(n-1)+1)))-1 if n>1 else 0 # _Chai Wah Wu_, Aug 19 2024
```

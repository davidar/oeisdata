## Python
```Python
def a(n, bound=10**4):
    k = f = 1
    while not str(f).count('7') == n and k <= bound: k += 1; f *= k
    return k * (k <= bound)
print([a(n) for n in range(1, 149)]) # _Michael S. Branicky_, Jun 14 2021
```

## Python
```Python
def aupto(lim):
  pows, p = [], 1
  while p < lim: pows.append(p); p *= 13
  return sorted([a+b for a in pows for b in pows if b >= a and a+b <= lim])
print(aupto(63119810)) # _Michael S. Branicky_, Feb 14 2021
```

## Python
```Python
from math import factorial
def a(n, limit=1000):
  fact = 1
  for t in range(limit+1):
    if str(fact).count('4') == n: return t
    fact *= (t+1)
  return 0
print([a(n) for n in range(1, 57)]) # _Michael S. Branicky_, Apr 19 2021
```

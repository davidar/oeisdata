## Maple
```Maple
for k from 1 do
    p := ithprime(k) ;
    n := numtheory[cyclotomic](7,p) ;
    pn := numtheory[cyclotomic](7,n) ;
    if isprime( pn) then
        print(n) ;
    end if;
end do: # _R. J. Mathar_, Feb 07 2014
```

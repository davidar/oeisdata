## Python
```Python
prpr, prev = 0, 1
for n in range(2, 100000):
    prpr, prev = prev, prpr+prev
    if n%2: continue
    if (prev % n, prev % (n-1), prev % (n+1)) == (0, 0, 0):
        print(n, end=", ")
```

## Python
```Python
from sympy import isprime, primefactors
def ok(n):
    pf = primefactors(n)
    if len(pf) < 2: return False
    return isprime(int("".join(str(p) for p in pf)))
print(list(filter(ok, range(2, 220)))) # _Michael S. Branicky_, Jun 12 2021
```

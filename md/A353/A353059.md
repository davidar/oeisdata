## Python
```Python
from sympy import factorint
def ok(n): return set("".join(str(p)+str(e) for p, e in factorint(n).items())) < set(str(n))
print([k for k in range(2, 9999) if ok(k)]) # _Michael S. Branicky_, Apr 20 2022
```

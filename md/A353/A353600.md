## Python
```Python
def b(n, startk=2):
    k, n2 = startk, n*n
    while pow(k, n-1, n2) != 1: k += 1
    return k
def a(n): return b(n, startk=b(n)+1)
print([a(n) for n in range(2, 54)]) # _Michael S. Branicky_, Apr 29 2022
```

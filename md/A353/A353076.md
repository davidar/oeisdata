## Python
```Python
from sympy import divisor_sigma, EulerGamma, E, log
print([k for k in range(3, 600, 2) if divisor_sigma(k) > (E**EulerGamma * k * log(log(k)) / 2)]) # _Karl-Heinz Hofmann_, Apr 22 2022
```

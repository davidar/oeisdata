## Python
```Python
from sympy import isprime
for n in range(1,100):
    s = n*'1'; c = 0
    for j in range(n+1):
        num = int(s[:j]+'0'+s[j:], 2)
        if isprime(num):
            c += 1
            if c == 1: r = num
            if c == 2: break
    if c == 1: print(r, end = ', ')
```

## Maple
```Maple
f:= proc(n) local k;
  for k from 2 do
    if k mod 10 <> 0 and numboccur(0, convert(k^n,base,10)) >= n then return k fi
  od
end proc:
map(f, [$1..50]);
```

```PARI
a(n) = {
	forstep(i = 11, oo, [1,1,1,1,1,1,1,1,2],
		d = digits(i^n);	
		t = 0;
		for(j = 1, #d,
			t+=(!d[j])
		);
		if(t >= n,
			return(i)
		)
	)
```

## Python
```Python
from gmpy2 import digits
from itertools import count
def a(n): return next(k for k in count(1) if k%10 and digits(k**n).count('0')>=n)
print([a(n) for n in range(1, 46)]) # _Michael S. Branicky_, Jan 05 2024
```

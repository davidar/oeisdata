## Python
```Python
from math import prod
from sympy import factorint
from sympy.ntheory.primetest import is_square
def A368544(n): return prod(e+1 for p, e in factorint(n).items() if is_square(p-1)) # _Chai Wah Wu_, Dec 30 2023
```

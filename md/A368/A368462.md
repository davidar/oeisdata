## Python
```Python
from sympy import nextprime, isprime
p, s, k = 2, 0, 0
while k < 3*10**7:
    p = nextprime(p); s += p; k += 1; r = divmod(s, k)
    if r[1] == 0 and isprime(r[0]): print(r[0], end = ", ")
```

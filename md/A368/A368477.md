## Python
```Python
def a(n):
  dp = [1] * 10
  for c in str(n):
    bdigit = int(c)
    new_dp = [0] * 10
    for j in range(10):
      if bdigit != 0 and j + bdigit <= 9:
        new_dp[j] += dp[j + bdigit]
      if j - bdigit >= 0:
        new_dp[j] += dp[j - bdigit]
    dp = new_dp
  return sum(dp)
```

## Python
```Python
from math import prod
q = 2
N = lambda n, m : (prod([q**(2*i)-1 for i in range(n-m+1, n+1)])//prod([q**i-1 for i in range(1, m+1)]))
print([N(n, m) for n in range(8) for m in range(n+1)])
```

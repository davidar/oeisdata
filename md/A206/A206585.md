## Python
```Python
from sympy import continued_fraction_periodic
def A206585(n):
    i = 2
    while True:
        s = continued_fraction_periodic(0,1,i)[-1]
        if isinstance(s, list) and s.count(5) == n:
            return i
        i += 1 # _Chai Wah Wu_, Jun 10 2017
```

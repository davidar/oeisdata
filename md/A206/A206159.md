## Python
```Python
from sympy import divisors
def ok(n):
    digits_used = set()
    for d in divisors(n, generator=True):
        digits_used |= set(str(d))
        if len(digits_used) > 2: return False
    return True
print([k for k in range(1, 9000) if ok(k)]) # _Michael S. Branicky_, May 02 2022
```

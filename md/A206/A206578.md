## Python
```Python
from sympy import continued_fraction_periodic
def A206578(n):
    m = 1
    while True:
        s = continued_fraction_periodic(0,1,m)[-1]
        if isinstance(s,list) and s.count(1) == n:
            return m
        m += 1 # _Chai Wah Wu_, Jun 12 2017
```

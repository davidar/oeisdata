## Python
```Python
def DS(n):
    return sum(int(i) for i in str(n))
def a(n):
    k = 0
    nDSn = n * DS(n)
    while k <= n:
        if k**3 == nDSn:
            return True
        if k**3 > nDSn:
            return False
        k += 1
[n for n in range(10**5) if a(n)]
# Simplified by _Derek Orr_, Mar 22 2015
```

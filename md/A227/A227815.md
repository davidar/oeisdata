## Python
```Python
from sympy import isprime, factorint
def ok(n): return n > 1 and not isprime(n) and n%int("".join(str(e) for e in factorint(n).values())) == 0
print([k for k in range(1728) if ok(k)]) # _Michael S. Branicky_, Feb 19 2023
```

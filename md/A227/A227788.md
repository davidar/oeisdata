## Python
```Python
A003714 = [n for n in range(1, 300) if 2*n & n == 0]
print(0, end=',')
for a in A003714:
  sum = 0
  i = 2
  while a:
    if a&1:  sum += i
    a >>= 1
    i += 1
  print(sum, end=',')
```

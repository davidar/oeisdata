## Python
```Python
# see A052224 for a faster version if going to high numbers
from sympy.ntheory import digits
def ok(n): return sum(digits(n, 6)[1:]) == 6
print([k for k in range(487) if ok(k)]) # _Michael S. Branicky_, Nov 16 2021
```

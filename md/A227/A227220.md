## Python
```Python
from sympy import sqrt_mod_iter
def A227220(n):
    k, a = 10**n<<1, (10**n-1)//3<<2
    m = (a<<2)+1
    return min(b for b in ((d>>1)*((d>>1)+1) for d in sqrt_mod_iter(m, k) if d&1) if b%k==a)>>1 # _Chai Wah Wu_, May 04 2024
```

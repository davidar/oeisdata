## Maple
```Maple
filter:= proc(n)
  local L;
  L:= convert(n,base,10);
  if has(L,0) then return false fi;
  verify(convert(convert(L,`*`),base,10),L,'sublist');
end proc:
select(filter, [$1..1000]); # _Robert Israel_, Aug 26 2014
```

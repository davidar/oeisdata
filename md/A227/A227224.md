## Python
```Python
def DS(n):
    s = 0
    for i in str(n):
        s += int(i)
    return s
for n in range(10**3):
    k = 0
    while k**2 <= n*DS(n):
        if k**2 == n*DS(n):
            print(n,end=', ')
            break
        else:
            k += 1
```

## Python
```Python
from sympy import jacobi_symbol as J
def a(n):
    i=1
    while True:
        if J(i, 2*n + 1)!=1: return i
        else: i+=1
print([a(n) for n in range(1, 103)]) # _Indranil Ghosh_, May 11 2017
```

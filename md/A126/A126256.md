## Python
```Python
def A126256(n):
    s, c = (1,), {1}
    for i in range(n):
        s = (1,)+tuple(s[j]+s[j+1] for j in range(len(s)-1)) + (1,)
        c.update(set(s))
    return len(c) # _Chai Wah Wu_, Oct 17 2023
```

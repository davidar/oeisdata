## Python
```Python
# Assuming offset (n, k) = (0, 0).
def T(n, k):
    if k >  n: return 0
    if k == 0: return 1
    return k*T(n - 1, k - 1) + (k + 1)*T(n - 1, k)
for n in range(9):
    print([T(n, k) for k in range(n + 1)])  # _Peter Luschny_, Apr 26 2022
```

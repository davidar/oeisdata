## Python
```Python
def seq():
    towers = [0]
    for i in range(1, 100):
        towers = sorted(towers)
        if i <= towers[0]:
            towers = [i] + towers
        else:
            towers = [towers[0] + i] + towers[1:]
        yield min(towers)
```

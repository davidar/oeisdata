## Python
```Python
from num2words import num2words
def a(n):
    x = num2words(n).replace(' and ', '')
    l = [chr(i) for i in range(97, 123)]
    return sum(1 for i in x if i in l)
print([a(n) for n in range(101)]) # _Indranil Ghosh_, Jul 05 2017
```

## Python
```Python
def ok(n): b = bin(n)[2:]; return b==b[::-1] and n%sum(map(int, b)) == 0
def aupto(nn): return [m for m in range(1, nn+1) if ok(m)]
print(aupto(1371877)) # _Michael S. Branicky_, Jan 21 2021
```

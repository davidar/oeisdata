f:= proc(n) local t, v, k;
   t:= 2^n; v:= -t+1;
   for k from 0 do
      v:= v+2*t;
      if isprime(v) then return k fi
   od
end proc:

## Python
```Python
from itertools import count
from sympy import isprime
def pow2p1() : # generates the sequence
  for n in count() :
    for k in count() :
      if isprime(((2*k+1)<<n)+1) :
        yield k
        break
```

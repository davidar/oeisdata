## PARI
```PARI
f(n) = n^4 + (n+1)^4;
a(n) = lift(1/Mod(f(n+1), f(n)));
```
## Python
```Python
import gmpy2
soc = [] # sum of 4d-centered cubes
a=0
b=1
for i in range(100):
    c = a**4 + b**4
    soc.append(c)
    a += 1
    b += 1
A334137 = []
for i in range(len(soc)-1):
    c = gmpy2.invert(soc[i+1], soc[i])
    A334137.append(int(c))
print(', '.join([str(x) for x in A334137]))
```

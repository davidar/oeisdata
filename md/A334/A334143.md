## Python
```Python
def NORprime(n):
    s = str(bin(primes[n]))[2:]
    t = str(bin(primes[n-1]))[2:]
    k = (len(s) -  len(t))
    t = k*'0' + t
    r = ''
    for i in range(len(s)):
        if s[i] == t[i] and s[i] == '0':
            r += '1'
        else:
            r += '0'
    return int(r,2)
```

## Python
```Python
list1 = []
list2 = []
nmax = 100
for i in range(1, nmax+1):
    list1.append(i*i)
    for j in range(1, i+1):
        for k in range(j+1, i+1):
            m = j*k
            if m in list1:
                list1.remove(m)
                list2.append(m)
    print(i, len(list2))
```

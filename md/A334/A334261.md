## Python
```Python
from sympy import factorint
def d(n): return sum(n*e//p for p, e in factorint(n).items())
def ok(m): return m > 1 and 2*d(d(m)) - d(m) == 4
print([k for k in range(1, 999) if ok(k)]) # _Michael S. Branicky_, Aug 15 2022
```

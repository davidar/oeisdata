## Python
```Python
from math import factorial, prod
def ok(n):
    d = list(map(int, str(n)))
    return sum(map(int, str(prod(map(factorial, d))))) == sum(d)
print([k for k in range(1042) if ok(k)]) # _Michael S. Branicky_, Aug 15 2022
```

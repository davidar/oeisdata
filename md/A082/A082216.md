## Python
```Python
def A082216(n):
    s = str(n)
    t = s[::-1]
    if s == t:
        return n
    for i in range(1,len(s)):
        if s[i:] == t[:-i]:
            return int(s+t[-i:]) # _Chai Wah Wu_, Apr 13 2021
```

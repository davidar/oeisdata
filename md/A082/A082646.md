## Python
```Python
from sympy import prime
A082646_list = []
for i in range(1,10**5):
    p = str(prime(i))
    h = [p.count(d) for d in '0123456789' if d in p]
    if min(h) == max(h):
        A082646_list.append(int(p)) # _Chai Wah Wu_, Mar 06 2016
```

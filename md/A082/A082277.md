## Python
```Python
from sympy import isprime, nextprime, prime, primerange
def a(n):
  pn = prime(prime(n))
  smallest = list(primerange(2, pn+1))
  while not isprime(sum(smallest)):
    pn = nextprime(pn)
    smallest = smallest[1:] + [pn]
  return sum(smallest)
print([a(n) for n in range(1, 42)]) # _Michael S. Branicky_, May 23 2021
```

## Python
```Python
from sympy.ntheory import digits
def ok(n):
    d = digits(n, 3)[1:]
    return len(d) < 2*sum(1 for di in d if di%2 == 0)
print([k for k in range(117) if ok(k)]) # _Michael S. Branicky_, Jul 03 2022
```

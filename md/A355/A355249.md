## Python
```Python
from math import gcd
def a(n): return max(gcd(i, j, n-i-j) for i in range(1, n//3+1) for j in range(i, n//3+1))
print([a(n) for n in range(3, 85)]) # _Michael S. Branicky_, Jun 26 2022
```

## Python
```Python
def a(n):
    s, m = str(n), "9"
    return int("".join((m:=min(m, s[-1-k])) for k in range(len(s)))[::-1])
print([a(n) for n in range(69)]) # _Michael S. Branicky_, Jun 24 2022
```

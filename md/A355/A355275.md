filter:= proc(n) local L;
  L:= convert(n,base,3);
  numboccur(1,L) > nops(L)/2
end proc:

## Python
```Python
from sympy.ntheory import digits
def ok(n):
    d = digits(n, 3)[1:]
    return len(d) < 2*sum(1 for di in d if di%2)
print([k for k in range(293) if ok(k)]) # _Michael S. Branicky_, Jul 03 2022
```

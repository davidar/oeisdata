## Python
```Python
for k in range(1, 200):
    max_v = 0
    for a in range(1, k):
        for b in range(1, a+1):
            for c in range(1, b+1):
                if 2*a*b + 2*a*c + 2*b*c <= k and a*b*c > max_v:
                    max_v = a*b*c
    print(max_v, end = ', ')
```

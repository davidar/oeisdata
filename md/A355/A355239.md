## Python
```Python
def f(x): return 3*x+1 if x%2 else x//2
def ok(n):
    if n < 5: return False
    ni, targets = n, {1, n-1, n+1}
    while ni not in targets: ni = f(ni)
    return ni in {n-1, n+1}
print([k for k in range(10**4) if ok(k)]) # _Michael S. Branicky_, Jul 04 2022
```

## Python
```Python
from sympy import multiplicity as v
def a(n): return (v(2, n)+1)*(v(3, n)+1)*(v(5, n)+1)
print([a(n) for n in range(1, 87)]) # _Michael S. Branicky_, Jul 08 2022
```

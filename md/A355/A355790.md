## Python
```Python
from sympy import divisors
def ok(n):
    s, divs = str(n), divisors(n)[1:-1]
    return any(s in str(d)+str(n//d) for d in divs)
print([k for k in range(1, 10**5) if ok(k)]) # _Michael S. Branicky_, Jul 27 2022
```

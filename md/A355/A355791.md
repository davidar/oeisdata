## Python
```Python
from sympy import divisors
def ok(n):
    b, divs = bin(n)[2:], divisors(n)[1:-1]
    return any(b in bin(d)[2:]+bin(n//d)[2:] for d in divs)
print([k for k in range(1, 2400) if ok(k)]) # _Michael S. Branicky_, Jul 27 2022
```

```PARI
a(n, base=10) = { my (d=digits(n, base), m=-oo); forstep (k=#d, 1, -1, d[k]=m=max(m, d[k])); fromdigits(d, base) }
```

## Python
```Python
def a(n):
    s, m = str(n), "0"
    return int("".join((m:=max(m, s[-1-k])) for k in range(len(s)))[::-1])
print([a(n) for n in range(68)]) # _Michael S. Branicky_, Jun 24 2022
```

## Python
```Python
from sympy import fibonacci as F, floor
def T(n, k): return floor(F(n)/F(n - k))
for n in range(3, 16): print([T(n, k) for k in range(1, n - 1)]) # _Indranil Ghosh_, Jul 17 2017
```

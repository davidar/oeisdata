## Python
```Python
from itertools import count
from collections import Counter
def A062859(n):
    c = Counter()
    for m in count(1):
        for i in range(1,m+1):
            ij = i*m
            c[ij] += 1
            if c[ij]>=n:
                return m # _Chai Wah Wu_, Oct 16 2023
```

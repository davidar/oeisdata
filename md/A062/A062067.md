## Python
```Python
from sympy import isprime
A062067, a = [1], 1
for _ in range(1,10000):
    a += 1
    b = 2*a*(a-1) + 1
    while not isprime(b):
        b += 4*(a+1)
        a += 2
    A062067.append(a**2) # _Chai Wah Wu_, Sep 01 2014
```

## Python
```Python
import math
f=math.factorial
def C(n, r):return f(n)//f(r)//f(n-r)
i=-1
for n in range(26):
    for m in range(n+1):
        i += 1
        print(str(i)+" "+str(((-1)**m)*f(n)*C(n+5, n-m)//f(m))) # _Indranil Ghosh_, Feb 24 2017
```

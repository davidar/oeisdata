## Maple
```Maple
F:= proc(d) local r; # to get all terms with d digits
r:= 8*(10^d-1)/9;
op(sort(convert(map(t -> r + add(10^(j-1),j=t), combinat:-powerset(d) minus {{$1..d}}),list)))
end proc:
map(F, [$1..5]); # _Robert Israel_, Apr 05 2017
```

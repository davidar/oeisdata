## Python
```Python
from sympy import primefactors
def A(n):
     for i in primefactors(n):
         if i>min(primefactors(n))**2: return i
     return 1
def a(n): return 1 if A(n) == 1 else A(n)*a(n//A(n))
print([a(n) for n in range(1, 151)]) # _Indranil Ghosh_, Mar 24 2017
```

## Maple
```Maple
isA284292 := proc(n)
    if isprime(n) then
        convert(convert(n,base,10),set) ;
        if 8 in % then
            true;
        else
            false;
        end if;
    else
        false;
    end if;
end proc:
for n from 1 to 2000 do
    if isA284292(n) then
        printf("%d,",n) ;
    end if;
end do: # _R. J. Mathar_, Mar 27 2017
```

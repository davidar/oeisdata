## Python
```Python
from sympy.ntheory import primefactors
l=[]
for x in range(2, 145):
    for y in range(2, x):
        if primefactors(x + y + 1) == sorted(primefactors(x) + primefactors(y)):
            l+=[x, y]
print(l) # _Indranil Ghosh_, Mar 21 2017
```

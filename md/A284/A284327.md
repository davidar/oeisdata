## Python
```Python
from sympy import isprime
def a(n):
    k=0
    while True:
        if isprime(n**2 + k**2) and isprime(n**2 + (k - 2)**2): return k
        else: k+=1
print([a(n) for n in range(1, 101)]) # _Indranil Ghosh_, Mar 31 2017
```

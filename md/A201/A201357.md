## Python
```Python
from sympy import isprime
def afind(limit, startk=1):
    pow2 = 2**startk
    for k in range(startk, limit+1):
        if isprime((pow2 + k + 1)*pow2 - 1):
            print(k, end=", ")
        pow2 *= 2
afind(1500) # _Michael S. Branicky_, Jan 12 2022
```

## Python
```Python
from sympy import isprime
def agen(maxdigits):
    for digs in range(1, maxdigits+1):
        for i in range(digs):
            t = int("1"*(digs-1-i) + "2" + "1"*i)
            if not isprime(t): yield t
print(list(agen(9))) # _Michael S. Branicky_, Dec 21 2021
```

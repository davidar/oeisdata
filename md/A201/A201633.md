## Python
```Python
from functools import cache
@cache
def a(n):
    if n < 6: return [11, 28, 424, 1001, 14453][n-1]
    return a(n-1) + 34*a(n-2) - 34*a(n-3) - a(n-4) + a(n-5)
print([a(n) for n in range(1, 23)]) # _Michael S. Branicky_, Nov 28 2021
```

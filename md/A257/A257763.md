## Maple
```Maple
a:= proc(n) option remember; local k, s;
      for k from 1+`if`(n=1, 0, a(n-1)) do
        s:= {convert(k, base, 10)[]};
        if not 0 in s and s={convert(k^2, base, 10)[]}
           then return k fi
      od
    end:
seq(a(n), n=1..10);
```

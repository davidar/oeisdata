## Maple
```Maple
f:= proc(n) local m;
   if n::even then m:= 2*2^(n/2)+1
   else m:= ceil(sqrt(2)*2^((n-1)/2))
   fi;
   m^2-2^n
end proc:
map(f, [$0..100]); # _Robert Israel_, Mar 02 2016
```

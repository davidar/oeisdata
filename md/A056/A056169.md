## Python
```Python
from sympy import factorint
def a(n):
    f=factorint(n)
    return 0 if n==1 else sum(1 for i in f if f[i]==1)
print([a(n) for n in range(1, 101)]) # _Indranil Ghosh_, Jun 19 2017
```

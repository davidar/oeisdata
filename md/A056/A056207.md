## Python
```Python
from itertools import accumulate
def f(anm1, _): return anm1**2 + 4*anm1 + 3
def aupton(terms): return list(accumulate([3]*terms, f))
print(aupton(8)) # _Michael S. Branicky_, Mar 24 2021
```

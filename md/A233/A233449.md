## Python
```Python
# Assuming sequence starts with a(0) = 0 (see formula section).
def a(n):
    if n == 0: return 0
    s = f = 1
    for k in range(1, n):
        f *= k / 2
        s += f
    return round(2**(n - 1) * s)  # 'round' only to assure the integer type.
print([a(n) for n in range(24)])  # _Peter Luschny_, Mar 05 2024
```

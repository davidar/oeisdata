## Maple
```Maple
N:= 30: # to get terms up to a(N)
for n from 2 to N do
  for k from 1 do
     if numboccur(0,convert(k,base,10)) = 0 and
        numboccur(0,convert(k^n, base, 10)) = n then
          A[n]:= k;
          break
     fi
  od
od:
seq(A[n],n=2..N); # _Robert Israel_, Aug 05 2014
```

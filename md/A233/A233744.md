## Python
```Python
from sympy import factorial, Integer
angel=[0,0,1,1]
A233744=[1,2]
n = 20
for i in range(4,n):
    new = 1+sum(angel[:-1])/Integer(i-1)
    angel.append(new)
    A233744.append(new*factorial(i-1))
print(A233744)
```

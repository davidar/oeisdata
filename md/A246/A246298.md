## Python
```Python
from sympy import sin
def ok(n):
    s0, s1, s2, s3 = sin(n), sin(n + 1), sin(n + 2), sin(n + 3)
    return s0>s1 and s1>s2 and s2<s3
print([n for n in range(1, 501) if ok(n)]) # _Indranil Ghosh_, Aug 03 2017
```

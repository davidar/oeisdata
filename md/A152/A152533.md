## Python
```Python
from sympy import isprime, prime
def afind(limit, startk=5):
    k, pow10 = startk, 10**startk
    for k in range(startk, limit+1):
        if isprime(pow10 + 1111): print(k, end=", ")
        pow10 *= 10
afind(800) # _Michael S. Branicky_, Jan 22 2022
```

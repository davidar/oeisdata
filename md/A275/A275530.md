## Python
```Python
from sympy import isprime
def a(n):
  m, pow2 = 1, 2**n
  while True:
    if isprime((m**pow2 + 1)//2): return m
    m += 2
print([a(n) for n in range(9)]) # _Michael S. Branicky_, Mar 03 2021
```

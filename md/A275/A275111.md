## Python
```Python
from sympy import prime
from sympy.core.numbers import igcdex
def A275111(n):
    p, q = prime(n), prime(n+1)
    a = q-1
    for i in range(p+1,q):
        a = (a*igcdex(i,q)[0]) % q
    return a # _Chai Wah Wu_, Jul 18 2016
```

## Python
```Python
from sympy import fibonacci, npartitions
def a(n): return sum([fibonacci(k)*npartitions(n - k) for k in range(1, n + 1)])
print([a(n) for n in range(101)]) # _Indranil Ghosh_, Jun 29 2017
```

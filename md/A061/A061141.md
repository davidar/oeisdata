## Python
```Python
from sympy import prime, divisor_sigma as sigma
def a(n): return max(sigma(c) for c in range(prime(n)+1, prime(n+1)))
print([a(n) for n in range(2, 57)]) # _Michael S. Branicky_, Jul 12 2021
```

## Python
```Python
from math import isqrt
from sympy import mobius, prime, nextprime
def A061398(n):
    p = prime(n)
    q = nextprime(p)
    r = isqrt(p-1)+1
    return sum(mobius(k)*((q-1)//k**2) for k in range(r,isqrt(q-1)+1))+sum(mobius(k)*((q-1)//k**2-(p-1)//k**2) for k in range(1,r))-1 # _Chai Wah Wu_, Jun 01 2024
```

## Python
```Python
from sympy import primerange, isprime, floor
def a(n):
    s=0
    for q in primerange(2, n//2 + 1): s+=isprime(n - q)
    return s
print([a(n) for n in range(101)]) # _Indranil Ghosh_, Jun 30 2017
```

## Python
```Python
from math import isqrt, factorial
from sympy import divisors
def A061057(n):
    k = factorial(n)
    m = max(d for d in divisors(k,generator=True) if d <= isqrt(k))
    return k//m-m # _Chai Wah Wu_, Apr 06 2022
```

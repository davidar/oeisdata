## Python
```Python
from sympy import integer_nthroot
def a(n):
  r, exact = integer_nthroot(10**(n-1), 3)
  return 10**(n-1) if exact else (r+1)**3
print([a(n) for n in range(1, 21)]) # _Michael S. Branicky_, Jan 27 2021
```

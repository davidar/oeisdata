## Python
```Python
from math import prod
def ok(n):
    d = list(map(int, str(n)))
    pod, sod = prod(d), sum(d)
    return pod and pod%sod == 0
print([k for k in range(416) if ok(k)]) # _Michael S. Branicky_, Mar 28 2022
```

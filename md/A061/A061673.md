## Python
```Python
from sympy import isprime
def abelow(limit):
  for k in range(2, limit, 2):
    if not isprime(k-1) and not isprime(k+1): yield k
print([an for an in abelow(327)]) # _Michael S. Branicky_, Jan 02 2021
```

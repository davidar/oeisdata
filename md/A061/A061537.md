## Python
```Python
from sympy import divisors, gcd, prod
def a(n): return prod(d for d in divisors(n) if gcd(d, n//d)==1)
print([a(n) for n in range(1, 51)]) # _Indranil Ghosh_, Aug 04 2017
```

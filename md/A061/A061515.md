## Python
```Python
from itertools import accumulate, repeat
def f(n, _): return int("".join(str(int(d)+3) for d in str(n)))
def aupton(nn): return list(accumulate(repeat(1, nn+1), f))
print(aupton(19)) # _Michael S. Branicky_, Mar 19 2022
```

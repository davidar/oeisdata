## Python
```Python
from math import factorial
from sympy import isprime
def ok(n):
    return isprime(int((s:=str(n))+str(sum(factorial(int(d)) for d in s))))
print([k for k in range(999) if ok(k)]) # _Michael S. Branicky_, Feb 22 2024
```

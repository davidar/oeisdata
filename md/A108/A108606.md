## Python
```Python
from sympy import isprime, factorint
def ok(n): return isprime(sum(map(int, str(n)))) and sum(factorint(n).values()) == 2
print([k for k in range(455) if ok(k)]) # _Michael S. Branicky_, Aug 22 2022
```

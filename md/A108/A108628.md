## Python
```Python
def A108628(n):
    m, g = 1, 0
    for k in range(n+1):
        g += m
        m *= (n+k+2)*(n-k)*(n-k+1)
        m //= (k+1)**3
    return g # _Chai Wah Wu_, Oct 03 2022
```

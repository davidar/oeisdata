## Python
```Python
from gmpy2 import isqrt, is_square
A260375_list, g = [0], 1
for i in range(1, 1001):
    g *= i
    s = isqrt(g)
    t = g-s**2
    if is_square(t if t-s <= 0 else 2*s+1-t):
        A260375_list.append(i) # _Chai Wah Wu_, Jul 23 2015
```

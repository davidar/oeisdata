## Python
```Python
from gmpy2 import mpz, digits
def A260031(n):
    s = digits(mpz(n)**mpz(n),12)
    t = s[-1]
    while t == '0':
        s = s[:-1]
        t = s[-1]
    return int(t,12) # _Chai Wah Wu_, Jul 19 2015
```

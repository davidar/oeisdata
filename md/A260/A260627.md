## Python
```Python
with open("pi-billion.txt", "r") as f:
    for a in range(1,10):
        g = f.read(10**a)
        for x in reversed(range((10**a)-50,(10**a)-1)):
            if g.find(str(x)) == -1:
                print(a,x)
                break
```

## Python
```Python
from sympy.ntheory.factor_ import core
def a(n):
  b, pow2, t = 1, 2**n, 2
  while core(t, 2) == t:
    b += 1
    t = b**(pow2) + 1
  return b
print([a(n) for n in range(4)]) # _Michael S. Branicky_, Mar 07 2021
```

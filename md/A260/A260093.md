## Python
```Python
TOP = 5000*5000
a = [0]*TOP
for y in range(2, 5000):
  z = TOP//y + 1
for x in range(y, z):
    n = x*y + x + y
    if n>=TOP: break
    a[n] += x+y
for n in range(1,TOP):
    if a[n]==a[n-1]+1 and a[n]>0: print str(n)+',',
```

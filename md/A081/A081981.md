## Python
```Python
from sympy import isprime
def ok(n): return all((n+1)%int(d) == 0 for d in set(str(n)) - {"0"}) and isprime(n)
print([k for k in range(2222) if ok(k)]) # _Michael S. Branicky_, Sep 06 2022
```

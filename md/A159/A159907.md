## Python
```Python
from fractions import Fraction
from sympy import divisor_sigma as sigma
def aupto(limit):
  for k in range(1, limit):
    if Fraction(int(sigma(k, 1)), k).denominator == 2:
      print(k, end=", ")
aupto(3*10**4) # _Michael S. Branicky_, Feb 24 2021
```

## Python
```Python
from sympy import isprime
from itertools import count
def a(n): return next(k for k in count(1) if not isprime(2*n*k-1) and not isprime(2*n*k+1))
print([a(n) for n in range(1, 86)]) # _Michael S. Branicky_, Jan 25 2023
```

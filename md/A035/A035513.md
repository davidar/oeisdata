## Python
```Python
from sympy import fibonacci as F, sqrt
import math
tau = (sqrt(5) + 1)/2
def T(n, k): return F(k + 1)*int(math.floor(n*tau)) + F(k)*(n - 1)
for n in range(1, 11): print([T(k, n - k + 1) for k in range(1, n + 1)]) # _Indranil Ghosh_, Apr 23 2017
```

## Python
```Python
from math import comb
def A263017(n):
    c, k = 1, 0
    for i,j in enumerate(bin(n)[-1:1:-1]):
        if j == '1':
            k += 1
            c += comb(i,k)
    return c # _Chai Wah Wu_, Mar 01 2023
```

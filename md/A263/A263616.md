## Python
```Python
from itertools import product
def pal(n): s = str(n); return s == s[::-1]
def a(n): return int(n==1) + sum(pal(i**2) for i in range(10**(n-1), 10**n))
print([a(n) for n in range(1, 8)]) # _Michael S. Branicky_, Apr 03 2021
```

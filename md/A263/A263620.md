## Python
```Python
def ispal(n): s = str(n); return s == s[::-1]
def a(n):
  c, k, kk = 0, 1, 1
  while kk < 10**(2*n): c, k, kk = c + (ispal(kk)), k+1, kk + 2*k + 1
  return c
print([a(n) for n in range(1, 8)]) # _Michael S. Branicky_, Mar 06 2021
```

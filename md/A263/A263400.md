## Python
```Python
from gmpy2 import fib2, digits
def A263400(n):
    b, a = fib2(n)
    s, m = digits(b), n
    while True:
        a, b, m = b, a+b, m+1
        t = digits(b)
        if b > a and s in t:
            return m # _Chai Wah Wu_, Oct 27 2015
```

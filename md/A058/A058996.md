## Python
```Python
from sympy import isprime
A058996_list = []
for i in range(10**6):
    if i % 10:
        p = int(str(i**4)[::-1])
        if isprime(p):
            A058996_list.append(p)
A058996_list = sorted(A058996_list) # _Chai Wah Wu_, Jun 02 2016
```

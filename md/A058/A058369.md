## Python
```Python
def ds(n): return sum(map(int, str(n)))
def ok(n): return ds(n) == ds(n**2)
def aupto(nn): return [m for m in range(nn+1) if ok(m)]
print(aupto(1189)) # _Michael S. Branicky_, Jan 09 2021
```

## Python
```Python
from sympy import divisor_sigma as sigma, totient as phi
def aupto(limit):
  for k in range(2, limit):
    if sigma(phi(k)) == phi(sigma(k) - k): print(k, end=", ")
aupto(10**4) # _Michael S. Branicky_, Feb 18 2021
```

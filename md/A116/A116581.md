## Python
```Python
from sympy import isprime
def aupton(terms):
  k, alst = 2, []
  while len(alst) < terms:
    if isprime(k**3-k-1): alst.append(k**3-k-1)
    k += 1
  return alst
print(aupton(37)) # _Michael S. Branicky_, May 23 2021
```

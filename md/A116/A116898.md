## Python
```Python
from sympy import isprime
from math import factorial
def ok(n):
  s, zeros = str(factorial(n)), 0
  while s[-1] == '0': s = s[:-1]; zeros += 1
  return isprime(int(s + '1'*zeros))
print([m for m in range(500) if ok(m)]) # _Michael S. Branicky_, Feb 10 2021
```

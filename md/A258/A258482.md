## Python
```Python
for p in range(1, 7):
    for i in range(10**p, 10**(p + 1)):
        c = 10**(int((p - 1) / 2) + 1)
        a, b = i // c, i % c
        if i == a**2 - b**2:
            print(i, end=",")
```

## Maple
```Maple
p:= proc(k,n) local t; t:= n mod 10^k; t >= 10^(k-1) and issqr(t*(n-t)/10^k) end proc:
filter:= n -> ormap(p, [$1..ilog10(n)], n):
select(filter, [seq(i^2, i=1..10^4)]); # _Robert Israel_, Sep 22 2015
```

## Python
```Python
from sympy.ntheory.factor_ import digits
l=[]
for n in range(3, 301, 2):
    c=1
    for b in range(2, n - 1):
        d=digits(n, b)[1:]
        if max(d)==min(d):
            c=0
            break
    if c: l.append(n)
print(l) # _Indranil Ghosh_, Jun 22 2017, after PARI program
```

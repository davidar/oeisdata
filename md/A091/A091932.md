## Python
```Python
from sympy import isprime, primerange
def ok(p): return isprime((1 << (p.bit_length()-1)) ^ p)
def aupto(lim): return [p for p in primerange(1, lim+1) if ok(p)]
print(aupto(1061)) # _Michael S. Branicky_, Jul 11 2021
```

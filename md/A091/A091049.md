## Python
```Python
def A091049(n):
    k = 1
    while True:
        m1 = k
        for i in range(n+1):
            m2 = int(str(m1),1+max(int(d) for d in str(m1)))
            if m1 == m2:
                if i == n:
                    return k
                else:
                    break
            m1 = m2
        k += 1 # _Chai Wah Wu_, Jan 07 2015
```

## Python
```Python
def a(n):
    SUM, SQR = set(), set(x**2 for x in range(1, n + 1))
    for i in SQR:
        SUM.add(i)
        for j in SQR: SUM.add(i + j)
    return len(SUM)
# Romain CARRE (romain.carre.2008(AT)enseirb.fr), Apr 16 2010
```

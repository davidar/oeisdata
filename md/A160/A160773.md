## Python
```Python
def A160773_gen(): # generator of terms
    p3, p5, p7 = [1]*3
    for k in count(0):
        if isprime(p3+p5+p7): yield k
        p3 *= 3
        p5 *= 5
        p7 *= 7
A160773_list = list(islice(A160773_gen(),6)) # _Chai Wah Wu_, Dec 27 2021
```

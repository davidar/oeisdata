## Python
```Python
def a(n):
    if n==0: return 0
    r=0
    dr=1
    while n%3==0:
        dr*=3
        n/=3
    while n!=0:
        q=(n + 1)/3
        r=3*r + dr*(n - 3*q)
        n=q
    return r
##print [a(n) for n in range(101)] # _Indranil Ghosh_, Jun 10 2017, after _Franklin T. Adams-Watters_
```

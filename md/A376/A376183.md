## Python
```Python
def A376183(n):
    c = 0
    for x in range(n):
        for y in range(x,n):
            xy,xyp = x*y%n-1,(x+y)%n
            c += sum(not (xy*z-xyp)%n for z in range(y,n))
    return c # _Chai Wah Wu_, Sep 19 2024
```

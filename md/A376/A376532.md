## Python
```Python
def A376532(n):
    m, r, c, a, b, d = 3**n, n**2, 1, 2, n**2, 1
    for k in range(1,r+1):
        c += b//d*a%m
        a = a*2%m
        b *= r-k
        d *= k+1
    return c//m # _Chai Wah Wu_, Oct 07 2024
```

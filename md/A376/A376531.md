## Python
```Python
def A376531(n):
    m, r, c, b, d = (1<<n)-1, n**2, 1, n**2, 1
    for k in range(1,r+1):
        c += b//d&m
        b *= r-k
        d *= k+1
    return c>>n # _Chai Wah Wu_, Oct 09 2024
```

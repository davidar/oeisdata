## Python
```Python
def A376240(n):
    c = 0
    for x in range(1,n):
        for y in range(x,n):
            xy,xyp = x*y%n-1,n-(x+y)%n
            c += sum(not (z==xyp or (xy*z+xyp)%n) for z in range(y,n))
    return c # _Chai Wah Wu_, Sep 30 2024
```

## Python
```Python
from sympy import mobius, totient, divisors
def a(n):
    tn = totient(n)
    return sum(mobius(totient(d))*tn//d for d in divisors(tn))
print([a(n) for n in range(1, 51)]) # _Indranil Ghosh_, Jul 17 2017
```

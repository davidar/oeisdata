## Python
```Python
from sympy.ntheory import digits
def a(n):
    d = digits(n, 3)[1:]
    return sum(3**(i+(1-2*(i&1)))*di for i, di in enumerate(d[::-1]))
print([a(n) for n in range(72)]) # _Michael S. Branicky_, Aug 05 2022
```

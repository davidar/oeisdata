## Python
```Python
from sympy import divisors
[n for n in range(1,10**5) if sum([d for d in range(2,n) if (n % d) and (2*n) % d in [d-1,0,1]]) == sum(divisors(n))] # _Chai Wah Wu_, Aug 07 2014
```

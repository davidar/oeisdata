## Python
```Python
from fractions import Fraction
def A087708(n):
    x = Fraction(n)
    while x.denominator > 1 or x<=n:
        x = Fraction(5*x.__ceil__(),3)
    return int(x) # _Chai Wah Wu_, Sep 02 2023
```

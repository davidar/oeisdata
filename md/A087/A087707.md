## Python
```Python
from fractions import Fraction
def A087707(n):
    x, c = Fraction(n), 0
    while x.denominator > 1 or x<=n:
        x = Fraction(5*x.__ceil__(),3)
        c += 1
    return c # _Chai Wah Wu_, Sep 02 2023
```

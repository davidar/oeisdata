## Python
```Python
def ispal(n):
    t = str(n)
    return t == t[::-1]
def A087990(n):
    s=0
    for i in range(1,n+1):
        if n%i==0 and ispal(i):
            s+=1
    return s # _Indranil Ghosh_, Feb 10 2017
```

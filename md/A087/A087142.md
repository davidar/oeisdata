## Python
```Python
def ok(n):
    d = list(map(int, str(n)))
    return 0 not in d and n%sum(d) and all(n%di == 0 for di in set(d))
print([k for k in range(1312) if ok(k)]) # _Michael S. Branicky_, Nov 15 2021
```

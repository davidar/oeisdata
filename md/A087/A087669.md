## Python
```Python
def A087669(n):
    c, x = 0, 2*n+1
    a, b = divmod(x,n)
    while b != 0:
        x *= a
        c += 1
        a, b = divmod(x,n)
    return c # _Chai Wah Wu_, Mar 01 2021
```

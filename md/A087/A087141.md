## Python
```Python
def ok(n):
    d = list(map(int, str(n)))
    if n == 0 or n%sum(d): return False
    return 0 in d or any(n%di for di in set(d))
print([k for k in range(271) if ok(k)]) # _Michael S. Branicky_, Oct 18 2021
```

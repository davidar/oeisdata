## Python
```Python
from sympy import nextprime
def a(n):
    if n == 1: return 0
    p, q, twon = 2, 3, 2**n
    while (1 + twon*p)%q: p, q = q, nextprime(q)
    return p
print([a(n) for n in range(1, 94)]) # _Michael S. Branicky_, Dec 11 2021
```

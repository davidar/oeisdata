## Python
```Python
from itertools import count, islice, takewhile
def agen():
    for d in count(2, 2):
        t = (int(str(k) + str(2*k)) for k in count(10**(d//2-1)))
        yield from takewhile(lambda x: x < 10**d, t)
print(list(islice(agen(), 43))) # _Michael S. Branicky_, Dec 24 2021
```

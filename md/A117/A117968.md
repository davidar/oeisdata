## Python
```Python
def a(n):
    if n==1: return 2
    if n%3==0: return 3*a(n//3)
    elif n%3==1: return 3*a((n - 1)//3) + 2
    else: return 3*a((n + 1)//3) + 1
print([a(n) for n in range(1, 101)]) # _Indranil Ghosh_, Jun 06 2017
```

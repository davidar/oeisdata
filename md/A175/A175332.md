## PARI
```PARI
is_11p0s(n)=
{ /* Return whether binary expansion has form 11+0* */
  local(b);
  if ( n<3, return(0) );
  b = binary( n/(2^valuation(n,2) ) );
  if ( #b<2, return(0) );
  for (j=1,#b, if(b[j]==0, return(0) ) );
  return(1);
}
for (n=1,2100, if (is_11p0s(n), print1(n,", ") ) ); /* show terms */
```

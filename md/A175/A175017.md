## Python
```Python
from sympy import nextprime
A175017_list, p = [],2
while len(A175017_list) <= 100:
    s = str(p)
    if '13' in s and sum(int(d) for d in s) == 13:
        A175017_list.append(p)
    p = nextprime(p) # _Chai Wah Wu_, Mar 05 2020
```

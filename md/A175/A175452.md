## Python
```Python
from sympy import nextprime
def a(n):
    p, m = 2, 2*n+1
    while (p+2)%m: p = nextprime(p)
    return p
print([a(n) for n in range(1, 61)]) # _Michael S. Branicky_, Jul 03 2021
```

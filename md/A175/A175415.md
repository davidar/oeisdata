## Python
```Python
from itertools import groupby
def ok(n):
    b = bin(n)[2:]
    return all(len(b)%len(list(g)) == 0 for k, g in groupby(b))
print([k for k in range(1, 173) if ok(k)]) # _Michael S. Branicky_, Nov 07 2022
```

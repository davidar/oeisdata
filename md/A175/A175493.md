## Python
```Python
from sympy import divisor_count
from itertools import count, islice
def agen():
    an = 1
    for k in count(2):
        yield an
        an *= k**divisor_count(k)
print(list(islice(agen(), 14))) # _Michael S. Branicky_, May 03 2022
```

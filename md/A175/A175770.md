## Python
```Python
from sympy import prime
def a(n):
  return int(str(prime(n)).translate({ord('1'):ord('3'), ord('3'):ord('1')}))
print([a(n) for n in range(1, 60)]) # _Michael S. Branicky_, Feb 01 2021
```

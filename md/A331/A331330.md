## Maple
```Maple
b:= proc(n, i) option remember; `if`(n=0, 1, add(
     `if`(i=j, 0, b(n-j, `if`(n<i+j, 0, i))), j=1..n))
    end:
a:= proc(n) option remember; add(b(n-j, j), j=1..n) end:
seq(a(n), n=0..50);  # _Alois P. Heinz_, Feb 06 2020
```

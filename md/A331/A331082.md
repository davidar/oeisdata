## Python
```Python
from sympy import divisors
def count37(iterable): return sum(i%10 in [3, 7] for i in iterable)
def a(n):
  m = 2
  while count37(divisors(m)) != n: m += 1
  return m
print([a(n) for n in range(1, 17)]) # _Michael S. Branicky_, Mar 21 2021
```

## Python
```Python
def Lah(n, k):
    if n == k: return 1
    if k<0 or  k>n: return 0
    return (k*n*gamma(n)^2)/(gamma(k+1)^2*gamma(n-k+1))
matrix(ZZ, 8, Lah) * matrix(ZZ, 8, stirling_number2)  # as a square matrix
```

## Python
```Python
from sympy.ntheory.primetest import is_square
def R(n): return int(str(n)[::-1])
def ok(n): Rn = R(n); return n != Rn and is_square(abs(n**2 - Rn**2))
print([k for k in range(10**6) if ok(k)]) # _Michael S. Branicky_, Mar 02 2022
```

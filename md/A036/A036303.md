## Python
```Python
from sympy import factorint
def ok(n):
    f = factorint(n)
    return sum(f.values()) > 1 and all(set(str(p)) <= set("13") for p in f)
print(list(filter(ok, range(2800)))) # _Michael S. Branicky_, Sep 27 2021
```

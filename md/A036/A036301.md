## Python
```Python
def eodiff(n):
  digs = list(map(int, str(n)))
  return abs(sum(d for d in digs if d%2==0)-sum(d for d in digs if d%2==1))
def aupto(lim): return [m for m in range(lim+1) if eodiff(m) == 0]
print(aupto(1298)) # _Michael S. Branicky_, Feb 21 2021
```

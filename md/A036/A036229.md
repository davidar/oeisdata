## Python
```Python
from sympy import isprime
def A036229(n):
    k, r, m = (10**n-1)//9, 2**n-1, 0
    while m <= r:
        t = k+int(bin(m)[2:])
        if isprime(t):
            return t
        m += 1
    return -1 # _Chai Wah Wu_, Aug 18 2021
```

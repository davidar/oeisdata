## Python
```Python
def aupton(terms):
  alst = [1, 10]
  while len(alst) < terms: alst.append(int(str(alst[-1]) + str(alst[-2])))
  return alst[:terms]
print(aupton(9)) # _Michael S. Branicky_, Jan 10 2021
```

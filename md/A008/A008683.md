## Sage
```Sage
@cached_function
def mu(n):
    if n < 2: return n
    return -sum(mu(d) for d in divisors(n)[:-1])
# Changing the sign of the sum gives the number of ordered factorizations of n A074206.
print([mu(n) for n in (1..96)])  # _Peter Luschny_, Dec 26 2016
```

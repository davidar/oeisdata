## Sage
```Sage
def A008904(n):
    # algorithm from David Wilson, http://oeis.org/A008904/a008904b.txt
    if n == 0 or n == 1: return 1
    dd = n.digits(base=5)
    x = sum(i*d for i,d in enumerate(dd))
    y = sum(d for d in dd if d % 2 == 0)/2
    z = 2**((x+y) % 4)
    if z == 1: z = 6
    return z # _D. S. McNeil_, Dec 09 2010
```

## Python
```Python
from sympy.ntheory.factor_ import primeomega
def nextsemiprime(n):
  while primeomega(n + 1) != 2: n += 1
  return n + 1
def a(n): return nextsemiprime(10*n)
print([a(n) for n in range(54)]) # _Michael S. Branicky_, Apr 14 2021
```

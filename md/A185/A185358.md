## Python
```Python
from sympy import factorint, floor
def a(n):
    f=factorint(n)
    return 1 if n==1 else max(1 - i*(1 + (-f[i])//i) for i in f)
print([a(n) for n in range(1, 201)]) # _Indranil Ghosh_, Jun 29 2017
```

## Python
```Python
from itertools import count
def A185076(n):
    for k in count(1):
        if n == (t:=len(s:=str(k**2)))+sum(map(int,s)):
            return k
        if t >= n:
            return 0 # _Chai Wah Wu_, Mar 15 2023
```

## Python
```Python
def a(n):
    if n%2: return 2**((n + 1)//2)
    k=bin(n - 1)[2:].count('1') - bin(n)[2:].count('1')
    return 2**(n//2 - 1) + 2**((n//2 - 2**k)//(2**(k + 1)))
print([a(n) for n in range(1, 101)]) # _Indranil Ghosh_, Jun 29 2017
```

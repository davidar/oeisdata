## Maple
```Maple
f:= proc(n) local L,j;
  L:= convert(n!,base,10);
  for j from 1 while L[j] = 0 do od:
  L:= Statistics:-Tally(L[j...-1]);
  L:= sort(L,(a,b) -> rhs(a) < rhs(b));
  if nops(L) >= 2 and rhs(L[2]) = rhs(L[1]) then -1 else lhs(L[1]) fi
end proc:
map(f, [$0..100]); # _Robert Israel_, Sep 02 2024
```

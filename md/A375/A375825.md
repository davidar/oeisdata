## Python
```Python
def A375825row(n):
    row = [0] * (n + 1)
    def e_rec(j, i):
        if j <= n:
            i = e_rec(2 * j, i)
            row[j] = i
            i = e_rec(2 * j + 1, i + 1)
        return i
    e_rec(1, 1)
    return row
```

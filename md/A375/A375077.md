## Python
```Python
from math import prod, comb
def A375077(n):
    a, c, k = prod(n+1-i for i in range(n+1)), comb(n+1<<1,n+1), n+1
    while c%a:
        k += 1
        a = a*k//(k-n-1)
        c = c*((k<<1)-1<<1)//k
    return k # _Chai Wah Wu_, Jul 30 2024
```

## Python
```Python
from sympy.ntheory import digits
def ok(n): return sum(digits(n, 2)[1:]) == sum(digits(n, 3)[1:]) + 1
print([k for k in range(604) if ok(k)]) # _Michael S. Branicky_, Aug 08 2024
```

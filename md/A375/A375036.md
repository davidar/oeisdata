## Python
```Python
def a(n):
    d = [0] + list(map(int, str(n))) + [0]
    return 2*(len(d)-2) + sum(abs(d[i+1]-d[i]) for i in range(len(d)-1))
print([a(n) for n in range(68)]) # _Michael S. Branicky_, Jul 28 2024
```

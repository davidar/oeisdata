## Python
```Python
from gmpy2 import digits, is_prime, mpz
def ok(n): return is_prime(n) and any(mpz(digits(n, b))%n == 0 for b in range(2, 10))
print([k for k in range(10**7) if ok(k)]) # _Michael S. Branicky_, Sep 03 2024
```

## Python
```Python
from fractions import Fraction
from itertools import count
def A375081(n):
    a = Fraction((n<<1)+1,n*(n+1))
    for k in count(n+1):
        if a.denominator > (a:=a+Fraction(1,k+1)).denominator:
            return k # _Chai Wah Wu_, Jul 30 2024
```

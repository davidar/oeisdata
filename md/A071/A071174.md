## Python
```Python
from math import prod
from sympy import factorint
def ok(n): f = factorint(n); return sum(f[p] for p in f)==prod(p for p in f)
print(list(filter(ok, range(10**6)))) # _Michael S. Branicky_, Apr 27 2021
```

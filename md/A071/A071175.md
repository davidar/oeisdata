## Python
```Python
from math import prod
from sympy import factorint
def ok(n): f = factorint(n); return prod(f[p] for p in f)==sum(p for p in f)
print(list(filter(ok, range(10**5)))) # _Michael S. Branicky_, Apr 27 2021
```

## Maple
```Maple
A071649 := proc(n)
      local a,d;
      a := 0 ;
      for d in convert(n,base,10) do
        if type(d,'odd') then
            a := a+d ;
        end if;
    end do:
    a ;
end proc: # _R. J. Mathar_, Feb 02 2015
```

## Python
```Python
from sympy import factorint
def stormer(n): return max(factorint(n*n + 1)) >= 2*n
def ok(n): return not stormer(n) and stormer(max(factorint(n)))
print(list(filter(ok, range(1, 438)))) # _Michael S. Branicky_, Aug 30 2021
```

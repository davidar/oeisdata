## Python
```Python
from gmpy2 import is_square, popcount
for n in range(1, 33):
    m=n+1
    for k in range(2**(n-1), 2**n):
        if is_square(k):
            m=min(m, n-popcount(k))
    print(n, -1 if m>n else m)
```

## Python
```Python
def A352654(n):
    if n == 0: return 0
    m, g = 1, 0
    for k in range(n+1):
        g += -m*n//(n+k) if (n+k)&1 else m*n//(n+k)
        m *= (n+k+1)*(n-k)*(n+k)
        m //= (k+1)**3
    return g # _Chai Wah Wu_, Oct 03 2022
```

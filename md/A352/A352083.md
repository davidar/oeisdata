## Python
```Python
from sympy import isprime
def afind(limit):
    for k in range(1, limit+1, 2):
        if isprime((3**k - k**3)//2):
            print(k, end=", ")
afind(1000) # _Michael S. Branicky_, Mar 03 2022
```

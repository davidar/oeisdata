## Python
```Python
from math import comb,isqrt
def A352666(n):
    if n <= 1: return 0
    r = isqrt(3*n-4)
    k0 = (n-r-1)//2
    return max(comb(k,3)*(n-k)+comb(n-k,3)*k for k in (k0,k0+1))
```

## Python
```Python
from sympy import isprime; L = [3, 5]
while 1:
    for j in range(2, len(L)+1):
        s = L[-1] + L[-j]
        if isprime(s+1): L.append(s+1); break
        elif isprime(s-1): L.append(s-1); break
    else: break
print(*L, sep = ', ')
```

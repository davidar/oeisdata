## Python
```Python
from sympy import divisors
def ok(n):
    s = sorted(set(d + n//d for d in divisors(n)))
    return 1 not in set(s[i+1]-s[i] for i in range(len(s)-1))
print([k for k in range(1, 75) if ok(k)]) # _Michael S. Branicky_, Jul 10 2022
```

## Python
```Python
def a(n):
    if n == 1: return 2
    p, m = 2, 2*n-1
    while p <= m or p % m != 1: p *= 2
    return p
print([a(n) for n in range(1, 34)]) # _Michael S. Branicky_, Mar 07 2022
```
```Python
from sympy import n_order
def a(n): return 2**n_order(2, 2*n-1)
```

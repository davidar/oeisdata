## Python
```Python
from sympy import factorint, primepi
def ok(n):
    if n%2 == 0: return False
    return all(primepi(p)%2==0 and e%2==1 for p, e in factorint(n).items())
print([k for k in range(318) if ok(k)]) # _Michael S. Branicky_, Mar 12 2022
```

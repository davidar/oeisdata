## Python
```Python
from sympy import isprime
def ok(n): return n%10 == 1 and isprime(n) and isprime((n**2+3*n+1)//5)
print([k for k in range(1, 5000, 10) if ok(k)]) # _Michael S. Branicky_, Apr 11 2022
```

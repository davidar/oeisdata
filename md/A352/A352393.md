## Python
```Python
from sympy import isprime
from itertools import count, islice
def agen(): yield from (k for k in count(0) if isprime(3**k + 5**k + 7**k + 11**k + 13**k))
print(list(islice(agen(), 9))) # _Michael S. Branicky_, Jun 07 2022
```

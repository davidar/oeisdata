## Python
```Python
from math import isqrt
from sympy.ntheory.factor_ import core, isprime
def ok(n): return isqrt(n)**2 == n or isprime(core(n))
print([k for k in range(1, 145) if ok(k)]) # _Michael S. Branicky_, Oct 03 2024
```

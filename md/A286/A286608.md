## Python
```Python
from sympy import factorint, primepi
def a(n):
    f=factorint(n)
    return sum([2**primepi(i - 1) for i in f])
print([n for n in range(1, 201) if a(n)<n]) # _Indranil Ghosh_, Jun 20 2017
```

## Python
```Python
from sympy import integer_nthroot, prime
def A047972(n):
    p = prime(n)
    a = integer_nthroot(p,2)[0]
    return min(p-a**2,(a+1)**2-p) # _Chai Wah Wu_, Apr 03 2021
```

## Python
```Python
def agen():
  m = 4
  while True:
    tstr = str(m*m-4)
    k = int(tstr[:len(tstr)//2])
    if tstr == str(k) + str(k+1):
      yield(int(tstr))
    m += 1
for an in agen(): print(an, end=", ") # _Michael S. Branicky_, Jan 02 2021
```

## Python
```Python
from math import prod
def pd(n): return prod(map(int, str(n)))
def a(n):
  pdn, f = pd(n), 2
  while pd(f*n) != pdn: f += 1
  return f*n
print([a(n) for n in range(1, 27)]) # _Michael S. Branicky_, Jan 16 2021
```

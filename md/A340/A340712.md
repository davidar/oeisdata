## Python
```Python
from sympy import nextprime
def aupto(limit):
  p, psharp = 3, 2
  while p <= limit:
    if (psharp+2)%p == 0: print(p, end=", ")
    psharp, p = psharp*p, nextprime(p)
aupto(500000) # _Michael S. Branicky_, Mar 24 2021
```

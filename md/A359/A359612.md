## Python
```Python
from sympy import factorint
def a(n):
    max_factor = 0
    min_exponent = float("inf")
    for p, exponent in factorint(n).items():
        if exponent < min_exponent:
            max_factor = p
            min_exponent = exponent
        elif exponent == min_exponent:
            max_factor = max(max_factor, p)
    return max_factor
```

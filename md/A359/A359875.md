## Python
```Python
from sympy import divisors, mobius, reduced_totient
def a002322(n): return reduced_totient(n)
def a023900(n): return sum([d*mobius(d) for d in divisors(n)])
print([k for k in range(1, 254) if a002322(k) == a023900(k)])
```

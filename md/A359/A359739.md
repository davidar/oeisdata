## Python
```Python
from math import factorial, comb
def oddfactorial(n: int) -> int: return factorial(2 * n) // (2**n * factorial(n))
def a(n: int) -> int:
    return sum(comb(n, j) * oddfactorial(j//2) * n**j for j in range(0, n+1, 2))
print([a(n) for n in range(19)])
```

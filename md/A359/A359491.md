## Python
```Python
from sympy import factorint
def ok(n): return sorted(str(n)) == sorted(s[0] for s in map(str, factorint(n, multiple=True)))
print([k for k in range(1, 10**5) if ok(k)]) # _Michael S. Branicky_, Jan 08 2023
```

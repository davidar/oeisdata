# Numbers k that can be written as the sum of a perfect square and a factorial in exactly 3 distinct ways\.

## Python
```Python
import math
for x in range(1, 120000000):
    total = 0
    prod = 1
    factInc = 2
    while prod <= x:
        sq = math.sqrt(x - prod)
        if sq % 1 == 0:
            total = total + 1
        prod = prod * factInc
        factInc = factInc + 1
    if total == 3:
        print(x)
```

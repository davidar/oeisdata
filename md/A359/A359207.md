## Python
```Python
def f(n): return 1 if n == 0 else (m:=3*n)^((1 << m.bit_length())-1)
def a(n):
    i, fi = 0, n
    while fi != 0: i, fi = i+1, f(fi)
    return i
print([a(n) for n in range(68)]) # _Michael S. Branicky_, Dec 20 2022
```

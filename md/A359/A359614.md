## Python
```Python
from itertools import permutations
from sympy import Matrix, I
def A359614(n): return min(Matrix(n,n,[(d[i-j] if i>j else -d[j-i]) if i!=j else d[0]*I for i in range(n) for j in range(n)]).det()*(1,-I,-1,I)[n&3] for d in permutations(range(1,n+1))) # _Chai Wah Wu_, Jan 25 2023
```

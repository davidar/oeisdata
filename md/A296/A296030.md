## Python
```Python
def get_coordinate(n):
....k=ceil((sqrt(n)-1)/2)
....t=2*k+1
....m=t**2
....t=t-1
....if n >= m - t:
........return k - (m-n), -k
....else:
........m -= t
....if n >= m - t:
........return -k, -k+(m-n)
....else:
........m -= t
....if n >= m-t:
........return -k+(m-n), k
....else:
........return k, k-(m-n-t)
```

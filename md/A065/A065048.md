## Python
```Python
from collections import Counter
def A065048(n):
    c = {1:1}
    for k in range(1,n+1):
        d = Counter()
        for j in c:
            d[j] += k*c[j]
            d[j+1] += c[j]
        c = d
    return max(c.values()) # _Chai Wah Wu_, Jan 31 2024
```

## Python
```Python
from sympy.ntheory.digits import digits
def a(n):
    return sum(bi*(-1)**k for k, bi in enumerate(digits(n, 3)[1:][::-1]))
print([a(n) for n in range(104)]) # _Michael S. Branicky_, Jul 28 2021
```

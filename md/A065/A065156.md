## Python
```Python
a, n = 0, 0
while n < 1000:
    a, f0, f1, i = a+1, 1, 2, 1
    if f1%a == 0:
        n = n+1
        print(n,a)
    else:
        while f0%a != 0 and i <= a:
            f0, f1, i = f0+f1, f0, i+1
        if i <= a:
            n = n+1
            print(n,a) # _A.H.M. Smeets_, Sep 20 2020
```

## Python
```Python
from math import prod
from sympy import factorint
def A065608(n):
    f = factorint(n).items()
    return prod((p**(e+1)-1)//(p-1) for p, e in f)-prod(e+1 for p,e in f) # _Chai Wah Wu_, Jul 16 2022
```

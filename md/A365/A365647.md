## Python
```Python
from sympy import divisors, primefactors, prod, reduced_totient
def psi(n):
    return n*prod(p+1 for p in primefactors(n))//prod(primefactors(n))
def a(n): return sum(psi(d) * reduced_totient(n//d) for d in divisors(n))
```

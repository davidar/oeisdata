def A365439(n): return (-(s:=isqrt(n))**2*comb(s+4,4)+sum((q:=n//k)*(5*comb(k+3,4)+comb(q+4,4)) for k in range(1,s+1)))//5 # _Chai Wah Wu_, Oct 26 2023

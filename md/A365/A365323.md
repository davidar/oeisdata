## Python
```Python
from sympy.utilities.iterables import partitions
def A365323(n):
    a = {tuple(sorted(set(p))) for p in partitions(n)}
    return sum(1 for k in range(1,n) for d in partitions(k) if tuple(sorted(set(d))) not in a) # _Chai Wah Wu_, Sep 12 2023
```

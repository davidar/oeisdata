## Python
```Python
from math import isqrt
def ok(n): s = str(n); return len(s)>=3 and all((x:=int(s[i:i+3]))>0 and isqrt(x)**2==x for i in range(len(s)-2))
print([k for k in range(10**6) if ok(k)]) # _Michael S. Branicky_, Dec 14 2023
```

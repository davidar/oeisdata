## Maple
```Maple
filter:= proc(n) local t, b;
  t:= 4^n;
b:= ilog10(t) - ilog10(n);
floor(t/10^b) = n
end proc:
select(filter, [$1..10^5]);
```
## Python
```Python
def afind(limit, startk=1):
    k, pow4 = startk, 4**startk
    for k in range(startk, limit+1):
        if str(pow4).startswith(str(k)):
            print(k, end=", ")
        pow4 *= 4
afind(10**4) # _Michael S. Branicky_, Oct 17 2021
```

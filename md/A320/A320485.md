## Maple
```Maple
f:= proc(n) local F,S;
  F:= convert(n,base,10);
  S:= select(t -> numboccur(t,F)>1, [$0..9]);
  if S = {} then return n fi;
  F:= subs(seq(s=NULL,s=S),F);
  if F = [] then -1
  else add(F[i]*10^(i-1),i=1..nops(F))
  fi
end proc:
map(f, [$0..200]); # _Robert Israel_, Oct 24 2018
```

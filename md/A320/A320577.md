## Python
```Python
for n in range(3, 101):
    m = n % 6
    if m == 1 or m == 5: a = (n*(n-1)) // 2
    elif m==2 or m == 4: a = (n*(n-2)) // 2
    elif m==3          : a = (n*(3*n-7)) // 6
    else               : a = (n*(3*n-10)) // 6
    print(a, end=", ")
```

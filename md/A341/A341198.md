## Python
```Python
def A341198(n):
  n2=n**2
  x=n
  y=A=0
  while y<=x:
    dx=x**2+(y+1)**2-n2-x>=0
    A+=x+(y!=0 and y!=x)*(x-2*y)+(dx and y==x-1)*(x-1)
    x-=dx
    y+=1
  return 4*A+1
```

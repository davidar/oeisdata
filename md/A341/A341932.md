## Python
```Python
from sympy import isprime
def A341932(n):
    k, m, r = n, n-1, 0 if isprime(n) else -1
    while m > 0:
        k = int(str(k)+str(m))
        if isprime(k):
            r = n-m
        m -= 1
    return r
```

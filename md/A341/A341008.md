## Python
```Python
def ok(n):
  s = str(n)
  return abs(sum(map(int, s))-2*sum(int(d) for d in s if d in "2468")) == 7
print(list(filter(ok, range(900)))) # _Michael S. Branicky_, Jul 18 2021
```

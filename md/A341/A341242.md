## C++
```C++
(first 2^12 terms)
for (int i=0; i<65536; ++i)  {
    int sum=0;
    for (int n=i, count=0; n>0; n>>=1,++count)
      if ((n&1)!=0)
        sum ^= count;
    if (sum==0)
      std::cout << i << ", ";
  }
```
## Python
```Python
def ok(n):
  xor, b = 0, (bin(n)[2:])[::-1]
  for i, c in enumerate(b):
    if c == '1': xor ^= i
  return xor == 0
print([m for m in range(937) if ok(m)]) # _Michael S. Branicky_, Feb 07 2021
```

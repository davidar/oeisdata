## Python
```Python
def runs(n): return bin(n^(n>>1)).count('1')
def a(n):
  nruns, m = runs(n), n + 1
  while runs(m) != nruns: m += 1
  return m
print([a(n) for n in range(1, 67)]) # _Michael S. Branicky_, Feb 24 2021
```

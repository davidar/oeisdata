## Python
```Python
from sympy import isprime, nextprime
p,q,r,s,A341934_list=2,3,5,7,[]
while p < 10**6:
    if isprime(2*q*(p-r)+r*s):
        A341934_list.append(p)
    p, q, r, s = q, r, s, nextprime(s) # _Chai Wah Wu_, Feb 24 2021
```

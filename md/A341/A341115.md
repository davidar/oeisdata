## Python
```Python
A341115_list, k, m, l, n = [], 1, 2**101, 2**101+1, 10**100
while k < 10**6:
    if pow(10,n,l) == l-1:
        A341115_list.append(k)
        print(len(A341115_list),k)
    k += 1
    l += m # _Chai Wah Wu_, Mar 28 2021
```

## Python
```Python
from sympy import fibonacci
f = [fibonacci(n) for n in range(100)]
def a(n):
    out = f[0] + f[1]
    for i in range(1, n):
        if i%2:
            out *= f[i+1]
        else:
            out += f[i+1]
    return out
```

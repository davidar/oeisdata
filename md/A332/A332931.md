## Python
```Python
from math import isqrt
from sympy import divisors
def A332931(n): return sum((m:=isqrt(d))+int(d-m*(m+1)>=1) for d in divisors(n,generator=True)) # _Chai Wah Wu_, Aug 03 2022
```

## Python
```Python
from math import isqrt
def auptorun(runs):
    alst = []
    for n in range(1, runs+1):
        alst.extend(list(range(isqrt(8*10**n) + 1, isqrt(9*10**n - 1)+1)))
    return alst
print(auptorun(5)) # _Michael S. Branicky_, Sep 29 2021
```

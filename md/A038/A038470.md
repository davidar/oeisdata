## Python
```Python
def aupto(limit):
  p = [4**i for i in range(limit//4+1) if 4**i < limit]
  p2 = set(a+b for i, a in enumerate(p) for b in p[i+1:] if a+b <= limit)
  return sorted(p2)
print(aupto(265000)) # _Michael S. Branicky_, May 17 2021
```

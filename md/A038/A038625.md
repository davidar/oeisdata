## Python
```Python
from math import exp
from sympy import primepi
def a(n):
  m = 2 if n == 2 else int(exp(n)) # pi(m) > m/log(m) for m >= 17
  while m != n*primepi(m): m += 1
  return m
print([a(n) for n in range(2, 10)]) # _Michael S. Branicky_, Feb 27 2021
```

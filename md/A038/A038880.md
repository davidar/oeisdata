## Python
```Python
from sympy import isprime, jacobi_symbol
def ok(n): return n%2 == 1 and isprime(n) and jacobi_symbol(10, n) == -1
print([k for k in range(575) if ok(k)]) # _Michael S. Branicky_, May 24 2022
```

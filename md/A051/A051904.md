## Python
```Python
from sympy import factorint
def a(n):
    f = factorint(n)
    l = [f[p] for p in f]
    return 0 if n == 1 else min(l)
print([a(n) for n in range(1, 51)]) # _Indranil Ghosh_, Jul 13 2017
```

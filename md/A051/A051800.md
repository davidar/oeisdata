## Python
```Python
from sympy import isprime, nextprime
def afind(limit):
    p, primorialk = 2, 2
    for k in range(1, limit+1):
        if isprime(2*primorialk + 1):
            print(k, end=", ")
        p = nextprime(p)
        primorialk *= p
afind(400) # _Michael S. Branicky_, Dec 24 2021
```

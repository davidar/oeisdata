## Python
```Python
from sympy import nextprime
def aupto(limit):
    alst, p, q, r = [], 2, 3, 5
    while q <= limit:
        if 2*q > p + r: alst.append(q)
        p, q, r = q, r, nextprime(r)
    return alst
print(aupto(701)) # _Michael S. Branicky_, Nov 17 2021
```

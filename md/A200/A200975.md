## Python
```Python
# prints all numbers on the diagonals of a sq*sq spiral
sq = 5
d = 1
while 2*d - 1 < sq:
    print(4*d*d - 4*d +1)
    print(4*d*d - 4*d +1 + 1* 2* d)
    print(4*d*d - 4*d +1 + 2* 2* d)
    print(4*d*d - 4*d +1 + 3* 2* d)
    d += 1
print(sq*sq)
```

## Sage
```Sage
def A205509(n) :
    f = bin(factorial(n)).lstrip("0b")
    g = bin(factorial(n-1)).lstrip("0b")
    h = "".zfill(len(f)-len(g)) + g
    return sum(a != b for a, b in zip(f, h))
[A205509(k) for k in (1..66)] # Peter Luschny, Jan 31 2012
```

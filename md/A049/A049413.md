## Python
```Python
from sympy import factorial as f, primefactors
def a(n): return max(primefactors(sum(f(k)*f(n-k) for k in range(n+1))))
print([a(n) for n in range(1, 37)]) # _Michael S. Branicky_, Jul 31 2021
```

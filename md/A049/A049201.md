## Python
```Python
from sympy import factorint
def a(n):
    if n == 0: return 1
    f = factorint(n)
    if n == 1 or sum(f.values()) == 1: return int(str(n)+'1')
    return int("".join(str(p)*e for p, e in f.items()))
print([a(n) for n in range(53)]) # _Michael S. Branicky_, Jul 21 2021
```

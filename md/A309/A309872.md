## Python
```Python
s, a, n = "1", [1], 1
while n < 100:
    n = n+1
    d = s[len(s)-1]
    i, aa = 0, 0
    while i < len(s):
        if s[i] == d:
            aa = aa+1
        i = i+1
    s, a = s+str(aa), a+[aa]
for n in range(1, 92): print(a[n-1], end=', ') # _A.H.M. Smeets_, Aug 22 2019
```

## Python
```Python
a, n, p, s = [2,5], 2, 2, 2
while n < 10:
    p, s, n = p*a[len(a)-1], s+a[len(a)-1], n+1
    a = a+[p-s]
for n in range(1, 11): print(a[n-1], end=', ') # _A.H.M. Smeets_, Aug 22 2019
```

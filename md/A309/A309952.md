## Python
```Python
def a(n):
    n = [int(k) for k in bin(n)[2:]]
    if len(n) % 2 != 0:
        n = [0] + n
    result = []
    for i in range(0, len(n), 2):
        result.append(n[i] ^ n[i+1]) #xor
    return int("".join([str(k) for k in result]), 2)
```

## Python
```Python
from sympy import isprime
def afind(limit, startk=0):
    for k in range(startk, limit+1):
        if isprime(8*100**k + 8*10**k + 1): print(k, end=", ")
afind(500) # _Michael S. Branicky_, Dec 12 2021
```

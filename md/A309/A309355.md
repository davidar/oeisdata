## Python
```Python
from sympy import primepi
def A309355(n):
    if n == 1: return 8
    m, k = n, primepi(n) + n + (n>>1)
    while m != k:
        m, k = k, primepi(k) + n + (k>>1)
    return m-1 # _Chai Wah Wu_, Aug 02 2024
```

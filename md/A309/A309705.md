## Python
```Python
def lcmMinusGcd(n):
    retlist = [1]
    for i in range(1, n):
        g = gcd(retlist[i-1], i+1)
        retlist.append( floor(retlist[i-1]*(i+1) / g) - g)
    return ', '.join(map(str,retlist))
```

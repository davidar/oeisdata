## Python
```Python
seq = []
for n in range(1, 100): seq += [n] + n*[-1]
for n in range(2, len(seq)):
  value = n
  gaps = 0
  position = n - 1
  while position < len(seq):
    if seq[position] == -1:
      if gaps > 0: gaps -= 1
      else:
        seq[position] = value
        gaps = value
        value += 1
    position += 1
print(seq)
```

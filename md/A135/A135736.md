## Python
```Python
n=100 #set the number of terms you want to calculate here
ans=0
finalans = []
finalans.append(0) #continuity with A135736
for i in range(1, n+1):
    ans+=(1/i)
    finalans.append(int(round(ans*i)))
print(finalans)
```

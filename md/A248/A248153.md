## Python
```Python
def aupton(nn):
    alst, astr = [10], "X10"
    for n in range(1, nn+1):
        alst.append(7 * int(astr[n]))
        astr += str(alst[-1])
    return alst
print(aupton(72)) # _Michael S. Branicky_, Oct 07 2021
```

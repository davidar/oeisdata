## Python
```Python
from math import prod
def ok(n):
    s = str(n)
    return '0' not in s and set(str(n-prod(int(d) for d in s))) == set(s)
print(list(filter(ok, range(5000)))) # _Michael S. Branicky_, Jul 18 2021
```

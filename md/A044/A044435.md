## Python
```Python
from sympy.ntheory import digits
def cond(n): return "02" in "".join(map(str, digits(n, 3)))
def ok(n): return cond(n) and not cond(n+1)
print([k for k in range(400) if ok(k)]) # _Michael S. Branicky_, Nov 20 2021
```

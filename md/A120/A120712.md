## Python
```Python
from sympy import divisors, isprime
def ok(n):
    s = "".join(str(d) for d in divisors(n)[1:-1])
    return s != "" and isprime(int(s))
print([k for k in range(319) if ok(k)]) # _Michael S. Branicky_, Oct 01 2024
```

## Python
```Python
from gmpy2 import is_prime
A120303 = [2]
for n in range(3, 801):
    for k in range(2*n-1, n, -2):
        if is_prime(k, n):
            A120303.append(k)
            break
for n in range(len(A120303)):
    print(n+2, A120303[n])  # _Gennady Eremin_, Mar 17 2021
```

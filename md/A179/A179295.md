## Python
```Python
from sympy import prime, isprime, nextprime
def a(n):
    p=2
    while not isprime(prime(n) + p + 1): p=nextprime(p)
    return p
print([a(n) for n in range(1, 101)]) # _Indranil Ghosh_, Jun 30 2017
```

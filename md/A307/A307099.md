## Python
```Python
from math import floor,ceil,log
x=2
mindif=1
while True:
    logn=x/log(x,2)
    dif=min(logn-floor(logn),ceil(logn)-logn)
    if dif!=0 and mindif>dif:
        mindif=dif
        print(x,end=", ")
    x+=1
```

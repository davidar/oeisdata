## Python
```Python
import sympy
def A307826(n):
    sympy.sieve.extend_to_no(n)
    s = list(sympy.sieve._list)
    prod = s[0]
    print("1")
    for i in range(1,n):
        prod*=s[i]
        print(sympy.ntheory.factor_.totient(prod)//2)
```

## Python
```Python
def od(n): return bin(n).count("1")%2 == 1
def a(n):
    k = 1
    while not (od(k) and od(k*n)): k += 1
    return k
print([a(n) for n in range(1, 81)]) # _Michael S. Branicky_, Feb 21 2022
```

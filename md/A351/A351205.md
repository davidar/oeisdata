## Python
```Python
from itertools import groupby, product
def ok(n):
    runs = [(k, len(list(g))) for k, g in groupby(bin(n)[2:])]
    return len(runs) > len(set(runs))
print([k for k in range(130) if ok(k)]) # _Michael S. Branicky_, Feb 09 2022
```

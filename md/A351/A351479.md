## Python
```Python
def ok(n):
    ds = list(map(int, str(n)))
    return sum(d for d in ds if d%2==1) == 2*sum(d for d in ds if d%2==0)
print([k for k in range(1, 2173) if ok(k)]) # _Michael S. Branicky_, Feb 12 2022
```

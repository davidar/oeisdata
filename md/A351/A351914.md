## Python
```Python
from sympy import primerange
def average_of_primes_up_to(i):
    primes_up_to_i =  list(primerange(2, i+1))
    return sum(primes_up_to_i) / len(primes_up_to_i)
def a_list():
    return [i for i in range(2, 20) if average_of_primes_up_to(i) >= i / 2]
```

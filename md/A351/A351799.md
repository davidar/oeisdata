## Python
```Python
from num2words import num2words
from unidecode import unidecode
def ok(n): return unidecode(num2words(n, lang='fr')).count("e") == 1
print([k for k in range(311) if ok(k)]) # _Michael S. Branicky_, Feb 19 2022
```

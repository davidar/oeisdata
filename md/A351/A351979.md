## Python
```Python
from sympy import factorint, primepi
def ok(n):
    return all(primepi(p)%2==1 and e%2==0 for p, e in factorint(n).items())
print([k for k in range(15500) if ok(k)]) # _Michael S. Branicky_, Mar 12 2022
```

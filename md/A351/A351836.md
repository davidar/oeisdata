## Python
```Python
def ev(n): return bin(n).count("1")%2 == 0
def a(n):
    k = 3
    while not (ev(k) and ev(k*n)): k += 1
    return k
print([a(n) for n in range(1, 87)]) # _Michael S. Branicky_, Feb 21 2022
```

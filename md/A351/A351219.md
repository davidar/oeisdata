## Python
```Python
from math import prod
from sympy import factorint, fibonacci
def a(n): return prod(fibonacci(e+1) for p, e in factorint(n).items())
print([a(n) for n in range(1, 88)]) # _Michael S. Branicky_, Feb 05 2022
```

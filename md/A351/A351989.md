## Python
```Python
from sympy import prime, fibonacci
from sympy.ntheory import jacobi_symbol
def A351989(n): return fibonacci((p := prime(n))-jacobi_symbol(p,5)) % p**3 # _Chai Wah Wu_, Feb 28 2022
```

## Python
```Python
from sympy import bernoulli, Rational
def a(n):
    return (3**n * bernoulli(n, Rational(1,3))).as_numer_denom()[1]
print([a(n) for n in range(101)])  # _Indranil Ghosh_, Jul 18 2017
```

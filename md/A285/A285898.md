## Python
```Python
from sympy import binomial
def T(n, k):
    n=n - binomial(k, 2)
    if n>0:
        return 1 if n%k==0 else 0
    return 0
for n in range(1, 21): print([T(n, k) for k in range(1, n + 1)]) # _Indranil Ghosh_, Apr 28 2017
```

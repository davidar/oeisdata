## Maple
```Maple
f:= proc(n) local k;
  for k from 1 do
    if has(convert(n*k,base,10),7) then return k fi
  od
end proc:
map(f, [$1..100]); # _Robert Israel_, Nov 26 2019
```

## Python
```Python
def a(n):
    k=1
    while True:
        if "7" in str(n*k): return k
        k+=1
print([a(n) for n in range(1, 101)]) # _Indranil Ghosh_, Apr 27 2017
```

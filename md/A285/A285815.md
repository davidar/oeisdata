## Python
```Python
from sympy import divisors
from sympy.ntheory.factor_ import digits
def ok(n):
   return all(n%sum(digits(d)[1:])==0 for d in divisors(n))
print([n for n in range(1, 20001) if ok(n)]) # _Indranil Ghosh_, Apr 28 2017
```

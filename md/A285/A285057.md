## Python
```Python
from sympy import fibonacci, lcm
def z(n):
    k=1
    while fibonacci(k)%n: k+=1
    return k
def a(n): return lcm(n, z(n))
print([a(n) for n in range(1,201)]) # _Indranil Ghosh_, Apr 09 2017
```

## Python
```Python
from sympy import binomial, bernoulli
def T(n, m):
    return (binomial(n, m) * 2**(n - m) * bernoulli(n - m)).denominator()
for n in range(21): print([T(n, m) for m in range(n + 1)]) # _Indranil Ghosh_, May 06 2017
```

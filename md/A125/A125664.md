## Python
```Python
from sympy import isprime
def ok(n):
    if n < 10: return False
    s = str(n)
    m = len(s)//2
    return isprime(int(s[-m:])) and not isprime(int(s[:m]))
print([k for k in range(188) if ok(k)]) # _Michael S. Branicky_, Dec 13 2021
```

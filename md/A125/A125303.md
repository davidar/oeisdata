## Python
```Python
def ok(n):
    w = str(n)
    ss = set(int(w[i:j+1]) for i in range(len(w)) for j in range(i, len(w)))
    return n > 9 and n == int(str(sum(s for s in ss if s != n))[::-1])
print([k for k in range(80000) if ok(k)]) # _Michael S. Branicky_, Apr 02 2022
```

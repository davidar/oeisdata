## Python
```Python
from itertools import combinations
def A364463(n): return sum(1 for l in range(n+1) for c in combinations(range(1,n+1),l) if set(c).isdisjoint({c[i+1]-c[i] for i in range(l-1)})) # _Chai Wah Wu_, Sep 26 2023
```

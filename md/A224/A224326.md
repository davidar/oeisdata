## Python
```Python
TOP = 777
for n in range(TOP):
  k = 0
  for x in range(TOP):
    s = x*(x+1)//2
    if s>n: break
    for y in range(x+1,TOP):
        sy = s + y*(y+1)//2
        if sy>n: break
        for z in range(y+1,TOP):
          sz = sy + z*(z+1)//2
          if sz>n: break
          if sz==n: k+=1
  print(str(k), end=',')
```

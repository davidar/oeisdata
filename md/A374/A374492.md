## Python
```Python
from math import isqrt
def ok(n): return n%10 in {0, 1, 4, 5, 6, 9} and isqrt(n)**2 < n
print([k for k in range(126) if ok(k)]) # _Michael S. Branicky_, Aug 04 2024
```

## Python
```Python
from sympy import integer_nthroot
def A374025(n): return max(sum(int(d) for d in str(m**5)) for m in range((lambda x:x[0]+(x[1]^1))(integer_nthroot(10**(n-1),5)),1+integer_nthroot(10**n-1,5)[0])) # _Chai Wah Wu_, Jun 26 2024
```

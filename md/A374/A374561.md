## Python
```Python
from sympy.ntheory import is_palindromic
def ok(n):
    c = 0
    for b in range(2, 11):
        c += int(is_palindromic(n, b))
        if c > 1: return True
    return False
print([k for k in range(1, 180) if ok(k)]) # _Michael S. Branicky_, Aug 02 2024
```

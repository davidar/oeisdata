## Python
```Python
def PS(n, k):
    if n == 1 or k == 1: return 1
    if n == 0 or k == 0: return 0
    m = pow(n, k // 2, k)
    if k // 2 < m: m -= k
    return m
for n in range(12): print([PS(n, k) for k in range(12)])
```

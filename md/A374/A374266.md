## Python
```Python
def a(n):
    reach = {(1, 1)}
    for _ in range(n-1):
        newreach = set()
        for a, b in reach:
            newreach.update([(b, a+b), (b, int(str(a+b).replace('0', '')))])
        reach = newreach
    return min(reach, key = lambda k:k[0])[0]
```

## Python
```Python
def a(n):
  s=''
  for b in bin(n)[2:]:
    s += '01'*(b=='1') + '10'*(b=='0')
  return int(s,2)
print([a(n) for n in range(0,51)])
```
```Python
def a(n):
  d = {'0': '10', '1': '01'}
  return int(''.join(map(d.get, bin(n)[2:])), 2)
```

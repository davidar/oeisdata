## Python
```Python
from sympy import compositepi, isprime
def ok(n):
    return not isprime(n) and isprime(n+1) and isprime(compositepi(n)-1)
print([k for k in range(1549) if ok(k)]) # _Michael S. Branicky_, Dec 28 2021
```

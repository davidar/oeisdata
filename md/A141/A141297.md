## Python
```Python
def a(n):
  b = bin(n)[2:]
  m = len(b)
  return len(set(b[i:j] for i in range(m) for j in range(i+1, m+1)))
print([a(n) for n in range(1, 78)]) # _Michael S. Branicky_, Jan 20 2021
```

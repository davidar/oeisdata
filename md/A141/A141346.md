## Python
```Python
from sympy import factorint
def a(n):
    s = set()
    for p in factorint(n): s |= set(str(p))
    return sum(map(int, s))
print([a(n) for n in range(1, 91)]) # _Michael S. Branicky_, Dec 12 2023
```

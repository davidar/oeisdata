## Python
```Python
def binpal(n): b = bin(n)[2:]; return b == b[::-1]
def a(n):
    m = 2*n - 1
    km = m
    while not binpal(km): km += m
    return km//m
print([a(n) for n in range(1, 80)]) # _Michael S. Branicky_, Mar 20 2022
```

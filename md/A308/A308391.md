## Python
```Python
import math
ende = 1
for i in range(1,10):
   anz = 0
   for a in range(ende, 10*ende):
      z = math.ceil((ende*ende*10)/a)
      if z < ende*10:
         anz = anz + ende*10 - z
   ende = ende*10
   print(i, anz)
```

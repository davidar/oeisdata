## Python
```Python
from sympy import isprime, lcm
def A074200(n):
    a = lcm(range(1,n+1))
    m = a
    while True:
        for k in range(n,0,-1):
            if not isprime(m//k+1):
                break
        else:
            return m
        m += a # _Chai Wah Wu_, Feb 27 2019
```

## Python
```Python
def aupton(nn):
  alst = [6, 1, 3, 7, 15, 31]
  for n in range(6, nn+1): alst.append(sum(alst[n-6:n]))
  return alst[:nn+1]
print(aupton(33)) # _Michael S. Branicky_, Jun 01 2021
```

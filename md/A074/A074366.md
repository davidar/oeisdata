## Python
```Python
from sympy import nextprime, prime, primerange
def a(n):
  return nextprime(int("".join(map(str, primerange(2, prime(n)+1)))))
print([a(n) for n in range(1, 17)]) # _Michael S. Branicky_, Mar 29 2021
```

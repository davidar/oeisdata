## Python
```Python
from sympy import divisor_sigma as sigma, totient as phi
def ok(n): return sigma(phi(n)) == phi(sigma(n))
def aupto(nn): return [m for m in range(1, nn+1) if ok(m)]
print(aupto(10**4)) # _Michael S. Branicky_, Jan 09 2021
```

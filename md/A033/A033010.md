## Python
```Python
from sympy.ntheory import digits
from itertools import groupby
def ok(n):
  return all(len(list(g))==2 for k, g in groupby(digits(n, 12)[1:]))
print(list(filter(ok, range(1, 7502)))) # _Michael S. Branicky_, Apr 27 2021
```

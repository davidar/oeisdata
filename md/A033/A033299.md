## Python
```Python
from sympy import isprime, nextprime
def a(n):
  p = nextprime(n)
  while not isprime((p-1)//2): p = nextprime(p)
  return p
print([a(n) for n in range(1, 68)]) # _Michael S. Branicky_, May 05 2021
```

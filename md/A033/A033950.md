## Maple
```Maple
with(numtheory):
A033950 := proc(n)
    option remember:
    local k:
    if n=1 then
        return 1:
    else
        for k from procname(n-1)+1 do
            if type(k/tau(k), integer) then
                return k:
            end if:
        end do:
    end if:
end proc:
seq(A033950(n), n=1..56); # _Nathaniel Johnston_, May 04 2011
```

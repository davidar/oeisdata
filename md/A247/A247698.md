## Maple
```Maple
Brady1 := proc(n::posint)
option remember, system;
if n = 1 then
  2308
elif n = 2 then
  4261
else
  thisproc( n - 1 ) + thisproc( n - 2 )
end if
end proc:
seq( Brady1( n ), n = 1 .. 100 );
```

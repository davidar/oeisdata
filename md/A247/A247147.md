## Python
```Python
from sympy import isprime
from itertools import count, islice
def agen(startk=1):
    for k in count(startk):
        if isprime(3*k-4) and isprime(2**k-1):
            yield k
print(list(islice(agen(), 12))) # _Michael S. Branicky_, Jul 31 2022
```

## Python
```Python
for n in range(60):
  sq=n*n
  cb=n**3
  s = t = 0
  while 1:
    tn = t*(t+1)/2
    if tn>sq and tn<cb:  s+=1
    elif tn>=cb:  break
    t+=1
  print str(s)+',',
```

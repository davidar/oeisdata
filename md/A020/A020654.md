## Julia
```Julia
function a(n)
    m, r, b = n, 0, 1
    while m > 0
        m, q = divrem(m, 4)
        r += b * q
        b *= 5
    end
r end; [a(n) for n in 0:66] |> println # _Peter Luschny_, Jan 03 2021
```

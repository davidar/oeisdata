## Python
```Python
from sympy import primeomega
def aupto(limit):
  prv, nxt, alst = 1, 2, []
  for n in range(3, limit+1, 2):
    if prv == nxt: alst.append(n)
    prv, nxt = nxt, primeomega(n+3)
  return alst
print(aupto(665)) # _Michael S. Branicky_, May 19 2021
```

```Maple
      if n=0 then 0
```

## Python
```Python
def A305876(n):
    m, tlist, s = 2**n, [1,2], 0
    while tlist[-1]+tlist[-2] <= m:
        tlist.append(tlist[-1]+tlist[-2])
    for d in tlist[::-1]:
        s *= 2
        if d <= m:
            s += 1
            m -= d
    return s # _Chai Wah Wu_, Jun 14 2018
```

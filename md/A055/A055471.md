## MATLAB
```MATLAB
m=1;
for n=1:1000
    v=dec2base(n,10)-'0';
    v = v(v~=0);
    if mod(n,prod(v))==0
        sol(m)=n;
        m=m+1;
    end
end
sol % _Marius A. Burtea_, May 07 2019
```

## Python
```Python
from math import prod
def ok(n): return n > 0 and n%prod([int(d) for d in str(n) if d!='0']) == 0
print(list(filter(ok, range(501)))) # _Michael S. Branicky_, Jul 27 2021
```

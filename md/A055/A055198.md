## Python
```Python
from sympy import cycle_length
f=lambda x:int(''.join(reversed(str(x+4))))
def is_A055198(n):
    return n in cycle_length(f,n,values=True)
print([n for n in range(1016) if is_A055198(n)]) # _Pontus von Br√∂mssen_, Mar 19 2022
```

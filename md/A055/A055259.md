## Python
```Python
def valuation(n, b):
  v = 0
  while n > 1: n //= b; v += 1
  return v
def aupto(lim):
  pows8 = [8**i for i in range(valuation(lim-1, 8) + 1)]
  sum_pows8 = sorted([a+b for i, a in enumerate(pows8) for b in pows8[i:]])
  return [s for s in sum_pows8 if s <= lim]
print(aupto(16777217)) # _Michael S. Branicky_, Feb 09 2021
```

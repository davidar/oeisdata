## Python
```Python
def ok(n):
    d, lim = 1, 1
    while lim < n*9*d: d, lim = d+1, lim*10
    return not any(sum(map(int, str(k**n))) == k for k in range(2, lim+1))
for k in range(195):
    if ok(k): print(k, end=", ") # _Michael S. Branicky_, Jul 06 2022
```

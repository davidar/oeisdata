## Python
```Python
from sympy import nextprime, ff
def A133907(n):
    p, m = 2, (n+2)*(n+1)>>1
    while m%p != 1:
        q = nextprime(p)
        m = m*ff(n+q,q-p)//ff(q,q-p)
        p = q
    return p # _Chai Wah Wu_, Feb 22 2023
```

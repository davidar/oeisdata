## Python
```Python
from math import isqrt
from sympy import prime, primepi
def A128301(n):
    m = prime(n)**2
    return int(sum(primepi(m//prime(k))-k+1 for k in range(1,n+1))) # _Chai Wah Wu_, Jul 23 2024
```

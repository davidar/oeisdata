## Python
```Python
from math import prod
def ok(n):
    d = list(map(int, str(n)))
    p, s = prod(d), sum(d)
    return sum(map(int, str(p))) == prod(map(int, str(s)))
print([k for k in range(1, 803) if ok(k)]) # _Michael S. Branicky_, Jun 15 2024
```

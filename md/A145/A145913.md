## Python
```Python
import math
def a(n):
    k=1
    while True:
        if (1/math.log((1 + k)**(1/k))) > n: return k
        else: k+=1
print([a(n) for n in range(1, 101)]) # _Indranil Ghosh_, Mar 23 2017
```

## Python
```Python
from math import *
i=1
while i<(10**10-1)/9:
    i=10*i+1
    for m in range(1,10):
        q=i*m
        q2=q
        for b in range(2,10):
            restes=[]
            q=q2
            while q>0:
                r=q%b
                q=q//b
                restes.append(r)
            if restes==[restes[0]]*len(restes):
                print(q2,restes,"en base ",b)
```

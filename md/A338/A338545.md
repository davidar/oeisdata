## Python
```Python
sequence = []
escape = [1090089, 99100089, 9900109899999990109989]
for k in range(1, 100000):
    x = k
    while not x==0:
       if x <= rev(x):
          x = rev(x) - x
       else:
          x = rev(x) + x
       if x in escape:
          x = 0
          sequence.append(k)
```

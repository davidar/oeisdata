## Python
```Python
import sympy
def A338400(n):
  T=[]
  for k in range(n+1):
    T.append(sympy.npartitions(k))
    T.reverse()
    for i in range(k):
      T[i+1]=T[i]-T[i+1]
  return T[-1]
```

## Python
```Python
from sympy import isprime
from math import sqrt
m = 2
while m >= 2:
    d = 0
    a = 0
    while d < sqrt(2*m - 1):
        p = m - d
        q = m + d
        if isprime(p) == 1 and isprime(q) == 1:
            a += 1
        d += 1
    print (a)
    m += 1
```

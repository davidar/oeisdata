## Python
```Python
#Defines digital root
def recsum(n):
    nstr = str(n)
    if len(nstr) == 1:
        return int(nstr)
    else:
        nl = sum([int(j) for j in nstr])
        return recsum(nl)
#Prints first 1000 terms
for i in range(1000):
    print(recsum(i)//len(str(i)))
```

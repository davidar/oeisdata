## Python
```Python
from sympy import prime
def a(n):
    pn = prime(n)
    m = 2*pn
    while str(m)[0] != '6': m += pn
    return m
print([a(n) for n in range(1, 53)]) # _Michael S. Branicky_, Jun 06 2021
```

## Python
```Python
from math import gcd
from functools import reduce
def ok(n):
    d = list(map(int, str(n)))
    return len(d) == 1 or reduce(gcd, d) == 1
print([k for k in range(100) if ok(k)]) # _Michael S. Branicky_, Dec 23 2022
```

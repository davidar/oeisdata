## Python
```Python
def sbd(n): return sum(map(int, str(bin(n)[2:])))
def niv2(n): return n%sbd(n) == 0
def aupto(nn): return [k for k in range(1, nn+1) if niv2(k) and niv2(k+1)]
print(aupto(1424)) # _Michael S. Branicky_, Jan 20 2021
```

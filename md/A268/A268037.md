## Python
```Python
from sympy import divisors
def ok(n): return n%len(divisors(n+2)) == 0 and (n+2)%len(divisors(n)) == 0
print(list(filter(ok, range(1, 9711)))) # _Michael S. Branicky_, Apr 30 2021
```

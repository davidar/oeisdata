## Python
```Python
def electron_period(n):
  wire_mask = (1 << n) - 1
  power = lam = 1
  tortoise, hare = 1, 2
  while tortoise != hare:
    if power == lam:
      tortoise = hare
      power *= 2
      lam = 0
    hare = ((hare << 1) ^ (hare >> 1)) & wire_mask
    lam += 1
  return lam
```

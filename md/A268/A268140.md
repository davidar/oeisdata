## Python
```Python
from sympy import isprime
def A268140(n):
    p, n2 = 2, 2**n+1
    while True:
        for i in range(1,n2):
            if isprime(p+i):
                p += i
                break
        else:
            return p # _Chai Wah Wu_, Feb 15 2016
```

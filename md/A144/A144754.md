## Python
```Python
from sympy import isprime
i=j=1
while j<=250:
    if isprime(bin(i)[2:].count("0")):
        print(str(j)+" "+str(i))
        j+=1
    i+=1 # _Indranil Ghosh_, Feb 03 2017
```

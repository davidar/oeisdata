## Python
```Python
from sympy import isprime
def ok(n): return isprime(c:=n.bit_count()) and isprime(n.bit_length()-c) and isprime(n)
print([k for k in range(710) if ok(k)]) # _Michael S. Branicky_, Dec 27 2023
```

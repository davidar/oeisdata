## Python
```Python
from sympy import isprime, prevprime
def A335313(n):
    m = 2**(3*2**n)
    p = prevprime(m)
    while not isprime((p-1)//2):
        p = prevprime(p)
    return m-p # _Chai Wah Wu_, Jul 09 2020
```

## Python
```Python
from math import ceil
sequence = [1]
terms = 1000
for n in range(2, terms + 1):
    nextnum = 1
    while nextnum in sequence:
        nextnum = nextnum * ceil(sequence[-1] / 2) + 1
    sequence.append(nextnum)
print(sequence)
```

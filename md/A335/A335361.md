## Python
```Python
from sympy import isprime, nextprime, factorial
A335361_list, p = [], 2
while p < 500:
    f, g = factorial(p), 1
    for i in range(1,p+1):
        g += f
        if isprime(g):
            break
    else:
        A335361_list.append(p)
    p = nextprime(p)
```

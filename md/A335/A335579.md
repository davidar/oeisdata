## Python
```Python
from sympy import prime, divisors
def a(n): return sum(set(d for c in range(prime(n)+1, prime(n+1)) for d in divisors(c)))
print([a(n) for n in range(2, 59)]) # _Michael S. Branicky_, Jul 12 2021
```

## Python
```Python
from sympy import factorint, prevprime
from operator import mul
def a(n):
    f=factorint(2*n - 1)
    return 1 if n==1 else reduce(mul, [prevprime(i)**f[i] for i in f]) # _Indranil Ghosh_, May 13 2017
```

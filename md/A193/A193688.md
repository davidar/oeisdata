## Python
```Python
for i in range(1, 100):
    n = 2**i - 1
    x = n
    c = 0
    while x != 1:
        c = c + 1
        if (x & 1) == 0:
            x = x//2
        else:
            x = 3*x + 1
    print(c)
# _David Rabahy_, Sep 18 2017
```

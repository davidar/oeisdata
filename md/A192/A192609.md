## Python
```Python
from sympy import isprime
def aupto(N): return [k for k in range(N+1) if isprime(int("".join(str(k**i) for i in [4, 3, 2, 1, 0])))]
print(aupto(745)) # _Michael S. Branicky_, Dec 16 2021
```

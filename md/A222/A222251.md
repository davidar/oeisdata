## Python
```Python
def flp69(s): return s.replace("6", "-").replace("9", "6").replace("-", "9")
def aupto(lim): return [int(flp69(s)) for s in map(str, range(lim+1))]
print(aupto(75)) # _Michael S. Branicky_, Sep 07 2021
```

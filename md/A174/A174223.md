## Python
```Python
from sympy import nextprime
def a(n):
    base = 2*n + 1
    p1 = nextprime(base)
    p2 = nextprime(p1)
    p3 = nextprime(p2)
    return base + p1 + p2 + p3
print([a(n) for n in range(74)]) # _Michael S. Branicky_, Nov 26 2021
```

## Python
```Python
from sympy import primefactors
def ok(n):
    if n==1 or max(primefactors(n))**3<=n: return True
    else: return False
print([n for n in range(1, 1001) if ok(n)]) # _Indranil Ghosh_, Apr 23 2017
```

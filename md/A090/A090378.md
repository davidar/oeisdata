## Python
```Python
# uses alst(), swap() in A089473
nn = 24
mid = (nn-1)//2
startlst = [chr(i) for i in range(45, 45+nn**2)] # chr(45) is "-"
startlst[0], startlst[mid] = startlst[mid], startlst[0]
start = "".join(startlst)
print(alst(start, (nn, nn), maxd=mid)) # _Michael S. Branicky_, Jan 02 2021
```

## Python
```Python
def a(n):
    b, m, t = 5**n+1, 10**n, 10**(n-1)
    return len(set(p for p in (pow(2, k, m) for k in range(b)) if p >= t))
print([a(n) for n in range(1, 10)]) # _Michael S. Branicky_, Jul 25 2022
```

## Python
```Python
from sympy import factorint
def a(n):
    if n == 1: return 1
    return int("".join(bin(p)[2:]*e for p, e in factorint(n).items()), 2)
print([a(n) for n in range(1, 65)]) # _Michael S. Branicky_, Oct 07 2022
```

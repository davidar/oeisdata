## Python
```Python
def ways (n, left = 5, last = 1):
  a = last; a3 = a**3; c = 0
  while a3 <= n-left+1:
    if left > 1:
       c += ways(n-a3, left-1, a)
    elif a3 == n:
       c += 1
    a += 1; a3 = a**3
  return c
```

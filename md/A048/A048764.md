## Python
```Python
from sympy import factorial as f
def a(n):
    k=1
    while f(k + 1)<=n: k+=1
    return f(k)
print([a(n) for n in range(1, 101)]) # _Indranil Ghosh_, Jun 21 2017, after Mathematica code
```

```Mathematica
a[n_] := Module[{i = 1}, While[DigitCount[i^2][[6]] != n, i++;]; i]; (* Sam Handler (sam_5_5_5_0(AT)yahoo.com), Aug 20 2006 *)
```

## Python
```Python
def a(n):
  k = 1
  while not str(k**2).count('6') == n: k += 1
  return k
print([a(n) for n in range(1, 11)]) # _Michael S. Branicky_, Apr 26 2021
```

## Python
```Python
from sympy import primerange, isprime
for n in primerange(2,600):
    t=int(''.join(str(int(i)**3) for i in str(n)))
    if sympy.isprime(t):
         print(n)
# _Abhiram R Devesh_, Feb 09 2015
```

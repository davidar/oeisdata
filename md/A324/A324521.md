## Python
```Python
from sympy import factorint, primepi
def ok(n):
    f = factorint(n)
    return primepi(max(f)) <= sum(f.values())
print([k for k in range(2, 257) if ok(k)]) # _Michael S. Branicky_, Nov 15 2022
```

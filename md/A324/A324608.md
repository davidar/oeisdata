## Python
```Python
def aupton(terms):
  alst, bstr = [1, 1], "110"
  for n in range(3, terms+1):
    an = int(bstr[:n], 2) - int(bstr[:n-1], 2)
    binan = bin(an)[2:]
    alst, bstr = alst + [binan.count('1')], bstr + binan
  return alst
print(aupton(68)) # _Michael S. Branicky_, Mar 30 2021
```

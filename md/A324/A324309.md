## Python
```Python
def A324309(n):
    m, k = 2, 2**n
    while True:
        s = str(k)
        for i in range(1,len(s)):
            if s[i] == s[i-1]:
                return m
        m += 1
        if m % 10 == 0:
            m += 1
        k = m**n
```

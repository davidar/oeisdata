## Python
```Python
from sympy import prime, nextprime
def aupton(terms):
  prod, p, alst = 1, 2, []
  while len(alst) < terms:
    p, prod = nextprime(p), prod * prime(p)
    alst.append(prod)
  return alst
print(aupton(16)) # _Michael S. Branicky_, Mar 15 2021
```

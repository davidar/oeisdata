## Python
```Python
def a(n):
  s = "".join(sorted("".join(map(str, range((n-1)*n//2+1, n*(n+1)//2+1)))))
  return int("".join(sorted(s, reverse=True)))
print([a(n) for n in range(1, 15)]) # _Michael S. Branicky_, Jan 23 2021
```

## MATLAB
```MATLAB
function [a] = A342131(max_n)
    for n = 1:max_n
        m = n-1;
        if floor(m/2) == m/2
            a(n) = (m/2)+floor(m/4);
        else
            a(n) = (m*3+1)/2;
        end
    end
end
```

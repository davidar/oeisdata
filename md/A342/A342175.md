## Python
```Python
from sympy import isprime, gcd, composite
def A342175(n):
    m = composite(n)
    k = m+1
    while gcd(k,m) != 1 or isprime(k):
        k += 1
    return k-m # _Chai Wah Wu_, Mar 28 2021
```

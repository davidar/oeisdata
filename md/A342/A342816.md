## Python
```Python
seq=[]
for n1 in range(20):
    for n2 in range(20):
        n=(2**(2*n1+6*n2+10) - 2**(2*n1+2) - 3)/9
        seq.append(n)
seq.sort()
print(seq[0:50])
```

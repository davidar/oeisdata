## Python
```Python
from math import prod
def ok(n):
  digs = list(map(int, str(n)))
  return sum(digs) == prod([d for d in digs if d != 0])
def aupto(lim): return [m for m in range(1, lim+1) if ok(m)]
print(aupto(2200)) # _Michael S. Branicky_, Feb 26 2021
```

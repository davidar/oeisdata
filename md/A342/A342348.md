# Smallest number \> 3 whose representation in all bases from 2 up to n consists only of '0's, '1's, '2's and '3's\.
https://oeis.org/A342348

## Python
```Python
def only0123(n, b):
  while n >= b:
    n, r = divmod(n, b)
    if r > 3: return False
  return n <= 3
def a(n):
  k = max(4, n)
  while not all(only0123(k, b) for b in range(2, n+1)): k += 1
  return k
print([a(n) for n in range(2, 10)]) # _Michael S. Branicky_, Mar 12 2021
```

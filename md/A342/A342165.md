## Python
```Python
from sympy import isprime
def aupton(terms):
  alst, idx = [1], 1
  for n in range(2, terms+1):
    if isprime(n): an = max(alst) + 1
    else: an, idx = alst[idx], idx + 1
    alst.append(an)
  return alst
print(aupton(78)) # _Michael S. Branicky_, Mar 03 2021
```

## Python
```Python
def ok(n):
  b = bin(n)[2:]
  for i in range(1, (len(b)+1)//2):
    if b[:i] == b[-i:]: return True
  return False
def aupto(lim): return [m for m in range(lim+1) if ok(m)]
print(aupto(101)) # _Michael S. Branicky_, Mar 09 2021
```

## Python
```Python
from sympy import isprime
A342221_list = []
for k in range(1,10**3):
    if k % 3 != 1:
        m, l = (10**k-1)//9, 2
        for i in range(k):
            if isprime(m+l):
                break
            l *= 10
        else:
            A342221_list.append(k) # _Chai Wah Wu_, Mar 15 2021
```

## Python
```Python
from sympy import factorint
def ispal(s): return s == s[::-1]
def ok(n): return n > 0 and all(ispal(bin(f)[2:]) for f in factorint(n))
print([k for k in range(528) if ok(k)]) # _Michael S. Branicky_, Jan 17 2022
```

## Python
```Python
def a(n):
  b = bin(n)[2:]
  for i in range(1, len(b)+1):
    if b[:i] == b[-i:]: return int(b[:i], 2)
print([a(n) for n in range(80)]) # _Michael S. Branicky_, Mar 07 2021
```

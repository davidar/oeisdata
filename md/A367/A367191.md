## Mathematica
```Mathematica
s[n_] := Solve[j > 0 && k > 0 && k^2 + k*n + n^2 == j^2, {j, k},Integers];
```
```Mathematica
a[n_] := If[n == 0, 0, With[{sn = s[n]}, Which[sn == {}, 0, IntegerQ[k /. sn[[1]]], Max[k /. sn], True, 0]]];
```

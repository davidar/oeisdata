## Sage
```Sage
def a(k, p):
    return (2/(p-1)) * sum(euler_phi(d) * binomial((p-1)/(2*d), k/(2*d)) for d in divisors(gcd(k, p-1)/2)) # see Mishna; beware the missing prefactor (2/(p-1))
print([a(2*n, 53) for n in range(27)]) # _Andrey Zabolotskiy_, Nov 22 2023
```

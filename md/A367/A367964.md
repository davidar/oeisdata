## Python
```Python
# A purely additive construction:
from functools import cache
@cache
def a_row(n: int) -> list[int]:
    if n == 0: return [0]
    row = a_row(n - 1) + [0]
    for k in range(n): row[k] += n
    row[n] = row[n - 1] + n
    return row
```

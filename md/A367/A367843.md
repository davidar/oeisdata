## Python
```Python
from collections import Counter
from sympy import prime
def A367843(n):
    c = {0:1}
    for k in range(1,n+1):
        p, b = prime(k), Counter(c)
        for j in c:
            b[j+p] -= c[j]
        c = b
    return max(map(abs,c.values())) # _Chai Wah Wu_, Feb 06 2024
```

## Maple
```Maple
g:= proc(x, y) option remember;
      `if`(y=0, 0, 1+g(y, irem(x, y)))
    end:
a:= proc(n) option remember; `if`(n=0, 0,
      a(n-1)+add(g(j, n), j=1..n-1))
    end:
seq(a(n), n=1..100);  # _Alois P. Heinz_, Dec 05 2023
```

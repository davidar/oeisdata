## Python
```Python
from sympy.ntheory.factor_ import digits
def a(n):
    b = n + 3*(n%3)
    return next((b+y for y in [1, 2] if digits(b+y, 3)[1] == y), -1)
print([a(n) for n in range(1, 82)]) # _Michael S. Branicky_, Dec 11 2023
```

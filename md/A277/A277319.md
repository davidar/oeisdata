## Python
```Python
from sympy import factorint, primepi, isprime
def a048675(n):
    if n==1: return 0
    f=factorint(n)
    return sum([f[i]*2**(primepi(i) - 1) for i in f])
print([n for n in range(1, 1001) if isprime(a048675(n))]) # _Indranil Ghosh_, Jun 19 2017
```

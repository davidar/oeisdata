## Python
```Python
from sympy import primefactors
from gmpy2 import is_prime, isqrt
A277624_list = []
for n in range(2,10**3):
    if not is_prime(n):
        for p in primefactors(n):
            if isqrt(p)*p > n:
                A277624_list.append(n)
                break # _Chai Wah Wu_, Oct 25 2016
```

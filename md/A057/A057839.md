## Python
```Python
def afind(limit):
  for k in range(1, limit+1, 2):
    if (1 + sum(pow(e, k, k) for e in [2, 4, 6, 8, 10])) % k == 0:
      print(k, end=", ")
afind(10**6) # _Michael S. Branicky_, Apr 01 2021 after _Robert G. Wilson v_
```

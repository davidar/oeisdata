## Python
```Python
from gmpy2 import isqrt_rem, lucas
def A057045(n):
    i, j = isqrt_rem(2*lucas(n-1))
    return int(i + int(4*(j-i) >= 1)) # _Chai Wah Wu_, Aug 16 2016
```

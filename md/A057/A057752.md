## Python
```Python
from sympy import N, li, primepi, floor
def round(n):
    return int(floor(n+0.5))
def A057752(n):
    return round(N(li(10**n),10*n)) - primepi(10**n) # _Chai Wah Wu_, Apr 30 2018
```

## Python
```Python
from sympy import isprime
def a(n):
    k, c = 1, 2**(2**n-1)
    while not isprime(k*c - 1): k += 1
    return k
print([a(n) for n in range(1, 12)]) # _Michael S. Branicky_, May 25 2022
```

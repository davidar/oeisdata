## Python
```Python
from sympy import isprime
A057699_list = []
for i in range(10**6):
    if i % 10:
        p = int(str(i**3)[::-1])
        if isprime(p):
            A057699_list.append(p)
A057699_list = sorted(A057699_list) # _Chai Wah Wu_, Jun 02 2016
```

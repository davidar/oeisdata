## Python
```Python
from sympy import isprime, prime
def a(n):
    m, pn = 1, prime(n)
    while not isprime(1 + pn*2**m): m += 1
    return 1 + pn*2**m
print([a(n) for n in range(1, 21)]) # _Michael S. Branicky_, May 27 2022
```

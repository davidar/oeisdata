## Python
```Python
from sympy import divisor_count as tau
def afind(klimit, kstart=1):
    for k in range(kstart, klimit+1):
        m = 2**k - 1
        if 2**tau(m) == m + 1: print(m, end=", ")
afind(klimit=100) # _Michael S. Branicky_, Dec 16 2021
```

## Python
```Python
from sympy import hermite, Poly, sqrt, symbols
x = symbols('x')
def a(n): return Poly(1/2**n*hermite(n, sqrt(x/2))**2, x).all_coeffs()[::-1]
for n in range(11): print(a(n)) # _Indranil Ghosh_, May 26 2017
```

## Python
```Python
from sympy import nextprime
def aupto(limit):
    p, s, alst = 2, 2, []
    for k in range(1, limit+1):
        if s == 0: alst.append(k)
        p = nextprime(p); s = (s + p)%11
    return alst
print(aupto(759)) # _Michael S. Branicky_, Aug 17 2021
```

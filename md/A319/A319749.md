## Python
```Python
def aupton(nn):
    hn, hd, alst = 3, 1, [3]
    for n in range(nn):
        hn, hd = (hn**2 + 13*hd**2)//2, hn*hd
        alst.append(hn)
    return alst
print(aupton(7)) # _Michael S. Branicky_, Mar 16 2022
```

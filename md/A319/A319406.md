## Python
```Python
from sympy import prime
def a(n):
    b = bin(prime(n))[2:]
    return int(b.replace("01", "001").replace("10", "110") + b[-1], 2)
print([a(n) for n in range(1, 48)]) # _Michael S. Branicky_, Dec 07 2021
```

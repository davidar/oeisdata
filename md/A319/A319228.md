## Python
```Python
from sympy import isprime
def A319228(n):
    c, b, b2, n10 = 0, 1, 3, 10**n
    while b <= n10:
        if isprime(b2):
            c += 1
        b += 1
        b2 += 2*b
    return c # _Chai Wah Wu_, Sep 17 2018
```

## Python
```Python
from sympy import divisors
def ok(n):
    if n < 2 or n%2 == 1: return False
    return not any(n//2 <= d*d < 2*n for d in divisors(n, generator=True))
print(list(filter(ok, range(285)))) # _Michael S. Branicky_, Oct 14 2021
```

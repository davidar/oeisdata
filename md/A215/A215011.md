## Python
```Python
TOP = 333
prpr = y = 0
prev = k = 1
res = [-1]*TOP
while y<TOP-1:
    for i in range(1, TOP):
      if res[i]<0 and prev % int(i*(i+1)/2) == 0:
        res[i] = k
        y += 1
    curr = prpr+prev
    prpr = prev
    prev = curr
    k += 1
for i in range(1, TOP):
    print(res[i], end=', ')
```

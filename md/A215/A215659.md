## Mathematica

## Python
```Python
from sympy import primorial, integer_nthroot
A215659_list = []
for i in range(1,10**2):
    a, b = integer_nthroot(4*primorial(i)+1,2)
    if b:
        A215659_list.append((a+1)//2) # _Chai Wah Wu_, Apr 01 2021
```

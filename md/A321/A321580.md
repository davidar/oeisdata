## Python
```Python
for n in range(1, 501):
    cards = [i for i in range(1, n + 1)]
    reverse = cards[::-1]
    shuffled = cards.copy()
    reversein = False
    for i in range(n):
        evens = shuffled[1::2]
        odds = shuffled[0::2]
        shuffled = evens + odds
        if shuffled == reverse:
            reversein = True
            print(n, end=", ")
            break
```

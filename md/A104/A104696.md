## Python
```Python
def f(d): return 10 - d if d%2 == 1 else d
def a(n): return int("".join(str(f(int(d))) for d in str(n)))
print([a(n) for n in range(1, 73)]) # _Michael S. Branicky_, Sep 02 2021
```

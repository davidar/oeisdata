## Python
```Python
from sympy import factorint
def A099303(n):
    for m in range(n**2>>2,0,-1):
        if sum((m*e//p for p,e in factorint(m).items())) == n:
            return m
    return 0 # _Chai Wah Wu_, Sep 12 2022
```

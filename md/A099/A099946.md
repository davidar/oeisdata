## Python
```Python
from math import gcd
def lcm(a, b):
    return (a * b) // gcd(a, b)
def f(lim):
    l = 1
    for n in range(2, lim + 1):
        l = lcm(n, l)
        print(n, l // (n * (n - 1)))
f(100) # _Luke March_, Jul 23 2014
```

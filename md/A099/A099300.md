## Python
```Python
# call with d = 0..8 for A099291..A099299
from sympy import S
def a(n, d=9): return str(S.Pi.n(10**n + 2))[2:-1].count(str(d))
print([a(n) for n in range(1, 7)]) # _Michael S. Branicky_, Jan 14 2021
```

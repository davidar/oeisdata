## Python
```Python
from sympy.ntheory.factor_ import primeomega
def ok(n): return primeomega(2**n-n-1) == 2
print([2**m-m-1 for m in range(2, 100) if ok(m)]) # _Michael S. Branicky_, Apr 26 2021
```

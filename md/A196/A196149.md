## Python
```Python
from sympy import divisors
def ok(n):
    d = divisors(n)
    return all(d[i]/d[i-1] <= 3 for i in range(1, len(d)))
print(list(filter(ok, range(1, 151)))) # _Michael S. Branicky_, Jun 25 2021
```

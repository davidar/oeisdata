## Python
```Python
from sympy import factorint
def ok(n):
    e = list(factorint(n).values())
    return 1 < len(e) == len(set(e)) and e == sorted(e, reverse=True)
print([k for k in range(289) if ok(k)]) # _Michael S. Branicky_, Dec 20 2021
```

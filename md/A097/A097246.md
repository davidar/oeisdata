## Python
```Python
from sympy import factorint, nextprime
from operator import mul
def a(n):
    f=factorint(n)
    return 1 if n==1 else reduce(mul, [(nextprime(i)**int(f[i]/2))*(i**(f[i]%2)) for i in f]) # _Indranil Ghosh_, May 15 2017
```

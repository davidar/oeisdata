## Sage
```Sage
def A097344_list(size):
    R, L = [1], [1]
    inc = sqr = 1
    for i in range(1, size):
        for j in range(i):
            L[j] *= i / (i - j)
        inc += 2; sqr += inc
        L.extend(1 / sqr)
        R.append(sum(L).numerator())
    return R
A097344_list(50) # (after _Chai Wah Wu_) _Peter Luschny_, Jun 05 2016
```

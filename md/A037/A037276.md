## Python
```Python
from sympy import factorint
def a(n):
    f=factorint(n)
    l=sorted(f)
    return 1 if n==1 else int("".join(str(i)*f[i] for i in l))
print([a(n) for n in range(1, 101)]) # _Indranil Ghosh_, Jun 23 2017
```

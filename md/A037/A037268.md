## Python
```Python
from fractions import Fraction
def ok(n):
  sn = str(n)
  return False if '0' in sn else sum(Fraction(1, int(d)) for d in sn) == 1
def aupto(limit): return [m for m in range(1, limit+1) if ok(m)]
print(aupto(8824)) # _Michael S. Branicky_, Jan 22 2021
```

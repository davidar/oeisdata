## Python
```Python
from itertools import groupby
A037015_list = []
for n in range(10**5):
    c = None
    for x, y in groupby(bin(n)[2:]):
        z = len(list(y))
        if c != None and z >= c:
            break
        c = z
    else:
        A037015_list.append(n) # _Chai Wah Wu_, Sep 14 2021
```

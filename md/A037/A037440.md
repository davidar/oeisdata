## Python
```Python
from sympy.ntheory import digits
def ok(n): return set(map(int, str(n))) == set(digits(n, 7)[1:])
print([k for k in range(1, 10**6) if ok(k)]) # _Michael S. Branicky_, Apr 22 2023
```

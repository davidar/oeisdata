## Python
```Python
from fractions import Fraction
def ok(n):
    ds = list(map(int, str(n)))
    return 0 not in ds and sum(Fraction(1, d) for d in ds).numerator == 1
print(list(filter(ok, range(1, 6337)))) # _Michael S. Branicky_, Aug 08 2021
```

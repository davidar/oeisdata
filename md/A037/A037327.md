## Python
```Python
from numpy import base_repr
def ok(n):
    return sum(map(int, base_repr(n, 6))) == sum(map(int, base_repr(n, 7)))
print([n for n in range(1, 10**5) if ok(n)])
# _Christoph B. Kassir_, Apr 05 2023
```

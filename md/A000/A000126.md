## Python
```Python
def seq(n):
    if n < 0:
        return 1
    a, b = 1, 1
    for i in range(n + 1):
        a, b = b, a + b + i
    return a
[seq(i) for i in range(n)] # _Reza K Ghazi_, Mar 03 2019
```

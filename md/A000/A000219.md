## Python
```Python
from sympy import cacheit
from sympy.ntheory import divisor_sigma
@cacheit
def A000219(n):
    if n <= 1:
        return 1
    return sum(A000219(n - k) * divisor_sigma(k, 2) for k in range(1, n + 1)) // n
print([A000219(n) for n in range(20)])
# _R. J. Mathar_, Oct 18 2009
```
## Julia
```Julia
using Nemo, Memoize
@memoize function a(n)
    if n == 0 return 1 end
    s = sum(a(n - j) * divisor_sigma(j, 2) for j in 1:n)
    return div(s, n)
end
[a(n) for n in 0:20] # _Peter Luschny_, May 03 2020
```

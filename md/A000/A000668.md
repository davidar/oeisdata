## Maple
```Maple
A000668 := proc(n) local i;
i := 2^(ithprime(n))-1:
if (isprime(i)) then
   return i
fi: end:
seq(A000668(n), n=1..31); # _Jani Melik_, Feb 09 2011
# Alternate:
seq(numtheory:-mersenne([i]),i=1..26); # _Robert Israel_, Jul 13 2014
```

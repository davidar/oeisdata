## Python
```Python
from gmpy2 import digits, mpq
def a(n): return len(digits(sum(mpq(1, n) for n in range(1, 10**n+1)).numerator))
print([a(n) for n in range(6)]) # _Michael S. Branicky_, Jun 22 2023
```

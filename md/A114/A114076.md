## Python
```Python
from sympy import integer_nthroot, totient as phi
def ok(k): return integer_nthroot(k * phi(k), 3)[1]
print([k for k in range(1, 60000) if ok(k)]) # _Michael S. Branicky_, Dec 08 2022
```

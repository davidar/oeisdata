## Python
```Python
from sympy import isprime
def aupto(N):
  return [t for t in (int(str(k)+str(k+1)+str(k)) for k in range(1, N+1, 2)) if isprime(t)]
print(aupto(217)) # _Michael S. Branicky_, Jul 09 2021
```

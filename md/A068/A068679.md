## Python
```Python
from sympy import isprime
A068679_list, n = [], 1
while len(A068679_list) < 1000:
    if isprime(10*n+1):
        s = str(n)
        for i in range(len(s)):
            if not isprime(int(s[:i]+'1'+s[i:])):
                break
        else:
            A068679_list.append(n)
    n += 1 # _Chai Wah Wu_, Oct 02 2019
```

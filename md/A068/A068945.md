## Python
```Python
from gmpy2 import popcount
from math import comb, prod
def a(n):
    return popcount(prod((n-m+1)**comb(m+n-2, m-1) for m in range(1, n+1)))
print([a(n) for n in range(16)]) # _Michael S. Branicky_, Apr 09 2023
```

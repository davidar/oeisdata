## Python
```Python
from __future__ import division
def A068084(n):
    u,v,t = 4*(n+1),(2*(n+1))**2-1,4*n*(n+1)
    while True:
        if not v % t:
            return v//8
        v += u+1
        u += 2 # _Chai Wah Wu_, Jan 13 2016
```

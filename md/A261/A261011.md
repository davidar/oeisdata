## Maple
```Maple
p:=3; a:=[]; M:=200; Digits:=30;
for n from 1 to M do
# is n a p-th power?
t1:=round(evalf(n^(1/p)));
if t1^p = n then a:=[op(a),n];
else t2:=ceil(evalf(n^(1/p)));
      if (n mod t2) = 0 then a:=[op(a),n]; fi;
fi;
od:
a;
```

## Python
```Python
from itertools import count
def a(n):
    b, k = bin(n)[2:], 1
    return next(k for k in count(1) if bin(k)[2:] not in b)
print([a(n) for n in range(86)]) # _Michael S. Branicky_, Feb 26 2023
```

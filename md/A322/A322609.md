## Python
```Python
from sympy import divisors, factorint
A322609_list = [k for k in range(1,10**3) if sum(d for d in divisors(k,generator=True) if max(factorint(d).values(),default=1) >= 2) == 2*k] # _Chai Wah Wu_, Sep 19 2021
```

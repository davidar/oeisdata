## Python
```Python
x = numpy.arange(n)
y = numpy.sin(x)
y = numpy.cumsum(y)
y = numpy.minimum.accumulate(y)
y = numpy.diff(y)
y = numpy.insert(y, 0, -1)
x[y < 0.]
```

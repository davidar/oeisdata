## Maple
```Maple
f:= proc(n) option remember;
local m,d;
d:= n mod 10; m:= floor(n/10);
if d >= 5 then 100*procname(m) + 2*d
else 10*procname(m)+2*d
fi
end proc:
f(0):= 0:
map(f, [$0..100]); # _Robert Israel_, Nov 28 2018
```

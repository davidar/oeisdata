## Python
```Python
from sympy import isprime, primerange
def ok(p): swb = str(p); return isprime(int(swb + '9' + swb))
def aupto(lim): return [p for p in primerange(1, lim+1) if ok(p)]
print(aupto(1260)) # _Michael S. Branicky_, Jul 13 2021
```

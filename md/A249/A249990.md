## Python
```Python
TOP = 100
a = list(range(TOP))
for step in range(2,TOP):
    numBlocks = (len(a)-1) // step
    if numBlocks==0:  break
    a = a[:(1+numBlocks*step)]
    for pos in range(1,len(a),step):
        a[pos:pos+step] = a[pos+step-1:pos-1:-1]
    for i in range(1, step+1):  print(str(a[i]), end=',')
    a[1:] = a[step+1:]
```

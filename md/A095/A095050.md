## Python
```Python
from sympy import divisors
def ok(n):
    digits_used = set()
    for d in divisors(n):
        digits_used |= set(str(d))
    return len(digits_used) == 10
print([k for k in range(1040) if ok(k)]) # _Michael S. Branicky_, Oct 28 2021
```

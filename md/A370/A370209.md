## Python
```Python
from itertools import count
from sympy import prime, isprime
def A370209(n):
    p = prime(n)
    return next((p<<k)*m for k in count(p.bit_length()-1) if isprime(m:=(p<<k+1)+1)) # _Chai Wah Wu_, Feb 17 2024
```

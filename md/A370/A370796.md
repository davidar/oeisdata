## Maple
```Maple
A370796:= proc (n)
local count, a, b, p:
count := 0:
a := (ithprime(n)+1)^2:
b := (ithprime(n+1)-1)^2:
p := n:
while ithprime(p) <= b do if a <= ithprime(p) then count := count+1 end if:
p := p+1 end do:
return count end proc:
A370796(1) := 2:
map(A370796, [$1 .. 100]);
```

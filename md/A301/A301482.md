## Python
```Python
from sympy import divisors
def ok(n):
    divs = divisors(n)[:-1]
    return len(divs) > 1 and sum(d**2 for d in divs)%sum(divs) == 0
print(list(filter(ok, range(4, 32769)))) # _Michael S. Branicky_, Aug 22 2021
```

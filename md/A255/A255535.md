# Numbers representable as both b^c \+ b \+ c and x^y \+ x \- y, where b, c, x, y are integers greater than 1\.

## Python
```Python
TOP = 100000000
a = [0]*TOP
for y in range(2,TOP//2):
  if 2**y+2+y>=TOP: break
  for x in range(2,TOP//2):
    k = x**y+x+y
    if k>=TOP: break
    a[k]=1
for y in range(2,TOP//2):
  if 2**y+2-y>=TOP: break
  for x in range(2,TOP//2):
    k = x**y+x-y
    if k>=TOP: break
    if k>=0: a[k]|=2
print([n for n in range(TOP) if a[n]==3])
```

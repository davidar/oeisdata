## Python
```Python
def a(n):
    s = str(n)
    t = s.strip('0')
    return (10 - int(t)%10) * 10**(len(s) - len(t))
print([a(n) for n in range(1, 85)]) # _Michael S. Branicky_, Sep 08 2021
```

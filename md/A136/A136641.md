## Python
```Python
from math import gcd
from sympy import divisor_count
def a(n):
    k = 1
    while gcd(n, k) != 1 or divisor_count(k) != n: k += 1
    return k
print([a(n) for n in range(1, 19)]) # _Michael S. Branicky_, Mar 25 2022
```

## Python
```Python
from sympy import isprime
def ok(p): return all(isprime(k) for k in [p, p+6, p-6, 2*p+3, 2*p-3])
print([k for k in range(10**6) if ok(k)]) # _Michael S. Branicky_, Jul 25 2022
```

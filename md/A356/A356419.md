## Python
```Python
from math import comb
def A356419(n):
    c, k = 0, 0
    for i,j in enumerate(bin(n)[-1:1:-1]):
        if j == '1':
            k += 1
            c += comb(i,k)
    return comb(n.bit_count()+c+1,2)-c # _Chai Wah Wu_, Mar 02 2023
```

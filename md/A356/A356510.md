## Python
```Python
from sympy import isprime
def ok(n): return isprime(n) and all(isprime(2*n*n-i) for i in [7, 1, -3])
print([k for k in range(10**6) if ok(k)]) # _Michael S. Branicky_, Aug 09 2022
```

## Python
```Python
from math import lcm
from sympy import factorint
def ok(n): f = factorint(n); return lcm(*f.keys()) == lcm(*f.values())
print([k for k in range(10**6) if ok(k)]) # _Michael S. Branicky_, Aug 07 2022
```

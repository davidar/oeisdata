  p:= nextprime(p);

## Python
```Python
from sympy import isprime, nextprime
def a(n):
    p = 2
    while not isprime(p**n - 2): p = nextprime(p)
    return p
print([a(n) for n in range(1, 70)]) # _Michael S. Branicky_, Aug 04 2022
```

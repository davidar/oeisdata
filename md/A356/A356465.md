## Python
```Python
from sympy import prime as p
a = [0,2,6,12,27] #first 4 area values
area = 27
for n in range(5,44+1):
  darea = (p(n) - p(n-2) + p(n-4)) * (p(n-1) - p(n-3))
  area += darea
  a.append(area)
print('a(n)=',a)
```

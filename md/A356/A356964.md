## Python
```Python
def A356964(n):
    a, b, c, s = 1,2,4,0
    for i in bin(n)[-1:1:-1]:
        s += int(i)*a
        a, b, c = b, c, a+b+c
    return s # _Chai Wah Wu_, Sep 10 2022
```

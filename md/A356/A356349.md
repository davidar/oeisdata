## Python
```Python
def ok(n):
    sd = sum(map(int, str(n)))
    return sd and not n%sd and (n%10 or (n//10)%sd)
print([k for k in range(235) if ok(k)]) # _Michael S. Branicky_, Oct 16 2022
```

## Python
```Python
from sympy import divisor_count
def ok(n): sd = sum(map(int, str(n))); return sd%divisor_count(sd) == 0
print([k for k in range(1, 200) if ok(k)]) # _Michael S. Branicky_, Aug 10 2022
```

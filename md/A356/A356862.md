## Python
```Python
from sympy import factorint
from collections import Counter
def ok(k):
    c = Counter(factorint(k)).most_common(2)
    return not (len(c) > 1 and c[0][1] == c[1][1])
print([k for k in range(2, 105) if ok(k)])
```

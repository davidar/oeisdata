## Python
```Python
from sympy import isprime
def ok(m): return 2*sum(1 for x in range(1, m+1) if isprime(x**2-x+m)) > m
print([m for m in range(1, 400) if ok(m)]) # _Michael S. Branicky_, Aug 26 2022
```

# Smallest prime p such that the decimal expansion of p remains prime through exactly n iterations of base\-10 to base\-2 conversion \(A007088\)\.
https://oeis.org/A361312

## Python
```Python
from sympy import isprime, nextprime
p = 1; mx = 5; I = [*range(mx)]; R = [*range(mx)]
while I:
    p = nextprime(p); ct = 0; q = p
    while isprime(int(bin(q)[2:])): ct += 1; q = int(bin(q)[2:])
    if ct in I: R[ct] = p; I.remove(ct)
print(*R, sep = ", ")
```

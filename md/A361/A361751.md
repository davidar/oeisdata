## Python
```Python
def a(n):
    d = len(str(n))
    m = 10**d
    return d*n*(n+1)//2 - ((m-11)*m + 10)//198
print([a(n) for n in range(1, 55)]) # _Michael S. Branicky_, Mar 24 2023 modified Mar 29 2023
```

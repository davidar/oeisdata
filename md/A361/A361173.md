## Python
```Python
from sympy.ntheory import digits, nextprime, prevprime
def ok(n):
    p, q = prevprime(4**n), nextprime(4**n)
    return set(digits(p, 4)[1:]) & set(digits(q, 4)[1:]) == set()
print([k for k in range(1, 99) if ok(k)]) # _Michael S. Branicky_, Mar 03 2023
```

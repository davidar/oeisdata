## Python
```Python
from math import isqrt
from sympy import integer_nthroot, factorint
def A361936(n):
    m = n**2
    return int(sum(isqrt(m//k**3) for k in range(1, integer_nthroot(m, 3)[0]+1) if all(d<=1 for d in factorint(k).values()))) # _Chai Wah Wu_, Sep 10 2024
```

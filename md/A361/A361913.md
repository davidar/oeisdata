## Python
```Python
from gmpy2 import *
def a(n):
  c,d,x,y,g = 0,1,2,2,lambda x:pow(x,2,n)-1
  while d == 1:
    c,x,y =c+1,g(x),g(g(y))
    d = gcd(abs(x-y), n)
  return c
```

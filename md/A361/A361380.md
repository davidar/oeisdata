## Python
```Python
from math import comb
from sympy import bell
def A361380(n): return sum(comb(i,j)*(i-n)**(i-j)*bell(j) for i in range(n+1) for j in range(i+1)) # _Chai Wah Wu_, Apr 05 2023
```

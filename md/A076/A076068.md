## Python
```Python
def a(n):
  s = "".join(sorted("".join(map(str, range((n-1)*n//2+1, n*(n+1)//2+1)))))
  if '0' not in s: return int(s)
  return int(s[s.rfind('0')+1:])
print([a(n) for n in range(1, 16)]) # _Michael S. Branicky_, Jan 23 2021
```

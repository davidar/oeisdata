## Python
```Python
def aupton(nn):
    tri = [i*(i+1)//2 for i in range(1, nn+1)]
    return [sum(t%t2 == 0 for t2 in tri[:j+1]) for j, t in enumerate(tri)]
print(aupton(102)) # _Michael S. Branicky_, Dec 10 2021
```

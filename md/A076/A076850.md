## Python
```Python
from sympy import isprime, prime, nextprime
def afind(limit, startk=1):
    k, tenk = startk, 10**startk
    for k in range(startk, limit+1):
        if isprime((tenk + 2)//6):
            print(k, end=", ")
        tenk *= 10
afind(1000) # _Michael S. Branicky_, Feb 26 2022
```

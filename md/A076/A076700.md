## Python
```Python
from sympy import primepi, integer_nthroot, primerange
def A076700(n): return int(sum(primepi(integer_nthroot(10**n, p)[0]) for p in primerange((10**n).bit_length()))) # _Chai Wah Wu_, Sep 12 2024
```

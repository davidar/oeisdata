## Python
```Python
def diff(lst):
    return [lst[i+1]-lst[i] for i in range(len(lst)-1)]
def auptomult(N):
    return diff(sorted(set([k*i for k in [11, 19] for i in range(N//k+1)])))
print(auptomult(630)) # _Michael S. Branicky_, Nov 08 2021
```

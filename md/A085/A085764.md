## Python
```Python
def ispal(n): s = str(n); return s == s[::-1]
def a(n):
    m, cm = 1, 2
    while cm != n:
        cm = cm - ispal(m) + ispal(2*m+1) + ispal(2*m+2)
        m += 1
    return m
print([a(n) for n in range(1, 55)]) # _Michael S. Branicky_, Oct 01 2021
```

## Python
```Python
from sympy import divisors
def ok(n):
    divs = divisors(n)[:-1]
    return sorted(str(n)) == sorted(str(sum(divs)))
print(list(filter(ok, range(1, 57000)))) # _Michael S. Branicky_, Jun 28 2021
```

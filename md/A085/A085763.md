## Python
```Python
def ispal(s): return s == s[::-1]
def a(n): return sum(ispal(str(i)) for i in range(n, 2*n+1))
print([a(n) for n in range(1, 100)]) # _Michael S. Branicky_, Oct 01 2021
```

## Python
```Python
from sympy import prime, isprime
def A085300(n):
    p = prime(n)
    q = p
    while True:
        m = int(str(q)[::-1])
        if isprime(m):
            return(m)
        q *= p # _Chai Wah Wu_, Jun 02 2016
```

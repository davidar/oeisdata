## Python
```Python
from sympy.ntheory.digits import digits
def ispal(n, b):
    digs = digits(n, b)[1:]
    return digs == digs[::-1]
def a(n): return sum(ispal(n, b) for b in range(2, 11))
print([a(n) for n in range(87)]) # _Michael S. Branicky_, Sep 09 2021
```

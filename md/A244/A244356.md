## Maple
```Maple
filter:= proc(n) local L;
  L:= convert(convert(n,base,10), set) minus {0};
  not ormap(t -> n mod t = 0, L)
end proc:
B:= select(filter, {$1..1000}):
sort(convert(B intersect map(`-`,B,1), list)); # _Robert Israel_, Dec 08 2019
```

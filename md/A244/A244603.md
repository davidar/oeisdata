# Least number k such that n^k contains the digit n n times\.
https://oeis.org/A244603
## JSON

## Python
```Python
def tes(n):
  for k in range(1,10**3):
    if(str(n**k).count(str(n))) == n:
      return k
n = 1
while n < 10:
  print(tes(n),end=', ')
  n += 1
```

# Least number k such that k^n contains the digit n n times\.

## Python
```Python
def tes(n):
  for k in range(1,10**4):
    if(str(k**n).count(str(n))) == n:
      return k
n = 0
while n < 10:
  print(tes(n),end=', ')
  n += 1
```

## Python
```Python
import math
for n in range(1, 40):
    ans = 0
    for k in range(0, n):
        for i in range(2, 5):
            for j in range(1, ((k/i)+1)):
                c = k - (i * j) + j
                ans = ans + (math.factorial(c) / (math.factorial(j) * math.factorial(k-(i*j))))
    print ans
```

## Sage
```Sage
def base32sum(n):
    L, i = [n], 1
    while L[i-1]>2:
        x=L[i-1]
        L[i-1]=x.mod(3)
        L.append(2*floor(x/3))
        i+=1
    return sum(L)
[base32sum(n) for n in [0..85]]
```

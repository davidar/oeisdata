## Python
```Python
from math import prod, factorial as f
def a(n): return prod(f(i)**2+f(j)**2 for i in range(n) for j in range(n))
print([a(n) for n in range(1, 7)]) # _Michael S. Branicky_, Feb 16 2021
```

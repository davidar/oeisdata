## Python
```Python
from sympy import factorint
def ok(n):
    e = sorted(factorint(n).values())
    return n > 1 and max(e) == 1 + min(e)
print([k for k in range(280) if ok(k)]) # _Michael S. Branicky_, Dec 18 2021
```

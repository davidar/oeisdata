## Python
```Python
from sympy import factorint, primepi
def ok(n):
    v = [0, 0]
    for p, e in factorint(n).items(): v[primepi(p)%2] += e
    return v[0] == v[1]
print([k for k in range(300) if ok(k)]) # _Michael S. Branicky_, Apr 16 2022 after _David A. Corneth_
```

## Python
```Python
from itertools import count, islice
def agen():
    for d in count(3):
        b = 1 << d
        for i in range(2, d):
            yield b - (b >> i) - 1
print(list(islice(agen(), 50))) # _Michael S. Branicky_, Oct 13 2022
```

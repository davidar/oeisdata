## Python
```Python
from sympy import factorial, prod
f = factorial
def v(n): return 1 if n<2 else prod(f(k) - f(j) for k in range(2, n + 1) for j in range(1, k))
print([v(n) for n in range(11)]) # _Indranil Ghosh_, Jul 24 2017
```

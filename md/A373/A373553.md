## Python
```Python
def a(n):
    mstar = bin(n)[2:]*2
    knot = next(k for k in range(2, n+2) if bin(k)[2:] not in mstar)
    return knot - 1
print([a(n) for n in range(1, 88)]) # _Michael S. Branicky_, Jun 14 2024
```

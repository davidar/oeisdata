## Python
```Python
def A373448(n):
    if n == 0: return 0
    k = 10**n
    a = 1<<(l:=k.bit_length()-2)
    m = a|(k&a-1)
    return (m<<l+1)+int(bin(m)[:1:-1]or'0',2) if a&k else (m<<l)+int(bin(m)[-2:1:-1]or'0',2)
```

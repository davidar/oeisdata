lastDigit := proc(n)
    return (n * 2^n + 1) mod 10;
end proc:

lastDigits := [seq(lastDigit(n), n = minN .. maxN)];
print(lastDigits);

## Python
```Python
def last_digit(n):
    return (n * 2**n + 1) % 10
# Example usage
min_n, max_n = 1, 10
last_digits = [last_digit(n) for n in range(min_n, max_n + 1)]
print(last_digits)
```

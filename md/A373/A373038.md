## PARI
```PARI
a(n) = { my (b = binary(n), w = hammingweight(n)); fromdigits(concat( b[1..#b-w], Vecrev(b[#b-w+1..#b])), 2); }
```

## Python
```Python
def a(n): w, b = n.bit_count(), bin(n)[2:]; return int(b[:-w]+b[-w:][::-1], 2)
print([a(n) for n in range(68)]) # _Michael S. Branicky_, May 24 2024
```

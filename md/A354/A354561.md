## Python
```Python
from sympy import factorint
def c(n, e): f = factorint(n); return f[max(f)] >= e
def ok(n): return n > 1 and c(n, 3)
print([k for k in range(2501) if ok(k)]) # _Michael S. Branicky_, May 30 2022
```

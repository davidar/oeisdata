## Python
```Python
def a(n):
    minSum = n-1
    for a in range(n-1):
        for b in range(n-a-1):
            if a**b>n:
                break
            c = n-a**b
            if a+b+c<minSum:
                minSum = a+b+c
    return minSum
```

## Python
```Python
from math import prod
from itertools import combinations
from sympy import primerange, prime, isprime
def A354950(n):
    plist = list(primerange(2,p:=prime(n)))
    return sum(1 for l in range(1,n) for d in combinations(plist,l) if isprime((q:= prod(d)*p)-1) and isprime(q+1)) # _Chai Wah Wu_, Jun 14 2022
```

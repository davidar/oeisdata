## Python
```Python
from sympy import factorint
def c(n): f = factorint(n); return f[max(f)] >= 3
def ok(n): return n > 1 and c(n) and c(n+1)
print([k for k in range(10**5) if ok(k)]) # _Michael S. Branicky_, May 30 2022
```

## Python
```Python
from num2words import num2words as n2w
def b(n): return set(c for c in n2w(n).replace(" and", "") if c.isalpha())
def a(n): return len(b(n) & b(n+1))
print([a(n) for n in range(101)]) # _Michael S. Branicky_, Aug 19 2022
```

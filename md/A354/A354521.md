## Python
```Python
from num2words import num2words as n2w
def a(i):
    w1 = n2w(i).replace(' ','').replace('-','')
    w2 = n2w(i+1).replace(' ','').replace('-','')
    for j in range(len(w1)):
        if w1[j] in w2:
            return j+1
```

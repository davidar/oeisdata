## Python
```Python
def a(n):
    digits = list(map(int, str(n)))
    out = (d if d%2 else digits[i-1] for i, d in enumerate(digits))
    return int("".join(map(str, out)))
print([a(n) for n in range(68)]) # _Michael S. Branicky_, Jun 04 2022
```

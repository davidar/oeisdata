## Python
```Python
from sympy import factorint
def c(n): f = factorint(n); return f[max(f)] >= 2
def ok(n): return n > 1 and c(n) and c(n+1)
print(list(filter(ok, range(235225)))) # _Michael S. Branicky_, May 30 2022
```

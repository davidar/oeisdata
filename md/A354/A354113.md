## Python
```Python
def a(n):
    if n == 0: return 10
    s = str(n)
    return int(s[0]+"0"+s[1:]) if s[0] == "1" else int("1"+s)
print([a(n) for n in range(54)]) # _Michael S. Branicky_, May 17 2022
```

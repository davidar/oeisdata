## Python
```Python
from itertools import count, islice
def agen(): # generator of terms
    for d in count(3):
        for i in range(2, d):
            yield int("1"*i + "0"*(d-i), 2)
print(list(islice(agen(), 50))) # _Michael S. Branicky_, Feb 20 2022
```

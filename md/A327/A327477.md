## Python
```Python
from sympy import totient, divisors
def A327477(n): return (1<<n-1)-sum(totient(d)<<n//d-1 for d in divisors(n>>(~n&n-1).bit_length(),generator=True))//n if n else 0 # _Chai Wah Wu_, Feb 21 2023
```

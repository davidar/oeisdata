## Python
```Python
def A327621_upto(N):
    "list(x < N | x = sum(3^j, j in J) + sum(4^k, k in K); J, K subset N*)."
    S = {0} # empty sum
    for b in (3,4):
        p = b
        while p < N: S |= {k+p for k in S if k+p < N} ; p *= b
    return sorted(S) # includes a(0) = 0, so a(1,2,3,...) = 3,4,9,...
# _M. F. Hasler_, Nov 09 2023
```

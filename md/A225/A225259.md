## Python
```Python
def a(n):
  zn = list(range(n+1))
  prod3 = list(set(i*j*k for i in zn for j in zn[i:] for k in zn[j:]))
  return len(set(a+b for i, a in enumerate(prod3) for b in prod3[i:]))
print([a(n) for n in range(1, 42)]) # _Michael S. Branicky_, Mar 12 2021
```

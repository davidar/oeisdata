## Python
```Python
from sympy import factorial as f
def a(n):
    s=0
    k=2
    while(n):
        d=n%k
        n=(n//k)
        if d: s=s+(k - d)*f(k - 1)
        k+=1
    return s
print([a(n) for n in range(101)]) # _Indranil Ghosh_, Jun 19 2017
```

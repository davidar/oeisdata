## Python
```Python
from sympy import divisor_sigma as sigma, totient as phi
def aupto(limit):
  for k in range(1, limit):
    if sigma(phi(k), 1) == k - phi(k): print(k, end=", ")
aupto(2*10**4) # _Michael S. Branicky_, Feb 18 2021
```

## Maple
```Maple
A070885 := proc(n)
    option remember;
    if n = 1 then
        return 1;
    elif type(procname(n-1),'even') then
        procname(n-1) ;
    else
        procname(n-1)+1 ;
    end if;
    %*3/2 ;
end proc:
seq(A070885(n),n=1..80) ; # _R. J. Mathar_, Jun 18 2018
```
